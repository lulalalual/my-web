动态规划最难的不是循环，而是给 `dp` 一个清楚的含义。含义清楚，转移就容易写；含义模糊，代码一定会乱。

## 先问一句

`dp[i]` 到底代表什么？

比如最长递增子序列里，可以定义成：`dp[i]` 表示“以第 `i` 个数结尾的最长长度”。

## 小代码

```cpp
vector<int> dp(n, 1);
for (int i = 0; i < n; i++) {
    for (int j = 0; j < i; j++) {
        if (nums[j] < nums[i]) {
            dp[i] = max(dp[i], dp[j] + 1);
        }
    }
}
```

这里的两层循环不是重点。重点是：如果 `nums[j]` 能接到 `nums[i]` 前面，就可以用 `dp[j] + 1` 更新 `dp[i]`。

## 四步走

- 定义状态。
- 找到选择。
- 写出转移。
- 补上初始值。

## 少踩坑

不要急着写代码。先用一句中文把状态说出来，再写方程。说不出来，就说明还没想明白。
