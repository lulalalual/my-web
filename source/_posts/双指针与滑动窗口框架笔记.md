双指针和滑动窗口都在做一件事：用两个指针圈住一段范围，然后一边移动，一边维护这段范围里的信息。

## 怎么理解

```text
left              right
  |-----------------|
        当前窗口
```

窗口里可以维护字符数量、数字和、最大值、最小值等信息。

## 小代码

```cpp
int left = 0;
for (int right = 0; right < n; right++) {
    add(nums[right]);
    while (needShrink()) {
        remove(nums[left]);
        left++;
    }
    updateAnswer();
}
```

`right` 负责扩张，`left` 负责收缩。只要两个指针都不回头，复杂度通常就是 `O(n)`。

## 适合什么题

- 最长无重复子串。
- 最小覆盖子串。
- 连续子数组条件统计。
- 有序数组两数之和。

## 少踩坑

先想清楚窗口里维护什么，再想什么时候收缩。很多滑动窗口写错，都是因为“合法条件”没定义清楚。
