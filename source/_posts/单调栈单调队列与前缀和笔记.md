这类技巧的共同点是：先维护一点信息，后面回答问题就快很多。

## 单调栈

单调栈适合找“旁边第一个更大或更小的数”。

```cpp
while (!st.empty() && st.top() <= x) {
    st.pop();
}
st.push(x);
```

这段代码会把没用的元素弹掉，只留下以后可能用得上的元素。

## 单调队列

单调队列常用在滑动窗口最大值。窗口往右走时，队头始终放当前窗口里最有用的值。

## 前缀和

前缀和适合快速算区间和：

```cpp
sum[i + 1] = sum[i] + nums[i];
int part = sum[r + 1] - sum[l];
```

原来每次都要从 `l` 加到 `r`，现在减一下就能得到结果。

## 怎么判断

- 问“下一个更大”：想单调栈。
- 问“窗口最大”：想单调队列。
- 问“区间总和”：想前缀和。

先看问题在反复查什么，再决定维护什么结构。
