这三类技巧都在做“预处理”或“在线维护”：先维护一个结构，后面每次查询就能更快回答。

## 单调栈

适合求：

- 下一个更大元素。
- 下一个更小元素。
- 柱状图最大矩形。

模板核心：

```cpp
stack<int> st;
for (int i = nums.size() - 1; i >= 0; --i) {
    while (!st.empty() && st.top() <= nums[i]) st.pop();
    ans[i] = st.empty() ? -1 : st.top();
    st.push(nums[i]);
}
```

## 单调队列

适合滑动窗口最值问题。队头永远保存当前窗口最优元素，下标过期就弹出。

## 前缀和

定义：

```cpp
prefix[i + 1] = prefix[i] + nums[i];
```

这样任意区间 `[l, r]` 的和就是：

```cpp
prefix[r + 1] - prefix[l]
```

## 怎么判断该用哪个

- 查询“前面/后面第一个更大更小”：单调栈。
- 窗口内最大最小值：单调队列。
- 区间和、区间计数、和为 k：前缀和。

这三类技巧的共同点是把重复计算挪到维护阶段，让单次查询变成 `O(1)` 或摊还 `O(1)`。
