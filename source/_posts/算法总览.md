## 通俗导读

算法不是背题，而是把问题归类。看到题目时，先判断它像哪一类：窗口、链表、搜索、动态规划，还是图论。

```text
题目 -> 识别模型 -> 套基本框架 -> 调整边界
```

```cpp
int left = 0;
for (int right = 0; right < n; ++right) {
    add(nums[right]);
    while (needShrink()) {
        remove(nums[left++]);
    }

```

这是滑动窗口的骨架。窗口题的关键不是这几行代码，而是你维护了什么状态，什么时候扩张，什么时候收缩。

# 双指针

## 一、双指针核心思想

双指针是一种**高效遍历数据结构**的算法技巧，通过使用两个指针（索引）协同工作，在特定规则下移动，以O(n)时间复杂度解决复杂问题。

### 核心优势

- **时间复杂度O(n)**：仅需单次遍历
    
- **空间复杂度O(1)**：原地操作，无需额外空间
    
- **解决复杂问题**：简化嵌套循环，避免O(n²)复杂度
    

## 二、双指针类型与应用场景

### 1. 对撞指针（左右指针）

- **特点**：一左一右，向中间移动
    
- **适用场景**：有序数组/字符串、回文判断、元素交换
    
- **经典问题**：
    
    - 两数之和（有序数组）
        
    - 三数之和
        
    - 反转数组/字符串
        
    - 奇偶数调整
        

### 2. 快慢指针

- **特点**：一快一慢，同向移动
    
- **适用场景**：链表操作、数组去重、检测环
    
- **经典问题**：
    
    - 链表环检测
        
    - 寻找链表中间节点
        
    - 数组去重
        
    - 删除有序数组重复项
        

### 3. 滑动窗口

- **特点**：维护动态变化的窗口边界
    
- **适用场景**：子串/子数组问题
    
- **经典问题**：
    
    - 最小覆盖子串
        
    - 最长无重复子串
        
    - 长度最小的子数组
        

## 三、元素逆序问题（对撞指针）

### 问题描述

将数组/字符串中的元素逆序排列

### 算法实现

```cpp
// 数组逆序
void reverseArray(int arr[], int n) {
    int left = 0, right = n - 1;
    while (left < right) {
        // 交换左右指针元素
        std::swap(arr[left], arr[right]);
        // 移动指针
        left++;
        right--;
    }
}

// 字符串逆序
void reverseString(std::string& s) {
    int left = 0, right = s.size() - 1;
    while (left < right) {
        std::swap(s[left], s[right]);
        left++;
        right--;
    }
}
```

### 复杂度分析

- 时间复杂度：O(n)（n为元素个数）
    
- 空间复杂度：O(1)（原地交换）
    

### 应用场景

1. 回文判断
    
2. 数字反转（如LeetCode 7）
    
3. 旋转数组
    
4. 图像旋转
    

## 四、奇偶数调整问题（对撞指针）

### 问题描述

将数组中的奇数移到左边，偶数移到右边

### 算法实现

```cpp
void adjustOddEven(int arr[], int n) {
    int left = 0, right = n - 1;
    
    while (left < right) {
        // 从左找偶数
        while (left < right && arr[left] % 2 != 0) 
            left++;
        
        // 从右找奇数
        while (left < right && arr[right] % 2 == 0) 
            right--;
        
        // 交换奇偶数
        if (left < right) {
            std::swap(arr[left], arr[right]);
            left++;
            right--;
        }
    }
}
```

### 扩展：荷兰国旗问题（三色分类）

```cpp
// 将数组分为<target, =target, >target三部分
void dutchFlagSort(int arr[], int n, int target) {
    int low = 0, mid = 0, high = n - 1;
    
    while (mid <= high) {
        if (arr[mid] < target) {
            std::swap(arr[low], arr[mid]);
            low++;
            mid++;
        } else if (arr[mid] == target) {
            mid++;
        } else {
            std::swap(arr[mid], arr[high]);
            high--;
        }
    }
}
```

## 五、双指针经典问题详解

### 1. 两数之和（有序数组）

```cpp
std::vector<int> twoSum(std::vector<int>& nums, int target) {
    int left = 0, right = nums.size() - 1;
    
    while (left < right) {
        int sum = nums[left] + nums[right];
        if (sum == target) {
            return {left, right};
        } else if (sum < target) {
            left++;
        } else {
            right--;
        }
    }
    return {};
}
```

### 2. 链表环检测（快慢指针）

```cpp
bool hasCycle(ListNode* head) {
    if (!head || !head->next) return false;
    
    ListNode* slow = head;
    ListNode* fast = head->next;
    
    while (fast && fast->next) {
        if (slow == fast) return true;
        slow = slow->next;
        fast = fast->next->next;
    }
    return false;
}
```

### 3. 无重复字符的最长子串（滑动窗口）

```cpp
int lengthOfLongestSubstring(std::string s) {
    std::unordered_map<char, int> map;
    int maxLen = 0;
    int left = 0;
    
    for (int right = 0; right < s.size(); right++) {
        char c = s[right];
        if (map.find(c) != map.end() && map[c] >= left) {
            left = map[c] + 1;
        }
        map[c] = right;
        maxLen = std::max(maxLen, right - left + 1);
    }
    return maxLen;
}
```

## 六、双指针解题框架

### 对撞指针通用模板

```cpp
void twoPointersSolve(Type* data, int n) {
    int left = 0, right = n - 1;
    
    while (left < right) {
        // 1. 处理左侧元素
        while (left < right && conditionLeft(data[left])) 
            left++;
        
        // 2. 处理右侧元素
        while (left < right && conditionRight(data[right])) 
            right--;
        
        // 3. 处理相遇情况
        if (left < right) {
            // 执行操作（交换/计算等）
            process(data[left], data[right]);
            left++;
            right--;
        }
    }
}
```

### 快慢指针通用模板

```cpp
void fastSlowPointersSolve(Type* data, int n) {
    int slow = 0, fast = 0;
    
    while (fast < n) {
        // 1. 快指针探索
        if (condition(data[fast])) {
            // 2. 慢指针更新
            data[slow] = data[fast];
            slow++;
        }
        fast++;
    }
}
```

## 七、双指针使用技巧

1. **明确指针含义**：
    
    - 左指针：通常表示已处理部分的边界
        
    - 右指针：通常表示未处理部分的边界
        
2. **确定移动条件**：
    
    - 何时移动左指针？
        
    - 何时移动右指针？
        
    - 何时同时移动？
        
3. **处理边界情况**：
    
    - 空数组/空字符串
        
    - 单元素数组
        
    - 指针相遇/交叉
        
4. **验证终止条件**：
    
    - 确保不会越界
        
    - 确保覆盖所有情况
        

## 八、双指针实战案例

### 案例1：盛最多水的容器

```cpp
int maxArea(std::vector<int>& height) {
    int left = 0, right = height.size() - 1;
    int maxWater = 0;
    
    while (left < right) {
        int width = right - left;
        int h = std::min(height[left], height[right]);
        maxWater = std::max(maxWater, width * h);
        
        // 移动较矮的一侧
        if (height[left] < height[right]) {
            left++;
        } else {
            right--;
        }
    }
    return maxWater;
}
```

### 案例2：三数之和

```cpp
std::vector<std::vector<int>> threeSum(std::vector<int>& nums) {
    std::sort(nums.begin(), nums.end());
    std::vector<std::vector<int>> result;
    
    for (int i = 0; i < nums.size() - 2; i++) {
        if (i > 0 && nums[i] == nums[i-1]) continue;
        
        int left = i + 1, right = nums.size() - 1;
        while (left < right) {
            int sum = nums[i] + nums[left] + nums[right];
            if (sum == 0) {
                result.push_back({nums[i], nums[left], nums[right]});
                while (left < right && nums[left] == nums[left+1]) left++;
                while (left < right && nums[right] == nums[right-1]) right--;
                left++;
                right--;
            } else if (sum < 0) {
                left++;
            } else {
                right--;
            }
        }
    }
    return result;
}
```




# 单链表

## 链表节点定义

```cpp
struct ListNode {
    int val;
    ListNode* next;
    ListNode(int x) : val(x), next(nullptr) {}
};
```

## 1. 单链表逆序（反转链表）

### 问题描述

将给定的单链表反转，例如：`1->2->3->4->5` 反转为 `5->4->3->2->1`

### 解题思路

使用迭代法或递归法反转指针方向：

- **迭代法**：三个指针（prev, curr, next）遍历链表，逐个反转节点指针
    
- **递归法**：递归到链表尾部，回溯时反转指针
    

### 代码实现（迭代法）

```cpp
ListNode* reverseList(ListNode* head) {
    ListNode* prev = nullptr;
    ListNode* curr = head;
    
    while (curr) {
        ListNode* next = curr->next;  
        // 保存下一个节点
        curr->next = prev;           
        // 反转当前节点指针
        prev = curr;                  
        // 移动prev指针
        curr = next;                  
        // 移动curr指针
    }
    return prev;  
    // prev成为新头节点
}
```

### 复杂度分析

- **时间复杂度**：O(n)
    
- **空间复杂度**：O(1)

## 2. 单链表倒数第K个节点

### 问题描述

找到单链表倒数第K个节点（从1开始计数）

### 解题思路

使用双指针技巧：

1. 快指针先走K步
    
2. 然后快慢指针同时移动
    
3. 当快指针到达末尾时，慢指针指向倒数第K个节点
    

### 代码实现

```cpp
ListNode* findKthFromEnd(ListNode* head, int k) {
    if (!head || k <= 0) return nullptr;
    
    ListNode* fast = head;
    ListNode* slow = head;
    
    // 快指针先走k步
    for (int i = 0; i < k; i++) {
        if (!fast) return nullptr;  
        // k大于链表长度
        fast = fast->next;
    }
    
    // 同时移动快慢指针
    while (fast) {
        slow = slow->next;
        fast = fast->next;
    }
    
    return slow;
}
```

### 复杂度分析

- **时间复杂度**：O(n)
    
- **空间复杂度**：O(1)

## 3. 合并两个有序单链表

### 问题描述

将两个升序链表合并为一个新的升序链表

### 解题思路

使用递归或迭代方法：

- **迭代法**：创建一个哑节点，比较两个链表节点值，依次连接较小节点
    
- **递归法**：比较两个链表头节点，递归合并剩余部分
    

### 代码实现（迭代法）

```cpp
ListNode* mergeTwoLists(ListNode* l1, ListNode* l2) {
    ListNode dummy(0);  // 哑节点
    ListNode* tail = &dummy;
    
    while (l1 && l2) {
        if (l1->val <= l2->val) {
            tail->next = l1;
            l1 = l1->next;
        } else {
            tail->next = l2;
            l2 = l2->next;
        }
        tail = tail->next;
    }
    
    // 连接剩余部分
    tail->next = l1 ? l1 : l2;
    
    return dummy.next;
}
```

### 复杂度分析

- **时间复杂度**：O(m+n)
    
- **空间复杂度**：O(1)

## 4. 判断单链表是否存在环

### 问题描述

判断给定的单链表是否有环

### 解题思路

使用快慢指针（Floyd判圈算法）：

- 快指针每次走两步，慢指针每次走一步
    
- 如果快慢指针相遇，则有环
    
- 如果快指针到达末尾，则无环
    

### 代码实现

```cpp
bool hasCycle(ListNode* head) {
    if (!head || !head->next) return false;
    
    ListNode* slow = head;
    ListNode* fast = head->next;
    
    while (fast && fast->next) {
        if (slow == fast) return true;
        slow = slow->next;
        fast = fast->next->next;
    }
    return false;
}
```

### 复杂度分析

- **时间复杂度**：O(n)
    
- **空间复杂度**：O(1)

## 5. 两个链表是否相交

### 问题描述

判断两个单链表是否相交（Y形相交，非X形）

### 解题思路

1. 计算两个链表的长度差d
    
2. 长链表先走d步
    
3. 然后两个链表同时移动，比较节点是否相同
    

### 代码实现

```cpp
ListNode* getIntersectionNode(ListNode* headA, ListNode* headB) {
    if (!headA || !headB) return nullptr;
    
    // 计算链表长度
    int lenA = 0, lenB = 0;
    ListNode* pA = headA;
    ListNode* pB = headB;
    
    while (pA) { lenA++; pA = pA->next; }
    while (pB) { lenB++; pB = pB->next; }
    
    // 重置指针
    pA = headA;
    pB = headB;
    
    // 长链表先走差值步
    if (lenA > lenB) {
        for (int i = 0; i < lenA - lenB; i++)
            pA = pA->next;
    } else {
        for (int i = 0; i < lenB - lenA; i++)
            pB = pB->next;
    }
    
    // 同时移动比较节点
    while (pA && pB) {
        if (pA == pB) return pA;
        pA = pA->next;
        pB = pB->next;
    }
    
    return nullptr;
}
```

### 复杂度分析

- **时间复杂度**：O(m+n)
    
- **空间复杂度**：O(1)

## 6. 删除链表倒数第K个节点

### 问题描述

删除单链表倒数第K个节点，返回修改后的链表头

### 解题思路

1. 使用双指针找到倒数第K+1个节点
    
2. 修改该节点的next指针，跳过倒数第K个节点
    
3. 处理头节点被删除的特殊情况
    

### 代码实现

```cpp
ListNode* removeNthFromEnd(ListNode* head, int n) {
    ListNode dummy(0);
    dummy.next = head;
    
    ListNode* fast = &dummy;
    ListNode* slow = &dummy;
    
    // 快指针先走n+1步
    for (int i = 0; i <= n; i++) {
        if (!fast) return head;  // n大于链表长度
        fast = fast->next;
    }
    
    // 同时移动快慢指针
    while (fast) {
        slow = slow->next;
        fast = fast->next;
    }
    
    // 删除倒数第n个节点
    ListNode* toDelete = slow->next;
    slow->next = toDelete->next;
    delete toDelete;
    
    return dummy.next;
}
```

### 复杂度分析

- **时间复杂度**：O(n)
    
- **空间复杂度**：O(1)

## 7. 旋转链表

### 问题描述

将链表向右旋转K个位置，例如：`1->2->3->4->5`, k=2 → `4->5->1->2->3`

### 解题思路

1. 计算链表长度n
    
2. 取模：k = k % n（处理k>n的情况）
    
3. 找到倒数第k+1个节点（新尾节点）
    
4. 将链表分为两部分，重新连接
    

### 代码实现

```cpp
ListNode* rotateRight(ListNode* head, int k) {
    if (!head || !head->next || k == 0) return head;
    
    // 计算链表长度
    int n = 1;
    ListNode* tail = head;
    while (tail->next) {
        tail = tail->next;
        n++;
    }
    
    // 取模处理
    k = k % n;
    if (k == 0) return head;  // 无需旋转
    
    // 找到新尾节点（倒数第k+1个节点）
    ListNode* newTail = head;
    for (int i = 0; i < n - k - 1; i++) {
        newTail = newTail->next;
    }
    
    // 新头节点
    ListNode* newHead = newTail->next;
    
    // 断开并重新连接
    newTail->next = nullptr;
    tail->next = head;
    
    return newHead;
}
```

### 复杂度分析

- **时间复杂度**：O(n)
    
- **空间复杂度**：O(1)

## 8. 链表的中间节点

### 问题描述

找到单链表的中间节点（偶数个节点时返回第二个中间节点）

### 解题思路

使用快慢指针：

- 慢指针每次走一步
    
- 快指针每次走两步
    
- 当快指针到达末尾时，慢指针指向中间节点
    

### 代码实现

```cpp
ListNode* middleNode(ListNode* head) {
    ListNode* slow = head;
    ListNode* fast = head;
    
    while (fast && fast->next) {
        slow = slow->next;
        fast = fast->next->next;
    }
    
    return slow;
}
```

### 复杂度分析

- **时间复杂度**：O(n)
    
- **空间复杂度**：O(1)

## 9. 两两交换链表中的节点

### 问题描述

交换链表中相邻的节点，例如：`1->2->3->4` → `2->1->4->3`

### 解题思路

使用迭代法，每次处理两个节点：

1. 保存前驱节点
    
2. 交换两个节点
    
3. 更新前驱节点
    

### 代码实现

```cpp
ListNode* swapPairs(ListNode* head) {
    ListNode dummy(0);
    dummy.next = head;
    ListNode* prev = &dummy;
    
    while (head && head->next) {
        // 保存节点
        ListNode* first = head;
        ListNode* second = head->next;
        
        // 交换节点
        prev->next = second;
        first->next = second->next;
        second->next = first;
        
        // 更新指针
        prev = first;
        head = first->next;
    }
    
    return dummy.next;
}
```

### 复杂度分析

- **时间复杂度**：O(n)
    
- **空间复杂度**：O(1)
    

## 10. 删除排序链表中的重复元素

### 问题描述

删除排序链表中所有重复元素，使每个元素只出现一次

### 解题思路

遍历链表，比较当前节点与前一节点值：

- 如果相同，删除当前节点
    
- 否则继续遍历
    

### 代码实现

```cpp
ListNode* deleteDuplicates(ListNode* head) {
    if (!head) return nullptr;
    
    ListNode* curr = head;
    while (curr->next) {
        if (curr->val == curr->next->val) {
            ListNode* temp = curr->next;
            curr->next = temp->next;
            delete temp;
        } else {
            curr = curr->next;
        }
    }
```

## 一、括号匹配问题

### 问题描述

给定一个只包含 `'('`, `')'`, `'{'`, `'}'`, `'['`, `']'` 的字符串，判断字符串中的括号是否正确匹配。

### 解题思路

使用栈数据结构：

1. 遍历字符串中的每个字符
    
2. 遇到左括号（`(`, `{`, `[`）时压入栈中
    
3. 遇到右括号（`)`, `}`, `]`）时：
    
    - 如果栈为空，说明没有匹配的左括号 → 不匹配
        
    - 弹出栈顶元素，检查是否与当前右括号匹配 → 不匹配则返回 false
        
4. 遍历结束后，如果栈不为空 → 不匹配
    

### 代码实现

```cpp
#include <iostream>
#include <stack>
#include <string>
#include <unordered_map>
using namespace std;

bool isValidParentheses(string s) {
    stack<char> st;
    // 映射右括号到对应的左括号
    unordered_map<char, char> mapping = {
        {')', '('},
        {']', '['},
        {'}', '{'}
    };
    
    for (char c : s) {
        if (mapping.find(c) != mapping.end()) { // 当前字符是右括号
            // 获取栈顶元素，如果栈为空则用特殊字符代替
            char top_element = st.empty() ? '#' : st.top();
            
            // 如果栈为空或不匹配
            if (st.empty() || mapping[c] != top_element) {
                return false;
            }
            st.pop(); // 匹配成功，弹出栈顶元素
        } else { // 当前字符是左括号
            st.push(c);
        }
    }
    
    // 如果栈为空，说明所有括号都匹配
    return st.empty();
}

```

### 复杂度分析

- **时间复杂度**：O(n)，其中 n 是字符串长度
    
- **空间复杂度**：O(n)，最坏情况下栈的大小
    

### 扩展应用

1. HTML/XML标签匹配
    
2. 数学表达式中的括号验证
    
3. 编译器语法分析

## 二、逆波兰表达式求解（后缀表达式求值）

### 问题描述

逆波兰表达式（后缀表达式）是一种将运算符放在操作数后面的表达式。例如：

- 中缀表达式：`(2 + 1) * 3`
    
- 后缀表达式：`2 1 + 3 *`
    

给定后缀表达式，计算表达式的值。

### 解题思路

使用栈数据结构：

1. 遍历后缀表达式：
    
    - 遇到操作数：压入栈中
        
    - 遇到运算符：弹出栈顶两个操作数，进行运算，将结果压入栈中
        
2. 表达式遍历结束后，栈顶元素即为结果
    

### 代码实现

```cpp

int evalRPN(vector<string>& tokens) {
    stack<int> st;
    
    for (string token : tokens) {
        // 如果是运算符
        if (token == "+" || token == "-" || token == "*" || token == "/") {
            // 弹出两个操作数
            int num2 = st.top(); st.pop();
            int num1 = st.top(); st.pop();
            
            // 执行运算
            if (token == "+") st.push(num1 + num2);
            else if (token == "-") st.push(num1 - num2);
            else if (token == "*") st.push(num1 * num2);
            else if (token == "/") st.push(num1 / num2);
        } else {
            // 操作数：转换为整数并入栈
            st.push(stoi(token));
        }
    }
    
    return st.top(); // 最终结果
}


```

|`^`|幂运算|`["2","3","^"]`|8|

## 三、中缀表达式转后缀表达式

### 问题描述

将中缀表达式（常见数学表达式，如 `3 + 4 * 2 / (1 - 5)`）转换为后缀表达式（逆波兰表达式）。

### 解题思路

使用栈管理运算符优先级：

1. 初始化一个运算符栈
    
2. 遍历中缀表达式：
    
    - 遇到操作数：直接输出
        
    - 遇到左括号：压入栈中
        
    - 遇到右括号：弹出栈顶运算符并输出，直到遇到左括号（左括号弹出但不输出）
        
    - 遇到运算符：
        
        - 如果栈为空或栈顶为左括号，直接压入
            
        - 如果当前运算符优先级高于栈顶运算符，压入栈中
            
        - 否则，弹出栈顶运算符并输出，重复此过程
            
3. 表达式遍历结束后，弹出栈中所有运算符并输出
    

### 运算符优先级

|运算符|优先级|
|:--|:--|
|`(`|0|
|`+`, `-`|1|
|`*`, `/`|2|
|`^`|3|

### 代码实现

```cpp

// 获取运算符优先级
int getPriority(char op) {
    if (op == '+' || op == '-') return 1;
    if (op == '*' || op == '/') return 2;
    if (op == '^') return 3;
    return 0; // 其他字符（如括号）
}

// 中缀转后缀
vector<string> infixToPostfix(string expression) {
    stack<char> ops;
    vector<string> output;
    istringstream iss(expression);
    string token;
    
    while (iss >> token) {
        // 处理多位数或小数
        if (all_of(token.begin(), token.end(), [](char c) { 
            return isdigit(c) || c == '.'; 
        })) {
            output.push_back(token);
            continue;
        }
        
        char c = token[0]; // 单字符操作符
        
        if (c == '(') {
            ops.push(c);
        } else if (c == ')') {
            // 弹出直到遇到左括号
            while (!ops.empty() && ops.top() != '(') {
                output.push_back(string(1, ops.top()));
                ops.pop();
            }
            ops.pop(); // 弹出左括号
        } else {
            // 处理运算符
            while (!ops.empty() && getPriority(ops.top()) >= getPriority(c)) {
                output.push_back(string(1, ops.top()));
                ops.pop();
            }
            ops.push(c);
        }
    }
    
    // 弹出栈中剩余运算符
    while (!ops.empty()) {
        output.push_back(string(1, ops.top()));
        ops.pop();
    }
    
    return output;
}

```

### 复杂度分析

- **时间复杂度**：O(n)
    
- **空间复杂度**：O(n)
    

### 应用场景

1. 计算器实现
    
2. 编译器前端处理
    
3. 数学表达式解析

```cpp
            else if (token == "*") st.push(num1 * num2);
            else if (token == "/") st.push(num1 / num2);
        } else {
            st.push(stod(token));
        }
    }
    
    return st.top();
}

}
```

# 二分搜索算法

## 一、二分搜索核心思想

### 基本原理

- **分治策略**：将有序数组不断二分，缩小搜索范围
    
- **前提条件**：数组必须**有序**（升序或降序）
    
- **核心操作**：比较中间元素与目标值，决定搜索方向
    
- **时间复杂度**：O(log n) - 每次搜索范围减半
    
- **空间复杂度**：O(1)（迭代）或 O(log n)（递归）
    

### 算法步骤

1. 初始化搜索范围：`left = 0`, `right = n-1`
    
2. 当 `left <= right` 时循环：
    
    - 计算中间位置：`mid = left + (right - left) / 2`
        
    - 比较 `arr[mid]` 与目标值 `target`：
        
        - 相等：找到目标，返回索引
            
        - 小于目标：搜索右半部分 (`left = mid + 1`)
            
        - 大于目标：搜索左半部分 (`right = mid - 1`)
            
3. 若循环结束未找到，返回 -1
    

---

## 二、二分搜索实现代码

### 1. 迭代实现（推荐）

```cpp
#include <iostream>
#include <vector>
using namespace std;

// 二分搜索迭代实现
int binarySearchIterative(vector<int>& nums, int target) {
    int left = 0;
    int right = nums.size() - 1;
    
    while (left <= right) {
        int mid = left + (right - left) / 2;  // 防止溢出
        
        if (nums[mid] == target) {
            return mid;  // 找到目标
        } else if (nums[mid] < target) {
            left = mid + 1;  // 搜索右半部分
        } else {
            right = mid - 1;  // 搜索左半部分
        }
    }
    
    return -1;  // 未找到
}

// 测试代码
int main() {
    vector<int> nums = {1, 3, 5, 7, 9, 11, 13};
    int target = 7;
    
    int result = binarySearchIterative(nums, target);
    if (result != -1) {
        cout << "Element found at index: " << result << endl;
    } else {
        cout << "Element not found" << endl;
    }
    
    return 0;
}
```

### 2. 递归实现

```cpp
    }
```

## 四、经典二分搜索题目

### 1. 搜索旋转排序数组

**问题描述**：

在旋转过的有序数组中搜索目标值（例如 [4,5,6,7,0,1,2] 是 [0,1,2,4,5,6,7] 的旋转）

**解题思路**：

- 确定哪一部分是有序的
    
- 判断目标值是否在有序部分内
    

```cpp
int searchRotatedArray(vector<int>& nums, int target) {
    int left = 0, right = nums.size() - 1;
    
    while (left <= right) {
        int mid = left + (right - left) / 2;
        
        if (nums[mid] == target) {
            return mid;
        }
        
        // 左半部分有序
        if (nums[left] <= nums[mid]) {
            if (nums[left] <= target && target < nums[mid]) {
                right = mid - 1;
            } else {
                left = mid + 1;
            }
        } 
        // 右半部分有序
        else {
            if (nums[mid] < target && target <= nums[right]) {
                left = mid + 1;
            } else {
                right = mid - 1;
            }
        }
    }
    
    return -1;
}
```

### 2. 在排序数组中查找元素的第一个和最后一个位置

**问题描述**：

给定升序数组和目标值，找出目标值的起始和结束位置

**解题思路**：

- 使用两次二分搜索：第一次找左边界，第二次找右边界
    

```cpp
vector<int> searchRange(vector<int>& nums, int target) {
    int left = 0, right = nums.size() - 1;
    int start = -1, end = -1;
    
    // 查找左边界
    while (left <= right) {
        int mid = left + (right - left) / 2;
        if (nums[mid] >= target) {
            right = mid - 1;
        } else {
            left = mid + 1;
        }
        if (nums[mid] == target) start = mid;
    }
    
    left = 0, right = nums.size() - 1;
    // 查找右边界
    while (left <= right) {
        int mid = left + (right - left) / 2;
        if (nums[mid] <= target) {
            left = mid + 1;
        } else {
            right = mid - 1;
        }
        if (nums[mid] == target) end = mid;
    }
    
    return {start, end};
}
```

```cpp
int findPeakElement(vector<int>& nums) {
    int left = 0, right = nums.size() - 1;
    
    while (left < right) {
        int mid = left + (right - left) / 2;
        
        if (nums[mid] > nums[mid + 1]) {
            right = mid;  // 峰值在左侧
        } else {
            left = mid + 1;  // 峰值在右侧
        }
    }
    
    return left;  // 峰值位置
}
```

### 4. 寻找两个正序数组的中位数

**问题描述**：

给定两个大小分别为 m 和 n 的正序数组，找出两数组的中位数

**解题思路**：

- 二分搜索较短的数组
    
- 时间复杂度 O(log(min(m,n)))
    

```cpp
double findMedianSortedArrays(vector<int>& nums1, vector<int>& nums2) {
    if (nums1.size() > nums2.size()) {
        return findMedianSortedArrays(nums2, nums1);
    }
    
    int m = nums1.size(), n = nums2.size();
    int left = 0, right = m;
    int totalLeft = (m + n + 1) / 2;
    
    while (left <= right) {
        int i = (left + right) / 2;
        int j = totalLeft - i;
        
        if (i < m && nums2[j - 1] > nums1[i]) {
            left = i + 1;  // i 太小
        } else if (i > 0 && nums1[i - 1] > nums2[j]) {
            right = i - 1;  // i 太大
        } else {
            // 找到合适的分割点
            int maxLeft = 0;
            if (i == 0) maxLeft = nums2[j - 1];
            else if (j == 0) maxLeft = nums1[i - 1];
            else maxLeft = max(nums1[i - 1], nums2[j - 1]);
            
            if ((m + n) % 2 == 1) {
                return maxLeft;
            }
            
            int minRight = 0;
            if (i == m) minRight = nums2[j];
            else if (j == n) minRight = nums1[i];
            else minRight = min(nums1[i], nums2[j]);
            
            return (maxLeft + minRight) / 2.0;
        }
    }
    
    return 0.0;
}
```

# 排序算法

## 排序算法概述

### 基本概念

- **排序**：将数据元素按照特定顺序排列的过程
    
- **稳定排序**：相等元素的相对位置保持不变
    
- **不稳定排序**：相等元素的相对位置可能改变
    
- **内部排序**：数据全部在内存中进行排序
    
- **外部排序**：数据量太大，需要借助外部存储
    

### 性能指标

- **时间复杂度**：算法执行时间随数据规模的增长率
    
- **空间复杂度**：算法执行过程中所需的额外存储空间
    
- **稳定性**：排序后相等元素的相对位置是否保持
    

---

## 一、冒泡排序（Bubble Sort）

### 算法思想

通过相邻元素的比较和交换，将较大的元素逐步"冒泡"到数组末端

### 算法步骤

1. 从头开始比较相邻元素
    
2. 如果前一个元素大于后一个元素，则交换它们
    
3. 对每一对相邻元素重复上述操作
    
4. 重复上述过程，每次减少一个比较元素，直到没有任何一对数字需要比较
    

### 代码实现

```cpp
#include <iostream>
#include <vector>
using namespace std;

void bubbleSort(vector<int>& arr) {
    int n = arr.size();
    for (int i = 0; i < n - 1; i++) {
        bool swapped = false;  // 优化标志
        for (int j = 0; j < n - 1 - i; j++) {
            if (arr[j] > arr[j + 1]) {
                swap(arr[j], arr[j + 1]);
                swapped = true;
            }
        }
        // 如果没有发生交换，说明已经有序
        if (!swapped) break;
    }
}

// 测试代码
int main() {
    vector<int> arr = {64, 34, 25, 12, 22, 11, 90};
    bubbleSort(arr);
    
    cout << "Sorted array: ";
    for (int num : arr) {
        cout << num << " ";
    }
    cout << endl;
    
    return 0;
}
```

### 复杂度分析

- **时间复杂度**：
    
    - 最好：O(n)（已有序，一次遍历）
        
    - 平均：O(n²)
        
    - 最坏：O(n²)（逆序）
        
- **空间复杂度**：O(1)
    
- **稳定性**：稳定
    

### 优缺点

|**优点**|**缺点**|
|:--|:--|
|实现简单，容易理解|效率低，不适合大数据量|
|稳定排序|时间复杂度高|
|原地排序，空间开销小|实际工程中很少使用|

## 二、选择排序（Selection Sort）

### 算法思想

每次从未排序部分选择最小（或最大）元素，放到已排序部分的末尾

### 算法步骤

1. 在未排序序列中找到最小元素
    
2. 将其存放到排序序列的起始位置
    
3. 从剩余未排序元素中继续寻找最小元素
    
4. 重复上述过程，直到所有元素均排序完毕
    

### 代码实现

```cpp
#include <iostream>
#include <vector>
#include <climits>
using namespace std;

void selectionSort(vector<int>& arr) {
    int n = arr.size();
    
    for (int i = 0; i < n - 1; i++) {
        int min_idx = i;
        
        // 找到未排序部分的最小元素
        for (int j = i + 1; j < n; j++) {
            if (arr[j] < arr[min_idx]) {
                min_idx = j;
            }
        }
        
        // 交换最小元素到已排序部分的末尾
        if (min_idx != i) {
            swap(arr[i], arr[min_idx]);
        }
    }
}

// 测试代码
int main() {
    vector<int> arr = {64, 25, 12, 22, 11};
    selectionSort(arr);
    
    cout << "Sorted array: ";
    for (int num : arr) {
        cout << num << " ";
    }
    cout << endl;
    
    return 0;
}
```

### 复杂度分析

- **时间复杂度**：O(n²)（所有情况）
    
- **空间复杂度**：O(1)
    
- **稳定性**：不稳定（交换可能改变相等元素位置）
    

### 优缺点

|**优点**|**缺点**|
|:--|:--|
|实现简单|效率低，时间复杂度高|
|原地排序|不稳定排序|
|交换次数少（最多n-1次）|不适合大数据量|

## 三、插入排序（Insertion Sort）

### 算法思想

将未排序元素逐个插入到已排序部分的正确位置

### 算法步骤

1. 从第一个元素开始，该元素可以认为已经被排序
    
2. 取出下一个元素，在已经排序的元素序列中从后向前扫描
    
3. 如果该元素大于新元素，将该元素移到下一位置
    
4. 重复步骤3，直到找到已排序的元素小于或者等于新元素的位置
    
5. 将新元素插入到该位置后
    
6. 重复步骤2~5
    

### 代码实现

```cpp
#include <iostream>
#include <vector>
using namespace std;

void insertionSort(vector<int>& arr) {
    int n = arr.size();
    
    for (int i = 1; i < n; i++) {
        int key = arr[i];
        int j = i - 1;
        
        // 将大于key的元素向后移动
        while (j >= 0 && arr[j] > key) {
            arr[j + 1] = arr[j];
            j--;
        }
        arr[j + 1] = key;  // 插入key到正确位置
    }
}

// 测试代码
int main() {
    vector<int> arr = {12, 11, 13, 5, 6};
    insertionSort(arr);
    
    cout << "Sorted array: ";
    for (int num : arr) {
        cout << num << " ";
    }
    cout << endl;
    
    return 0;
}
```

### 复杂度分析

- **时间复杂度**：
    
    - 最好：O(n)（已有序）
        
    - 平均：O(n²)
        
    - 最坏：O(n²)（逆序）
        
- **空间复杂度**：O(1)
    
- **稳定性**：稳定
    

### 优缺点

|**优点**|**缺点**|
|:--|:--|
|实现简单|效率低，时间复杂度高|
|稳定排序|不适合大数据量|
|对小规模数据效果好|-|
|原地排序|-|
|对几乎有序的数据效率高|-|

## 四、希尔排序（Shell Sort）

### 算法思想

改进的插入排序，通过将数组分成若干子序列进行插入排序，逐步减小间隔直至为1

### 算法步骤

1. 选择一个增量序列 t₁, t₂, ..., tk，其中 ti > tj (i < j)，tk = 1
    
2. 按增量序列个数k，对序列进行k趟排序
    
3. 每趟排序，根据对应增量ti，将待排序列分割成若干长度为m的子序列
    
4. 分别对各子表进行直接插入排序
    
5. 仅增量因子为1时，整个序列作为一个表来处理，表长度即为整个序列的长度
    

### 代码实现

```cpp
#include <iostream>
#include <vector>
using namespace std;

void shellSort(vector<int>& arr) {
    int n = arr.size();
    
    // 使用Knuth序列：h = 3*h + 1
    int gap = 1;
    while (gap < n / 3) {
        gap = 3 * gap + 1;
    }
    
    while (gap >= 1) {
        // 对每个子序列进行插入排序
        for (int i = gap; i < n; i++) {
            int temp = arr[i];
            int j;
            for (j = i; j >= gap && arr[j - gap] > temp; j -= gap) {
                arr[j] = arr[j - gap];
            }
            arr[j] = temp;
        }
        gap /= 3;
    }
}

// 测试代码
int main() {
    vector<int> arr = {12, 34, 54, 2, 3};
    shellSort(arr);
    
    cout << "Sorted array: ";
    for (int num : arr) {
        cout << num << " ";
    }
    cout << endl;
    
    return 0;
}
```

### 复杂度分析

- **时间复杂度**：
    
    - 最好：O(n log n)
        
    - 平均：取决于增量序列，约为O(n¹·³)
        
    - 最坏：O(n²)
        
- **空间复杂度**：O(1)
    
- **稳定性**：不稳定
    

### 优缺点

|**优点**|**缺点**|
|:--|:--|
|比插入排序效率高|不稳定排序|
|原地排序|实现复杂|
|适合中等规模数据|增量序列选择影响性能|

## 五、快速排序（Quick Sort）

### 算法思想

分治法：选择一个基准元素，将数组分为两部分，左边都小于基准，右边都大于基准，然后递归处理

### 算法步骤

1. 选择基准元素（pivot）
    
2. 分区操作：重新排列数组，所有小于基准的元素放在基准前面，大于基准的放在后面
    
3. 递归地对两个子数组进行快速排序
    

### 代码实现（基础版）

```cpp
#include <iostream>
#include <vector>
using namespace std;

// 分区函数
int partition(vector<int>& arr, int low, int high) {
    int pivot = arr[high];  // 选择最后一个元素作为基准
    int i = low - 1;        // 小于基准的元素的边界
    
    for (int j = low; j < high; j++) {
        if (arr[j] < pivot) {
            i++;
            swap(arr[i], arr[j]);
        }
    }
    swap(arr[i + 1], arr[high]);
    return i + 1;
}

// 快速排序递归函数
void quickSortHelper(vector<int>& arr, int low, int high) {
    if (low < high) {
        int pi = partition(arr, low, high);
        quickSortHelper(arr, low, pi - 1);
        quickSortHelper(arr, pi + 1, high);
    }
}

// 快速排序入口函数
void quickSort(vector<int>& arr) {
    quickSortHelper(arr, 0, arr.size() - 1);
}

// 测试代码
int main() {
    vector<int> arr = {10, 7, 8, 9, 1, 5};
    quickSort(arr);
    
    cout << "Sorted array: ";
    for (int num : arr) {
        cout << num << " ";
    }
    cout << endl;
    
    return 0;
}
```

### 复杂度分析

- **时间复杂度**：
    
    - 最好：O(n log n)
        
    - 平均：O(n log n)
        
    - 最坏：O(n²)（已排序或逆序）
        
- **空间复杂度**：O(log n)（递归栈）
    
- **稳定性**：不稳定

```cpp
int medianOfThree(vector<int>& arr, int low, int high) {

    // 对三个数进行排序
    if (arr[low] > arr[mid])
        swap(arr[low], arr[mid]);
    if (arr[low] > arr[high])
        swap(arr[low], arr[high]);
    if (arr[mid] > arr[high])
```

### 3. 三路快排（处理重复元素）

```cpp
void quickSort3Way(vector<int>& arr, int low, int high) {
    if (low >= high) return;
    
    int lt = low, gt = high;
    int pivot = arr[low];
    int i = low;
    
    while (i <= gt) {
        if (arr[i] < pivot) {
            swap(arr[lt], arr[i]);
            lt++;
            i++;
        } else if (arr[i] > pivot) {
            swap(arr[i], arr[gt]);
            gt--;
        } else {
            i++;
        }
    }
    
    // 递归处理两个子数组
    quickSort3Way(arr, low, lt - 1);
    quickSort3Way(arr, gt + 1, high);
}
```

### 4. 尾递归优化

```cpp
void quickSortOptimized(vector<int>& arr, int low, int high) {
    while (low < high) {
        int pi = partition(arr, low, high);
        
        // 递归处理较小的子数组，迭代处理较大的子数组
        if (pi - low < high - pi) {
            quickSortOptimized(arr, low, pi - 1);
            low = pi + 1;
        } else {
            quickSortOptimized(arr, pi + 1, high);
            high = pi - 1;
        }
    }
}
```

## 七、归并排序（Merge Sort）

### 算法思想

分治法：将数组分成两半，分别排序，然后合并两个有序数组

### 算法步骤

1. 分解：将当前区间一分为二
    
2. 递归：递归地对两个子区间进行归并排序
    
3. 合并：将两个有序子区间合并成一个有序区间
    

### 代码实现

```cpp
#include <iostream>
#include <vector>
using namespace std;

// 合并两个有序数组
void merge(vector<int>& arr, int left, int mid, int right) {
    int n1 = mid - left + 1;
    int n2 = right - mid;
    
    // 创建临时数组
    vector<int> L(n1), R(n2);
    
    // 复制数据到临时数组
    for (int i = 0; i < n1; i++)
        L[i] = arr[left + i];
    for (int j = 0; j < n2; j++)
        R[j] = arr[mid + 1 + j];
    
    // 合并临时数组
    int i = 0, j = 0, k = left;
    while (i < n1 && j < n2) {
        if (L[i] <= R[j]) {
            arr[k] = L[i];
            i++;
        } else {
            arr[k] = R[j];
            j++;
        }
        k++;
    }
    
    // 复制剩余元素
    while (i < n1) {
        arr[k] = L[i];
        i++;
        k++;
    }
    while (j < n2) {
        arr[k] = R[j];
        j++;
        k++;
    }
}

// 归并排序递归函数
void mergeSortHelper(vector<int>& arr, int left, int right) {
    if (left < right) {
        int mid = left + (right - left) / 2;
        
        mergeSortHelper(arr, left, mid);
        mergeSortHelper(arr, mid + 1, right);
        
        merge(arr, left, mid, right);
    }
}

// 归并排序入口函数
void mergeSort(vector<int>& arr) {
```

## 八、基数排序（Radix Sort）

### 算法思想

按照低位先排序，然后收集；再按照高位排序，然后再收集；依次类推，直到最高位

### 算法步骤

1. 找到数组中最大数字的位数
    
2. 从最低位到最高位，依次进行计数排序
    
3. 每次排序都是稳定的
    

### 代码实现

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

// 获取数字的某一位
int getDigit(int num, int digit) {
    return (num / (int)pow(10, digit)) % 10;
}

// 基数排序
void radixSort(vector<int>& arr) {
    if (arr.empty()) return;
    
    // 找到最大值
    int max_val = *max_element(arr.begin(), arr.end());
    int max_digits = 0;
    while (max_val > 0) {
        max_digits++;
        max_val /= 10;
    }
    
    // 桶的数量（0-9）
    const int RADIX = 10;
    vector<vector<int>> buckets(RADIX);
    
    // 从最低位到最高位
    for (int digit = 0; digit < max_digits; digit++) {
        // 清空桶
        for (int i = 0; i < RADIX; i++) {
            buckets[i].clear();
        }
        
        // 分配到桶中
        for (int num : arr) {
            int d = getDigit(num, digit);
            buckets[d].push_back(num);
        }
        
        // 从桶中收集元素
        int index = 0;
        for (int i = 0; i < RADIX; i++) {
            for (int num : buckets[i]) {
                arr[index++] = num;
            }
        }
    }
}

// 测试代码
int main() {
    vector<int> arr = {170, 45, 75, 90, 802, 24, 2, 66};
    radixSort(arr);
    
    cout << "Sorted array: ";
    for (int num : arr) {
        cout << num << " ";
    }
    cout << endl;
    
    return 0;
}
```

### 复杂度分析

- **时间复杂度**：O(d × (n + k))，其中d是最大位数，k是基数
    
- **空间复杂度**：O(n + k)
    
- **稳定性**：稳定

## 九、堆排序（Heap Sort）

### 算法思想

利用堆这种数据结构所设计的一种排序算法，分为构建堆和提取最大元素两个阶段

### 算法步骤

1. 构建最大堆（Build Max Heap）：将待排序序列构造成一个大顶堆
    
2. 交换堆顶元素与末尾元素：将堆顶元素与末尾元素进行交换
    
3. 调整堆：重新调整结构，使其满足堆定义
    
4. 重复步骤2~3，直到整个序列有序
    

### 代码实现

```cpp
#include <iostream>
#include <vector>
using namespace std;

// 调整堆
void heapify(vector<int>& arr, int n, int i) {
    int largest = i;        // 初始化最大值为根节点
    int left = 2 * i + 1;   // 左子节点
    int right = 2 * i + 2;  // 右子节点
    
    // 如果左子节点大于根节点
    if (left < n && arr[left] > arr[largest])
        largest = left;
    
    // 如果右子节点大于当前最大值
    if (right < n && arr[right] > arr[largest])
        largest = right;
    
    // 如果最大值不是根节点
    if (largest != i) {
        swap(arr[i], arr[largest]);
        // 递归调整受影响的子树
        heapify(arr, n, largest);
    }
}

// 堆排序
void heapSort(vector<int>& arr) {
    int n = arr.size();
    
    // 构建最大堆
    for (int i = n / 2 - 1; i >= 0; i--)
        heapify(arr, n, i);
    
    // 一个个从堆顶取出元素
    for (int i = n - 1; i > 0; i--) {
        swap(arr[0], arr[i]);  // 将当前最大值移到末尾
        heapify(arr, i, 0);     // 调整剩余元素为最大堆
    }
}

// 测试代码
int main() {
    vector<int> arr = {12, 11, 13, 5, 6, 7};
    heapSort(arr);
    
    cout << "Sorted array: ";
    for (int num : arr) {
        cout << num << " ";
    }
    cout << endl;
    
    return 0;
}
```

### 复杂度分析

- **时间复杂度**：O(n log n)（所有情况）
    
- **空间复杂度**：O(1)
    
- **稳定性**：不稳定

## 一、查重算法

### 1. 哈希表查重

#### 算法思想

- 利用哈希表的O(1)查找特性
    
- 将元素存入哈希表，遇到重复时检测
    
- 适用于精确查重
    

#### 实现代码

```cpp
#include <iostream>
#include <unordered_set>
#include <vector>
#include <string>
using namespace std;

class HashDeduplication {
private:
    unordered_set<string> hashSet;

public:
    // 添加元素并检查重复
    bool add(const string& item) {
        if (hashSet.find(item) != hashSet.end()) {
            return false;  // 重复元素
        }
        hashSet.insert(item);
        return true;       // 新元素
    }
    
    // 批量查重
    vector<string> findDuplicates(const vector<string>& items) {
        vector<string> duplicates;
        unordered_set<string> seen;
        
        for (const string& item : items) {
            if (seen.find(item) != seen.end()) {
                duplicates.push_back(item);
            } else {
                seen.insert(item);
            }
        }
        return duplicates;
    }
    
    // 获取唯一元素数量
    int uniqueCount() {
        return hashSet.size();
    }
};

// 测试代码
int main() {
    HashDeduplication dedup;
    vector<string> data = {"apple", "banana", "apple", "orange", "banana", "grape"};
    
    cout << "Original data: ";
    for (const string& item : data) cout << item << " ";
    cout << endl;
    
    vector<string> duplicates = dedup.findDuplicates(data);
    cout << "Duplicates: ";
    for (const string& dup : duplicates) cout << dup << " ";
    cout << endl;
    
    return 0;
}
```

#### 复杂度分析

- **时间复杂度**：O(n)
    
- **空间复杂度**：O(n)
    
- **优点**：精确查重，实现简单
    
- **缺点**：内存消耗大

### 2. 位图查重（Bitmap）

位图，就是用一个位（0/1）来存储数据的状态，适合状态简单，数据量大，要求内存使用率低的问题场景

很大的缺点：数据没有多少，但最大值却很大，这样非常浪费存储空间

#### 算法思想

- 使用二进制位表示元素是否存在
    
- 适用于整数范围已知且集中的场景
    
- 极大节省内存空间
    

#### 实现代码

```cpp
#include <iostream>
#include <vector>
#include <bitset>
#include <cmath>
using namespace std;

class BitmapDeduplication {
private:
    vector<unsigned char> bitmap;
    int size;
    
public:
    BitmapDeduplication(int max_num) {
        size = (max_num + 7) / 8;  
        // 计算需要的字节数
        bitmap.resize(size, 0);
    }
    
    // 设置位
    void set(int num) {
        int byte_idx = num / 8;
        int bit_idx = num % 8;
        if (byte_idx < size) {
            bitmap[byte_idx] |= (1 << 
            bit_idx);
        }
    }
    
    // 检查位
    bool check(int num) {
        int byte_idx = num / 8;
        int bit_idx = num % 8;
        if (byte_idx >= size) return false;
        return (bitmap[byte_idx] & (1 << 
        bit_idx)) != 0;
    }
    
    // 添加元素
    bool add(int num) {
        if (check(num)) {
            return false;  // 重复
        }
        set(num);
        return true;     // 新元素
    }
};

// 测试代码
int main() {
    BitmapDeduplication bitmap(100);  
    // 支持0-99的数字
    
    vector<int> data = {1, 3, 5, 7, 1, 3, 9, 11};
    
    cout << "Original data: ";
    for (int num : data) 
	    cout << num << " ";
    cout << endl;
    
    cout << "Unique elements: ";
    for (int num : data) {
        if (bitmap.add(num)) {
            cout << num << " ";
        }
    }
    cout << endl;
    
    return 0;
}
```

### 3. 布隆过滤器（Bloom Filter）

#### 算法思想

- 使用多个哈希函数映射到位向量
    
- 可能存在假阳性（误报），但无假阴性
    
- 适用于大规模数据的初步筛选
    

#### 实现代码

```cpp
#include <iostream>
#include <vector>
#include <string>
#include <functional>
using namespace std;

class BloomFilter {
private:
    vector<bool> bits;
    int size;
    int hash_count;
    
    // 哈希函数
    unsigned int hash1(const string& key) {
        unsigned int hash = 5381;
        for (char c : key) {
            hash = ((hash << 5) + hash) + c;
        }
        return hash;
    }
    
    unsigned int hash2(const string& key) {
        unsigned int hash = 0;
        for (char c : key) {
            hash = 31 * hash + c;
        }
        return hash;
    }
    
    unsigned int hash3(const string& key) {
        unsigned int hash = 0;
        for (char c : key) {
            hash = hash * 131 + c;
        }
        return hash;
    }

public:
    BloomFilter(int n, double p) {
        // 计算最优大小和哈希函数数量
        size = (int)(-(n * log(p)) / (log(2) * log(2)));
        hash_count = (int)((size / n) * log(2));
        bits.resize(size, false);
    }
    
    // 添加元素
    void add(const string& item) {
        if (hash_count >= 3) {
            bits[hash1(item) % size] = true;
            bits[hash2(item) % size] = true;
            bits[hash3(item) % size] = true;
        }
    }
    
    // 检查元素（可能存在假阳性）
    bool contains(const string& item) {
        return bits[hash1(item) % size] && 
               bits[hash2(item) % size] && 
               bits[hash3(item) % size];
    }
};

// 测试代码
int main() {
    BloomFilter bf(1000, 0.01);  // 预期1000个元素，误判率1%
    
    vector<string> data = {"apple", "banana", "cherry", "date", "elderberry"};
    
    // 添加元素
    for (const string& item : data) {
        bf.add(item);
    }
    
    // 检查元素
    cout << "Contains 'apple': " << bf.contains("apple") << endl;    // true
    cout << "Contains 'fig': " << bf.contains("fig") << endl;        // 可能是false
    cout << "Contains 'banana': " << bf.contains("banana") << endl;  // true
    
    return 0;
}
```

```cpp
class AdvancedBloomFilter {
private:
    bitset<1000000> bits;
    int hash_count;
    vector<function<unsigned int(string)>> hash_functions;
    
public:
    AdvancedBloomFilter(int k = 3) : hash_count(k) {
        // 初始化多个哈希函数
        hash_functions.push_back([](string key) {
            unsigned int hash = 5381;
            for (char c : key) hash = ((hash << 5) + hash) + c;
            return hash;
        });
        hash_functions.push_back([](string key) {
            unsigned int hash = 0;
            for (char c : key) hash = 31 * hash + c;
            return hash;
        });
        hash_functions.push_back([](string key) {
            unsigned int hash = 0;
            for (char c : key) hash = hash * 131 + c;
            return hash;
        });
    }
    
    void add(const string& item) {
        for (int i = 0; i < hash_count; i++) {
            bits.set(hash_functions[i](item) % bits.size());
        }
    }
    
    bool contains(const string& item) {
        for (int i = 0; i < hash_count; i++) {
            if (!bits.test(hash_functions[i](item) % bits.size())) {
                return false;
            }
        }
        return true;
    }
};
```

- **优点**：空间效率极高，查询速度快
    
- **缺点**：有误判率，不能删除元素

## 二、Top K问题

### 1. 小根堆解决Top K问题

#### 算法思想

- 维护大小为K的小根堆
    
- 遍历所有元素，保持堆中为最大的K个元素
    
- 堆顶即为第K大的元素
    

#### 实现代码

```cpp
#include <iostream>
#include <vector>
#include <queue>
#include <algorithm>
using namespace std;

class TopKFinder {
public:
    // 找最大的K个元素
    static vector<int> findTopK(vector<int>& nums, int k) {
        priority_queue<int, vector<int>, greater<int>> min_heap;
        
        for (int num : nums) {
            if (min_heap.size() < k) {
                min_heap.push(num);
            } else if (num > min_heap.top()) {
                min_heap.pop();
                min_heap.push(num);
            }
        }
        
        // 提取结果
        vector<int> result;
        while (!min_heap.empty()) {
            result.push_back(min_heap.top());
            min_heap.pop();
        }
        reverse(result.begin(), result.end());  // 从大到小排序
        return result;
    }
    
    // 找最小的K个元素（使用大根堆）
    static vector<int> findBottomK(vector<int>& nums, int k) {
        priority_queue<int> max_heap;
        
        for (int num : nums) {
            if (max_heap.size() < k) {
                max_heap.push(num);
            } else if (num < max_heap.top()) {
                max_heap.pop();
                max_heap.push(num);
            }
        }
        
        vector<int> result;
        while (!max_heap.empty()) {
            result.push_back(max_heap.top());
            max_heap.pop();
        }
        return result;
    }
};

// 测试代码
int main() {
    vector<int> nums = {3, 2, 1, 5, 6, 4, 8, 7, 9};
    int k = 3;
    
    vector<int> top_k = TopKFinder::findTopK(nums, k);
    cout << "Top " << k << " largest: ";
    for (int num : top_k) cout << num << " ";
    cout << endl;
    
    vector<int> bottom_k = TopKFinder::findBottomK(nums, k);
    cout << "Top " << k << " smallest: ";
    for (int num : bottom_k) cout << num << " ";
    cout << endl;
    
    return 0;
}
```

#### 手动实现小根堆

```cpp
class MinHeap {
private:
    vector<int> data;
    
    void swim(int i) {
        while (i > 0 && data[(i-1)/2] > data[i]) {
            swap(data[(i-1)/2], data[i]);
            i = (i-1)/2;
        }
    }
    
    void sink(int i, int n) {
        while (2*i+1 < n) {
            int j = 2*i+1;
            if (j+1 < n && data[j+1] < data[j]) j++;
            if (data[i] <= data[j]) break;
            swap(data[i], data[j]);
            i = j;
        }
    }

public:
    void push(int val) {
        data.push_back(val);
        swim(data.size()-1);
    }
    
    int top() {
        return data[0];
    }
    
    void pop() {
        data[0] = data.back();
        data.pop_back();
        sink(0, data.size());
    }
    
    int size() {
        return data.size();
    }
};
```

- **时间复杂度**：O(n log k)
    
- **空间复杂度**：O(k)
    
- **优点**：适合大数据流，内存占用小
    
- **缺点**：需要维护堆结构

### 2. 快速选择算法（快排分割）

#### 实现代码

```cpp
#include <iostream>
#include <vector>
#include <algorithm>
#include <cstdlib>
#include <ctime>
using namespace std;

class QuickSelect {
public:
    // 找第K大的元素
    static int findKthLargest(vector<int>& nums, int k) {
        int left = 0, right = nums.size() - 1;
        k = nums.size() - k;  // 转换为第K小
        
        while (left <= right) {
            int pivot_idx = partition(nums, left, right);
            if (pivot_idx == k) {
                return nums[pivot_idx];
            } else if (pivot_idx < k) {
                left = pivot_idx + 1;
            } else {
                right = pivot_idx - 1;
            }
        }
        return -1;
    }
    
    // 找第K小的元素
    static int findKthSmallest(vector<int>& nums, int k) {
        int left = 0, right = nums.size() - 1;
        
        while (left <= right) {
            int pivot_idx = partition(nums, left, right);
            if (pivot_idx == k - 1) {
                return nums[pivot_idx];
            } else if (pivot_idx < k - 1) {
                left = pivot_idx + 1;
            } else {
                right = pivot_idx - 1;
            }
        }
        return -1;
    }
    
private:
    // Lomuto分区方案
    static int partition(vector<int>& nums, int left, int right) {
        int pivot = nums[right];
        int i = left;
        
        for (int j = left; j < right; j++) {
            if (nums[j] <= pivot) {
                swap(nums[i], nums[j]);
                i++;
            }
        }
        swap(nums[i], nums[right]);
        return i;
    }
    
    // Hoare分区方案（更高效）
    static int hoarePartition(vector<int>& nums, int left, int right) {
        int pivot = nums[left + (right - left) / 2];
        int i = left - 1, j = right + 1;
        
        while (true) {
            do { i++; } while (nums[i] < pivot);
            do { j--; } while (nums[j] > pivot);
            if (i >= j) return j;
            swap(nums[i], nums[j]);
        }
    }
};

// 找最大的K个元素
vector<int> findTopKWithQuickSelect(vector<int>& nums, int k) {
    int kth_largest = QuickSelect::findKthLargest(nums, k);
    vector<int> result;
    int count = 0;
    
    for (int num : nums) {
        if (num >= kth_largest && count < k) {
            result.push_back(num);
            count++;
        }
    }
    return result;
}

// 测试代码
int main() {
    vector<int> nums = {3, 2, 1, 5, 6, 4, 8, 7, 9};
    int k = 3;
    
    int kth_largest = QuickSelect::findKthLargest(nums, k);
    cout << k << "rd largest element: " << kth_largest << endl;
    
    int kth_smallest = QuickSelect::findKthSmallest(nums, k);
    cout << k << "rd smallest element: " << kth_smallest << endl;
    
    vector<int> top_k = findTopKWithQuickSelect(nums, k);
    cout << "Top " << k << " largest: ";
    for (int num : top_k) cout << num << " ";
    cout << endl;
    
    return 0;
}
```

```cpp
#include <thread>
#include <mutex>
class ParallelTopK {
private:
    mutex mtx;
    priority_queue<int> global_topk;
    
    void processChunk(const vector<int>& chunk, int k) {
        priority_queue<int, vector<int>, greater<int>> local_heap;
        
        for (int num : chunk) {
            if (local_heap.size() < k) {
                local_heap.push(num);
            } else if (num > local_heap.top()) {
                local_heap.pop();
                local_heap.push(num);
            }
        }
        
        lock_guard<mutex> lock(mtx);
        while (!local_heap.empty()) {
            int val = local_heap.top();
            local_heap.pop();
            if (global_topk.size() < k) {
                global_topk.push(val);
            } else if (val > global_topk.top()) {
                global_topk.pop();
                global_topk.push(val);
            }
        }
    }
    
public:
    vector<int> findTopKParallel(const vector<int>& data, int k, int num_threads) {
        vector<thread> threads;
        int chunk_size = data.size() / num_threads;
        
        for (int i = 0; i < num_threads; i++) {
            int start = i * chunk_size;
            int end = (i == num_threads - 1) ? data.size() : (i + 1) * chunk_size;
            vector<int> chunk(data.begin() + start, data.begin() + end);
            
            threads.emplace_back(&ParallelTopK::processChunk, this, chunk, k);
        }
        
        for (auto& t : threads) {
            t.join();
        }
        
        // 提取结果
        vector<int> result;
        while (!global_topk.empty()) {
            result.push_back(global_topk.top());
            global_topk.pop();
        }
        reverse(result.begin(), result.end());
        return result;
    }
};
```

## 一、BST树区间搜索问题

### 问题描述

给定一棵BST和一个区间[L, R]，找出所有值在[L, R]范围内的节点

### 算法思路

- 利用BST的性质：左子树 < 根 < 右子树
    
- 如果当前节点值 < L，只需搜索右子树
    
- 如果当前节点值 > R，只需搜索左子树
    
- 如果当前节点值在[L, R]内，搜索左右子树
    

### 代码实现

```cpp
#include <iostream>
#include <vector>
using namespace std;

struct TreeNode {
    int val;
    TreeNode* left;
    TreeNode* right;
    TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
};

// 区间搜索
void rangeSearch(TreeNode* root, int L, int R, vector<int>& result) {
    if (!root) return;
    
    if (root->val > L) {
        rangeSearch(root->left, L, R, result);  // 左子树可能有符合条件的节点
    }
    
    if (root->val >= L && root->val <= R) {
        result.push_back(root->val);  // 当前节点符合条件
    }
    
    if (root->val < R) {
        rangeSearch(root->right, L, R, result);  // 右子树可能有符合条件的节点
    }
}

// 非递归版本
vector<int> rangeSearchIterative(TreeNode* root, int L, int R) {
    vector<int> result;
    if (!root) return result;
    
    stack<TreeNode*> st;
    TreeNode* curr = root;
    
    while (curr || !st.empty()) {
        while (curr) {
            st.push(curr);
            if (curr->val > L) {
                curr = curr->left;  // 可能还有更小的节点在区间内
            } else {
                curr = nullptr;  // 停止向左
            }
        }
        
        curr = st.top();
        st.pop();
        
        if (curr->val >= L && curr->val <= R) {
            result.push_back(curr->val);
        }
        
        if (curr->val < R) {
            curr = curr->right;
        } else {
            curr = nullptr;  // 停止向右
        }
    }
    
    return result;
}

// 测试代码
void testRangeSearch() {
    // 构建BST: [10, 5, 15, 3, 7, null, 18]
    TreeNode* root = new TreeNode(10);
    root->left = new TreeNode(5);
    root->right = new TreeNode(15);
    root->left->left = new TreeNode(3);
    root->left->right = new TreeNode(7);
    root->right->right = new TreeNode(18);
    
    vector<int> result;
    rangeSearch(root, 7, 15, result);
    
    cout << "区间[7,15]内的节点: ";
    for (int val : result) cout << val << " ";  // 输出: 7 10 15
    cout << endl;
}
```

## 二、判断二叉树是否是BST树

### 问题描述

判断给定的二叉树是否满足BST的性质

### 算法思路1：中序遍历验证

- BST的中序遍历结果是严格递增的
    
- 进行一次中序遍历，检查是否递增
    

### 代码实现1

```cpp
bool isValidBST_Inorder(TreeNode* root) {
    TreeNode* prev = nullptr;
    return inorderCheck(root, prev);
}

bool inorderCheck(TreeNode* root, TreeNode*& prev) {
    if (!root) return true;
    
    if (!inorderCheck(root->left, prev)) return false;
    
    if (prev && prev->val >= root->val) return false;
    prev = root;
    
    return inorderCheck(root->right, prev);
}
```

### 算法思路2：递归验证范围

- 每个节点都有一个有效范围(min, max)
    
- 根节点范围(-∞, +∞)
    
- 左子树范围(min, root->val)
    
- 右子树范围(root->val, max)
    

### 代码实现2

```cpp
bool isValidBST_Range(TreeNode* root) {
    return validate(root, LONG_MIN, LONG_MAX);
}

bool validate(TreeNode* root, long min_val, long max_val) {
    if (!root) return true;
    if (root->val <= min_val || root->val >= max_val) return false;
    
    return validate(root->left, min_val, root->val) && 
           validate(root->right, root->val, max_val);
}

// 非递归版本
bool isValidBST_Iterative(TreeNode* root) {
    if (!root) return true;
    
    stack<TreeNode*> st;
    TreeNode* curr = root;
    long prev_val = LONG_MIN;
    
    while (curr || !st.empty()) {
        while (curr) {
            st.push(curr);
            curr = curr->left;
        }
        
        curr = st.top();
        st.pop();
        
        if (curr->val <= prev_val) return false;
        prev_val = curr->val;
        
        curr = curr->right;
    }
    
    return true;
}
```

## 四、求LCA最近公共祖先节点

### 问题描述

在BST或普通二叉树中找到两个节点的最近公共祖先

### BST中的LCA

```cpp
// BST的最近公共祖先
TreeNode* lowestCommonAncestor_BST(TreeNode* root, TreeNode* p, TreeNode* q) {
    if (!root || !p || !q) return nullptr;
    
    // 如果p和q都在左子树
    if (p->val < root->val && q->val < root->val) {
        return lowestCommonAncestor_BST(root->left, p, q);
    }
    // 如果p和q都在右子树
    if (p->val > root->val && q->val > root->val) {
        return lowestCommonAncestor_BST(root->right, p, q);
    }
    // 否则当前节点就是LCA
    return root;
}
```

```cpp
// 普通二叉树的LCA
TreeNode* lowestCommonAncestor_BinaryTree(TreeNode* root, TreeNode* p, TreeNode* q) {
    if (!root || root == p || root == q) return root;
    
    TreeNode* left = lowestCommonAncestor_BinaryTree(root->left, p, q);
    TreeNode* right = lowestCommonAncestor_BinaryTree(root->right, p, q);
    
    if (left && right) return root;  // p和q分别在左右子树
    return left ? left : right;      // 都在左子树或右子树
```

### 算法思路

- 比较左子树的右子树与右子树的左子树
    

### 代码实现

```cpp
bool isSymmetric(TreeNode* root) {
    if (!root) return true;

bool isMirror(TreeNode* t1, TreeNode* t2) {
    if (!t1 && !t2) return true;
    if (!t1 || !t2) return false;
    
    return (t1->val == t2->val) && 
           isMirror(t1->left, t2->right) && 
           isMirror(t1->right, t2->left);

// 非递归版本
bool isSymmetricIterative(TreeNode* root) {
    if (!root) return true;
    
    queue<TreeNode*> q1, q2;
    q1.push(root->left);
    q2.push(root->right);
    
    while (!q1.empty() && !q2.empty()) {
        TreeNode* node1 = q1.front(); q1.pop();
        TreeNode* node2 = q2.front(); q2.pop();
        
        if (!node1 && !node2) continue;
        if (!node1 || !node2) return false;
        if (node1->val != node2->val) return false;
        
        q1.push(node1->left);
        q1.push(node1->right);
        q2.push(node2->right);
        q2.push(node2->left);
    }
    
    return true;
}
```

## 七、前序遍历和中序遍历重建二叉树

### 问题描述

根据前序遍历和中序遍历的结果重建二叉树

### 算法思路

- 前序遍历的第一个元素是根节点
    
- 在中序遍历中找到根节点，左边是左子树，右边是右子树
    
- 递归构建左右子树
    

### 代码实现

```cpp
TreeNode* buildTree(vector<int>& preorder, vector<int>& inorder) {
    return buildTreeHelper(preorder, 0, preorder.size() - 1,
                         inorder, 0, inorder.size() - 1);
}

TreeNode* buildTreeHelper(vector<int>& preorder, int pre_start, int pre_end,
                         vector<int>& inorder, int in_start, int in_end) {
    if (pre_start > pre_end || in_start > in_end) return nullptr;
    
    int root_val = preorder[pre_start];
    TreeNode* root = new TreeNode(root_val);
    
    // 找到根节点在中序遍历中的位置
    int root_idx = in_start;
    while (root_idx <= in_end && inorder[root_idx] != root_val) {
        root_idx++;
    }
    
    int left_size = root_idx - in_start;
    
    root->left = buildTreeHelper(preorder, pre_start + 1, pre_start + left_size,
                                inorder, in_start, root_idx - 1);
    root->right = buildTreeHelper(preorder, pre_start + left_size + 1, pre_end,
                                 inorder, root_idx + 1, in_end);
    
    return root;
}
```

## 八、判断二叉树是否是平衡树

### 问题描述

判断二叉树是否是高度平衡的（任意节点的左右子树高度差不超过1）

### 算法思路

- 计算每个节点的高度
    
- 检查左右子树高度差
    
- 递归验证所有节点
    

### 代码实现

```cpp
bool isBalanced(TreeNode* root) {
    return checkHeight(root) != -1;
}

int checkHeight(TreeNode* root) {
    if (!root) return 0;
    
    int left_height = checkHeight(root->left);
    if (left_height == -1) return -1;
    
    int right_height = checkHeight(root->right);
    if (right_height == -1) return -1;
    
    if (abs(left_height - right_height) > 1) return -1;
    
    return max(left_height, right_height) + 1;
}
```

```cpp
    TreeNode* result = findKthReverseInorder(root->right, k, count);
    if (result) return result;
    
    // 访问当前节点
    count++;
    if (count == k) return root;

    return findKthReverseInorder(root->left, k, count);
}
```

- 如果右子树节点数+1 == k，当前节点就是答案
    
- 如果右子树节点数 >= k，在右子树中查找
    
- 否则在左子树中查找第(k - 右子树节点数 - 1)个节点

## 十、BST树的构造和析构

### BST类的完整实现

```cpp
class BST {
private:
    TreeNode* root;
    
    TreeNode* insertNode(TreeNode* node, int val) {
        if (!node) return new TreeNode(val);
        if (val < node->val) {
            node->left = insertNode(node->left, val);
        } else if (val > node->val) {
            node->right = insertNode(node->right, val);
        }
        return node;
    }
    
    void deleteNode(TreeNode*& node) {
        if (!node) return;
        
        deleteNode(node->left);
        deleteNode(node->right);
        delete node;
        node = nullptr;
    }
    
    void inorderPrint(TreeNode* node) {
        if (!node) return;
        inorderPrint(node->left);
        cout << node->val << " ";
        inorderPrint(node->right);
    }

public:
    BST() : root(nullptr) {}
    
    ~BST() {
        clear();
    }
    
    void insert(int val) {
        root = insertNode(root, val);
    }
    
    bool search(int val) {
        TreeNode* curr = root;
        while (curr) {
            if (val == curr->val) return true;
            if (val < curr->val) curr = curr->left;
            else curr = curr->right;
        }
        return false;
    }
    
    void remove(int val) {
        root = removeNode(root, val);
    }
    
    void clear() {
        deleteNode(root);
    }
    
    void print() {
        inorderPrint(root);
        cout << endl;
    }
    
    TreeNode* getRoot() { return root; }
};
```

## 十三、常见问题解答

### Q1: 什么时候用递归，什么时候用非递归？

- **递归**：代码简洁，易于理解，适合树深度不大的情况
    
- **非递归**：避免栈溢出，性能稍好，适合生产环境
    

### Q2: BST和普通二叉树的区别？

- **BST**：左子树 < 根 < 右子树，支持高效查找
    
- **普通二叉树**：无特定顺序要求
    

### Q3: 如何优化重建二叉树的算法？

使用哈希表存储中序遍历中每个值的索引，可以将时间复杂度从O(n²)降到O(n)

### Q4: LCA还有其他解法吗？

- **倍增法**：预处理每个节点的2^k祖先
    
- **Tarjan离线算法**：配合并查集使用

# 回溯

## 一、回溯算法核心思想

### 1. 定义与本质

**回溯算法**是一种通过**深度优先搜索（DFS）** 探索所有可能解的算法，当发现当前路径无法得到有效解时，会**回退到上一步**（“回溯”）并尝试其他路径。本质是**“试错+回退”**，通过递归或迭代实现，核心是**维护“当前路径”** 和**“剪枝条件”**。

### 2. 核心要素

- **路径**：已做出的选择（如选了哪些数、皇后放在哪）。
    
- **选择列表**：当前可做的选择（如剩余未选的数、棋盘空位）。
    
- **结束条件**：找到一个可行解或全部尝试完毕。
    
- **剪枝**：提前排除不可能得到有效解的路径（优化关键）。
    

### 3. 算法框架（伪代码）

```plaintext
def backtrack(路径, 选择列表):
    if 满足结束条件:
        记录结果
        return
    for 选择 in 选择列表:
        if 选择无效（剪枝）:
            continue
        做选择（加入路径）
        backtrack(新路径, 新选择列表)
        撤销选择（回溯，从路径移除）
```

## 二、经典回溯问题详解

### 1. 整数选择问题（子集和问题）

#### 问题描述

从整数集合 `nums` 中选择若干个数，使其和为 `target`，返回所有可能的子集（不考虑顺序）。

#### 算法思路

- **路径**：当前选择的整数列表。
    
- **选择列表**：剩余未选的整数（避免重复选同一元素）。
    
- **剪枝**：当前和超过 `target` 时停止深入；排序后可通过上下界剪枝（如当前和+剩余最小数>target）。
    

#### 代码实现（C++）

```cpp
#include <vector>
#include <algorithm>
using namespace std;

class SubsetSum {
private:
    vector<vector<int>> result;
    vector<int> path;

    void backtrack(vector<int>& nums, int target, int start, int current_sum) {
        if (current_sum == target) {
            result.push_back(path);
            return;
        }
        if (current_sum > target) return; // 剪枝：和超过目标

        for (int i = start; i < nums.size(); ++i) {
            // 避免重复子集（如[1,2]和[2,1]视为同一子集，通过start控制顺序）
            path.push_back(nums[i]);
            backtrack(nums, target, i + 1, current_sum + nums[i]); // 不选重复元素
            path.pop_back(); // 回溯
        }
    }

public:
    vector<vector<int>> subsetSum(vector<int>& nums, int target) {
        sort(nums.begin(), nums.end()); // 排序便于剪枝
        backtrack(nums, target, 0, 0);
        return result;
    }
};
```

### 2. 2N整数选择问题（剪枝优化）

#### 问题描述

给定 `2N` 个整数，从中选择 `N` 个，使它们的和最大（或满足其他条件），用回溯法+剪枝优化。

#### 算法思路

- **剪枝策略**：
    
    1. 排序后优先选较大数（贪心预处理）；
        
    2. 当前和 + 剩余可选最大数之和 < 当前最优解时，剪枝；
        
    3. 对称性剪枝（如选前k个和后k个等价）。
        

#### 代码实现（Python简化版）

```python
def select_n_max(nums, n):
    nums.sort(reverse=True)  # 降序排序
    total = sum(nums[:n])    # 初始最优解（选前n个最大数）
    best = nums[:n]
    path = []
    
    def backtrack(start, current_sum, selected):
        nonlocal total, best
        if selected == n:
            if current_sum > total:
                total = current_sum
                best = path.copy()
            return
        # 剪枝：剩余数不足（理论上不会发生，因start控制）
        if len(nums) - start < n - selected:
            return
        # 剪枝：当前和 + 剩余最大数之和 <= 当前最优解
        remaining = nums[start:start + (n - selected)]
        if current_sum + sum(remaining) <= total:
            return
        # 选择当前数
        path.append(nums[start])
        backtrack(start + 1, current_sum + nums[start], selected + 1)
        path.pop()
        # 不选当前数（剪枝：若剩余数不够选）
        if len(nums) - (start + 1) >= n - selected:
            backtrack(start + 1, current_sum, selected)
    
    backtrack(0, 0, 0)
    return best
```

### 3. 挑数字问题（组成最大数）

#### 问题描述

从数字集合 `digits` 中挑选 `k` 个数字，使其组成的数最大（如 `[1, 4, 3, 2]` 选2个，最大为 `43`）。

#### 算法思路

- **贪心+回溯**：优先选大数，但需考虑后续可选数的数量（如高位选次大数可能留出更大数给低位）。
    
- **剪枝**：当前路径长度+剩余可选数 < k 时剪枝；当前数小于已选最后一个数且后续有足够数时剪枝（类似单调栈）。
    

#### 代码实现（C++）

```cpp
#include <vector>
#include <string>
#include <algorithm>
using namespace std;

string maxNumber(vector<int>& digits, int k) {
    string res;
    vector<int> path;
    int n = digits.size();
    
    function<void(int)> backtrack = [&](int start) {
        if (path.size() == k) {
            string s;
            for (int num : path) s += to_string(num);
            if (s > res) res = s; // 更新最大数
            return;
        }
        if (start >= n || path.size() + (n - start) < k) return; // 剪枝
        
        // 选择当前数
        path.push_back(digits[start]);
        backtrack(start + 1);
        path.pop_back();
        
        // 不选当前数（剪枝：若当前数较小且后续有足够数）
        if (path.empty() || digits[start] <= path.back()) { 
            backtrack(start + 1);
        }
    };
    
    backtrack(0);
    return res.empty() ? "0" : res;
}
```

### 4. 背包问题（0-1背包）

#### 问题描述

给定 `n` 个物品（重量 `w[i]`，价值 `v[i]`）和背包容量 `C`，选若干物品使总价值最大（不超过容量）。

#### 算法思路

- **路径**：当前选择的物品列表、总重量、总价值。
    
- **剪枝**：
    
    1. 当前重量 > C 时剪枝；
        
    2. 当前价值 + 剩余物品最大价值 ≤ 最优解时剪枝（需预处理后缀最大值）。
        

#### 代码实现（C++）

```cpp
#include <vector>
#include <algorithm>
using namespace std;

class Knapsack {
private:
    int max_value;
    vector<int> best_path;

    void backtrack(vector<int>& w, vector<int>& v, int C, int index, 
                  int current_w, int current_v, vector<int>& path) {
        if (current_w > C) return; // 剪枝：超重
        if (index == w.size()) {
            if (current_v > max_value) {
                max_value = current_v;
                best_path = path;
            }
            return;
        }
        // 不选当前物品
        backtrack(w, v, C, index + 1, current_w, current_v, path);
        // 选当前物品
        path.push_back(index);
        backtrack(w, v, C, index + 1, current_w + w[index], current_v + v[index], path);
        path.pop_back(); // 回溯
    }

public:
    int knapsack(vector<int>& w, vector<int>& v, int C) {
        max_value = 0;
        vector<int> path;
        backtrack(w, v, C, 0, 0, 0, path);
        return max_value;
    }
};
```

## 三、排列树理论与代码

### 1. 排列树定义

**排列树**是回溯中用于生成全排列的树形结构：

- **根节点**：空排列；
    
- **第k层节点**：长度为k的部分排列；
    
- **叶子节点**：长度为n的全排列（共n!个）。
    

### 2. 全排列生成（无重复元素）

#### 算法思路

- 用 `used` 数组标记已选元素，递归生成排列。
    
- 或用**交换法**（更高效）：固定第i个元素，交换i与后续元素，递归生成剩余排列。
    

#### 代码实现（交换法，C++）

```cpp
#include <vector>
#include <iostream>
using namespace std;

class PermutationTree {
private:
    vector<vector<int>> result;

    void swap(int& a, int& b) {
        int temp = a;
        a = b;
        b = temp;
    }

    void backtrack(vector<int>& nums, int start) {
        if (start == nums.size()) {
            result.push_back(nums); // 保存当前排列
            return;
        }
        for (int i = start; i < nums.size(); ++i) {
            swap(nums[start], nums[i]); // 固定第start位为nums[i]
            backtrack(nums, start + 1); // 递归生成剩余排列
            swap(nums[start], nums[i]); // 回溯（恢复原数组）
        }
    }

public:
    vector<vector<int>> permute(vector<int>& nums) {
        backtrack(nums, 0);
        return result;
    }
};

// 测试：输入[1,2,3]，输出6种排列
```

### 3. 含重复元素的全排列（剪枝去重）

#### 算法思路

- 排序后，若当前元素与前一个元素相同且前一个未被使用（或已使用），则跳过（避免重复排列）。
    

#### 代码实现（Python）

```python
def permute_unique(nums):
    nums.sort()
    result = []
    used = [False] * len(nums)
    
    def backtrack(path):
        if len(path) == len(nums):
            result.append(path.copy())
            return
        for i in range(len(nums)):
            if used[i]:
                continue
            # 剪枝：重复元素跳过（前一个相同元素未使用时，当前元素不选）
            if i > 0 and nums[i] == nums[i-1] and not used[i-1]:
                continue
            used[i] = True
            path.append(nums[i])
            backtrack(path)
            path.pop()
            used[i] = False
    
    backtrack([])
    return result
```

## 四、八皇后问题（经典回溯案例）

### 1. 问题描述

在8x8棋盘放置8个皇后，使其互不攻击（不同行、列、对角线）。

### 2. 算法思路

- **路径**：一维数组 `queen[row] = col`（第row行皇后在第col列）。
    
- **剪枝**：放置第row行时，检查列冲突（`queen[i] == col`）和对角线冲突（`abs(row - i) == abs(col - queen[i])`）。
    

### 3. 代码实现（C++）

```cpp
#include <vector>
#include <iostream>
using namespace std;

class EightQueens {
private:
    vector<vector<string>> result;
    vector<int> queen; // queen[row] = col

    bool isValid(int row, int col) {
        for (int i = 0; i < row; ++i) {
            if (queen[i] == col || abs(row - i) == abs(col - queen[i])) {
                return false; // 列冲突或对角线冲突
            }
        }
        return true;
    }

    void backtrack(int row, int n) {
        if (row == n) {
            // 生成棋盘
            vector<string> board(n, string(n, '.'));
            for (int i = 0; i < n; ++i) {
                board[i][queen[i]] = 'Q';
            }
            result.push_back(board);
            return;
        }
        for (int col = 0; col < n; ++col) {
            if (isValid(row, col)) {
                queen[row] = col;
                backtrack(row + 1, n); // 递归下一行
                // 无需显式回溯queen[row]，下次覆盖即可
            }
        }
    }

public:
    vector<vector<string>> solveNQueens(int n) {
        queen.resize(n, -1);
        backtrack(0, n);
        return result;
    }
};
```

## 五、基于穷举法的全排列实现（对比回溯）

### 1. 穷举法（多重循环）

适用于小规模（n≤5），直接用嵌套循环枚举所有排列。

#### 代码实现（C++，n=3）

```cpp
#include <iostream>
#include <vector>
using namespace std;

void brute_force_permute() {
    vector<int> nums = {1, 2, 3};
    vector<vector<int>> res;
    // 三重循环枚举所有排列
    for (int i = 0; i < 3; ++i) {
        for (int j = 0; j < 3; ++j) {
            if (j == i) continue;
            for (int k = 0; k < 3; ++k) {
                if (k == i || k == j) continue;
                res.push_back({nums[i], nums[j], nums[k]});
            }
        }
    }
    // 输出6种排列
}
```

### 2. 回溯法 vs 穷举法

|**维度**|**回溯法**|**穷举法**|
|:--|:--|:--|
|**适用规模**|大规模（n≤10，n!可接受）|小规模（n≤5，循环嵌套爆炸）|
|**灵活性**|易扩展（加剪枝、条件）|固定结构，难扩展|
|**效率**|剪枝后效率高|无剪枝，效率低|





# 分治

## 一、分治算法核心思想

### 1. 定义与本质

**分治算法**（Divide and Conquer）是一种将**复杂问题分解为若干个规模较小的同类子问题**，递归解决子问题后**合并结果**以得到原问题解的算法范式。核心是“**分而治之**”，体现“化整为零、积零为整”的思想。

### 2. 三要素

1. **分解（Divide）**：将原问题划分为若干个相互独立、结构与原问题相同的子问题（规模更小）。
    
2. **解决（Conquer）**：递归求解子问题（若子问题规模足够小，则直接求解）。
    
3. **合并（Combine）**：将子问题的解合并为原问题的解。
    

### 3. 算法框架（伪代码）

```plaintext
def divide_and_conquer(problem):
    if problem is small enough:  # 基线条件（直接求解）
        return solve_directly(problem)
    
    subproblems = divide(problem)  # 分解子问题
    subresults = [divide_and_conquer(sub) for sub in subproblems]  # 递归求解子问题
    result = combine(subresults)  # 合并子问题解
    return result
```

### 4. 适用场景

- 问题可分解为**独立子问题**（子问题间无重叠）；
    
- 子问题的解可**合并**为原问题的解；
    
- 子问题结构与**原问题相同**（递归基础）。
    

## 二、分治算法经典应用

### 1. 快速排序（Quick Sort）

#### 算法思想

快排是分治的典型应用：通过**分区操作**将数组分为“小于基准值”和“大于基准值”的两部分，递归排序子数组。

#### 核心步骤

1. **分解**：选择基准值（pivot），将数组分为左（< pivot）、中（= pivot）、右（> pivot）三部分。
    
2. **解决**：递归排序左子数组和右子数组（中间部分无需排序）。
    
3. **合并**：无需显式合并（子数组排序后自然有序）。
    

#### 分区实现（挖坑法）

```cpp
// 分区函数：返回基准值最终位置
int partition(vector<int>& nums, int left, int right) {
    int pivot = nums[left];  // 选择最左元素为基准（可优化为随机选择）
    int i = left, j = right;
    
    while (i < j) {
        // 从右向左找第一个 < pivot的元素
        while (i < j && nums[j] >= pivot) j--;
        nums[i] = nums[j];  // 填坑
        
        // 从左向右找第一个 > pivot的元素
        while (i < j && nums[i] <= pivot) i++;
        nums[j] = nums[i];  // 填坑
    }
    nums[i] = pivot;  // 基准值放入最终位置
    return i;
}

// 递归排序
void quickSort(vector<int>& nums, int left, int right) {
    if (left >= right) return;  // 基线条件：子数组长度为1或0
    int mid = partition(nums, left, right);  // 分解
    quickSort(nums, left, mid - 1);  // 解决左子数组
    quickSort(nums, mid + 1, right); // 解决右子数组
}
```

#### 复杂度分析

- **时间复杂度**：
    
    - 最好/平均：O(n\log n)（每次分区平衡）；
        
    - 最坏：O(n^2)（数组有序且基准选两端，分区极度不平衡）。
        
- **空间复杂度**：O(\log n)（递归栈深度，最坏O(n)）。
    

#### 优化策略

- **基准选择**：随机选择基准（`rand()%(right-left+1)+left`）或三数取中（首、尾、中间元素的中位数）；
    
- **小规模数组切换插入排序**（减少递归开销）；
    
- **尾递归优化**（减少栈深度）。
    

### 2. 快排求TOP K（第K大/小元素）

#### 问题描述

在无序数组中查找第K大（或小）的元素，要求时间复杂度优于O(n\log n)。

#### 算法思路

利用快排的**分区思想**，无需完全排序：每次分区后，基准值的位置`mid`即为其在有序数组中的索引。

- 若`mid == K-1`（第K小）或`mid == n-K`（第K大），则基准值为所求；
    
- 若`mid > K-1`，则在左子数组递归查找；
    
- 若`mid < K-1`，则在右子数组递归查找。
    

#### 代码实现（第K小元素）

```cpp
int quickSelect(vector<int>& nums, int left, int right, int k) {
    if (left == right) return nums[left];  // 基线条件：子数组长度为1
    
    int mid = partition(nums, left, right);  // 分区，基准位置mid
    if (mid == k - 1) {  // 基准是第k小元素
        return nums[mid];
    } else if (mid > k - 1) {  // 第k小在左子数组
        return quickSelect(nums, left, mid - 1, k);
    } else {  // 第k小在右子数组
        return quickSelect(nums, mid + 1, right, k);
    }
}

// 调用：查找第k小元素（k∈[1,n]）
int findKthSmallest(vector<int>& nums, int k) {
    return quickSelect(nums, 0, nums.size() - 1, k);
}
```

#### 复杂度分析

- **时间复杂度**：平均O(n)，最坏O(n^2)（可通过随机基准优化至期望O(n)）；
    
- **空间复杂度**：O(\log n)（递归栈）。
    

### 3. 归并排序（Merge Sort）

#### 算法思想

归并排序是分治的稳定排序算法：分解数组至单个元素（天然有序），递归合并有序子数组。

#### 核心步骤

1. **分解**：将数组二分至子数组长度为1；
    
2. **解决**：单个元素默认有序；
    
3. **合并**：双指针合并两个有序子数组（关键步骤）。
    

#### 合并两个有序数组

```cpp
void merge(vector<int>& nums, int left, int mid, int right) {
    vector<int> temp(right - left + 1);  // 临时数组存储合并结果
    int i = left, j = mid + 1, k = 0;
    
    // 双指针合并
    while (i <= mid && j <= right) {
        if (nums[i] <= nums[j]) temp[k++] = nums[i++];
        else temp[k++] = nums[j++];
    }
    // 处理剩余元素
    while (i <= mid) temp[k++] = nums[i++];
    while (j <= right) temp[k++] = nums[j++];
    
    // 拷贝回原数组
    for (int p = 0; p < k; p++) nums[left + p] = temp[p];
}
```

#### 递归实现

```cpp
void mergeSort(vector<int>& nums, int left, int right) {
    if (left >= right) return;  // 基线条件
    int mid = (left + right) / 2;  // 分解
    mergeSort(nums, left, mid);   // 解决左子数组
    mergeSort(nums, mid + 1, right);  // 解决右子数组
    merge(nums, left, mid, right);  // 合并
}
```

#### 复杂度分析

- **时间复杂度**：O(n\log n)（所有情况，分解O(\log n)层，每层合并O(n)）；
    
- **空间复杂度**：O(n)（临时数组，递归栈O(\log n)）。
    

#### 特点

- **稳定排序**（相等元素相对位置不变）；
    
- 适合**外部排序**（数据量大到内存放不下时，可分块读入合并）。
    

### 4. 合并K个有序单链表

#### 问题描述

给定K个升序单链表，合并为一个升序单链表。

#### 分治思路

将K个链表**两两分组**，递归合并每组，直到只剩一个链表。

- **分解**：将K个链表分为 $$\lceil K/2 \rceil$$组，每组2个（或1个）链表；
    
- **解决**：递归合并每组链表（合并两个有序链表是基础）；
    
- **合并**：将子问题的解（合并后的链表）继续合并。
    

#### 合并两个有序链表（基础）

```cpp
ListNode* mergeTwoLists(ListNode* l1, ListNode* l2) {
    ListNode* dummy = new ListNode(0);
    ListNode* cur = dummy;
    while (l1 && l2) {
        if (l1->val <= l2->val) {
            cur->next = l1;
            l1 = l1->next;
        } else {
            cur->next = l2;
            l2 = l2->next;
        }
        cur = cur->next;
    }
    cur->next = l1 ? l1 : l2;  // 拼接剩余节点
    return dummy->next;
}
```

#### 分治合并K个链表

```cpp
ListNode* mergeKLists(vector<ListNode*>& lists) {
    if (lists.empty()) return nullptr;
    int n = lists.size();
    // 分解：两两合并
    while (n > 1) {
        int k = (n + 1) / 2;  // 分组数（向上取整）
        for (int i = 0; i < n / 2; i++) {
            lists[i] = mergeTwoLists(lists[i], lists[i + k]);  // 合并第i组和第i+k组
        }
        n = k;  // 更新剩余链表数
    }
    return lists[0];
}
```

#### 复杂度分析

- **时间复杂度**：O(N\log K)（N为总节点数，共\log K层合并，每层合并O(N)）；
    
- **空间复杂度**：O(1)（递归栈O(\log K)，若用迭代合并）。
    

### 5. 对数时间求中位数（两个有序数组）

#### 问题描述

给定两个升序数组`A`和`B`（长度分别为`m`和`n`），求合并后的中位数，要求时间复杂度O(\log(m+n))。

#### 分治思路

利用**二分查找**思想，每次排除一半不可能包含中位数的元素：

1. 保证m \leq n（若m > n，交换A和B）；
    
2. 取`A`的左半部分（`A[0..i-1]`）和`B`的左半部分（`B[0..j-1]`），其中i + j = \frac{m + n + 1}{2}（左半部分总长度为合并后左半部分长度）；
    
3. 若`A[i-1] ≤ B[j]`且`B[j-1] ≤ A[i]`，则左半部分最大值为中位数（或左右平均）；
    
4. 否则，调整`i`的范围（若`A[i-1] > B[j]`，则`i`过大，需减小；反之增大`i`）。
    

#### 代码实现

```cpp
double findMedianSortedArrays(vector<int>& A, vector<int>& B) {
    if (A.size() > B.size()) return findMedianSortedArrays(B, A);  // 保证A短于B
    int m = A.size(), n = B.size();
    int left = 0, right = m, half_len = (m + n + 1) / 2;  // 左半部分总长度
    
    while (left <= right) {
        int i = (left + right) / 2;  // A的分割点（左半部分i个元素）
        int j = half_len - i;        // B的分割点（左半部分j个元素）
        
        // 边界条件处理（i=0或m，j=0或n）
        int A_left = (i == 0) ? INT_MIN : A[i - 1];
        int A_right = (i == m) ? INT_MAX : A[i];
        int B_left = (j == 0) ? INT_MIN : B[j - 1];
        int B_right = (j == n) ? INT_MAX : B[j];
        
        if (A_left <= B_right && B_left <= A_right) {
            // 找到正确分割点，计算中位数
            if ((m + n) % 2 == 1) {
                return max(A_left, B_left);  // 奇数：左半部分最大值
            } else {
                return (max(A_left, B_left) + min(A_right, B_right)) / 2.0;  // 偶数：平均
            }
        } else if (A_left > B_right) {
            right = i - 1;  // A分割点过大，左移
        } else {
            left = i + 1;   // A分割点过小，右移
        }
    }
    return 0.0;  // 无解（题目保证有解）
}
```

#### 复杂度分析

- **时间复杂度**：O(\log(\min(m,n)))（二分查找最短数组）；
    
- **空间复杂度**：O(1)（常数空间）。
    

## 三、分治算法总结

### 1. 核心对比表

|**算法**|**分解方式**|**合并方式**|**时间复杂度**|**空间复杂度**|**稳定性**|
|:--|:--|:--|:--|:--|:--|
|快速排序|分区（基准划分）|无需显式合并|O(n\log n)（平均）|O(\log n)（递归栈）|不稳定|
|归并排序|二分至单个元素|双指针合并有序数组|O(n\log n)（所有）|O(n)（临时数组）|稳定|
|快排求TOP K|分区（基准划分）|无需合并|O(n)（平均）|O(\log n)|不稳定|
|合并K个有序链表|两两分组|合并两个有序链表|O(N\log K)|O(1)|稳定|
|两个有序数组求中位数|二分排除法|直接计算|O(\log(\min(m,n)))|O(1)|-|

### 2. 分治算法优势

- **高效性**：将指数级问题转化为多项式级（如快排O(n\log n)优于冒泡O(n^2)）；
    
- **模块化**：子问题独立，便于并行计算（如MapReduce框架）；
    
- **普适性**：适用于排序、查找、矩阵乘法、大整数乘法等多种场景。
    

### 3. 注意事项

- **子问题独立性**：避免子问题重叠（否则应考虑动态规划）；
    
- **递归终止条件**：明确最小子问题的直接求解方法；
    
- **合并效率**：合并步骤的复杂度直接影响整体效率（如归并排序的合并是瓶颈）。
    


# 动态规划

## 一、动态规划核心思想

### 1. 定义与本质

**动态规划（Dynamic Programming, DP）** 是一种通过将复杂问题分解为**重叠子问题**，并利用**最优子结构**（原问题的最优解包含子问题的最优解）和**记忆化存储**（避免重复计算）来求解的算法范式。核心是“**以空间换时间**”，通过存储子问题解提升效率。

### 2. 核心要素

- **状态（State）**：描述子问题的关键变量（如“前i个物品”“容量为j的背包”）。
    
- **状态转移方程（Transition Equation）**：子问题间的递推关系（核心！）。
    
- **初始化（Initialization）**：最小子问题的直接解（边界条件）。
    
- **计算顺序**：自底向上（迭代）或自顶向下（递归+记忆化）。
    

### 3. 适用场景

- 问题可分解为**重叠子问题**（子问题重复出现）；
    
- 具有**最优子结构**（大问题最优解依赖小问题最优解）；
    
- 无后效性（当前状态只与之前有限状态有关）。
    

## 二、经典动态规划问题详解

### 1. 硬币选择问题（最少硬币数/组合数）

#### 问题描述

给定不同面额的硬币 `coins` 和目标金额 `amount`，求**最少硬币数**（或**凑成金额的方案数**）。

#### （1）最少硬币数（完全背包求最小）

##### 动态规划思路

- **状态定义**：`dp[i]` 表示凑成金额 `i` 所需的最少硬币数。
    
- **转移方程**：`dp[i] = min(dp[i], dp[i - coin] + 1)`（对每个硬币 `coin ≤ i`）。
    
- **初始化**：`dp[0] = 0`（金额0需0个硬币），其余 `dp[i] = ∞`（初始不可达）。
    
- **计算顺序**：从小到大遍历金额（0→amount）。
    

##### 代码实现（C++）

```cpp
int coinChange(vector<int>& coins, int amount) {
    vector<int> dp(amount + 1, INT_MAX - 1);  // 避免溢出
    dp[0] = 0;
    for (int i = 1; i <= amount; ++i) {
        for (int coin : coins) {
            if (coin <= i) {
                dp[i] = min(dp[i], dp[i - coin] + 1);
            }
        }
    }
    return dp[amount] == INT_MAX - 1 ? -1 : dp[amount];
}
```

#### （2）凑金额方案数（完全背包求组合数）

##### 动态规划思路

- **状态定义**：`dp[i]` 表示凑成金额 `i` 的方案数。
    
- **转移方程**：`dp[i] += dp[i - coin]`（对每个硬币 `coin ≤ i`）。
    
- **初始化**：`dp[0] = 1`（金额0只有1种方案：不选硬币）。
    

##### 代码实现（Python）

```python
def change(amount, coins):
    dp = [0] * (amount + 1)
    dp[0] = 1
    for coin in coins:  # 先遍历硬币（避免重复组合）
        for i in range(coin, amount + 1):
            dp[i] += dp[i - coin]
    return dp[amount]
```

### 2. 斐波那契数列（入门DP）

#### 问题描述

求斐波那契数列第 `n` 项：`F(n) = F(n-1) + F(n-2)`，`F(0)=0, F(1)=1`。

#### 动态规划思路（递推法）

- **状态定义**：`dp[i]` 表示第 `i` 项斐波那契数。
    
- **转移方程**：`dp[i] = dp[i-1] + dp[i-2]`。
    
- **初始化**：`dp[0]=0, dp[1]=1`。
    
- **优化**：滚动数组（仅保留前两项，空间O(1)）。
    

##### 代码实现（C++，滚动数组优化）

```cpp
int fib(int n) {
    if (n <= 1) return n;
    int a = 0, b = 1, c;
    for (int i = 2; i <= n; ++i) {
        c = a + b;
        a = b;
        b = c;
    }
    return b;
}
```

### 3. 最大子段和（Kadane算法）

#### 问题描述

给定数组 `nums`，求连续子数组的最大和（子数组至少含一个元素）。

#### 动态规划思路

- **状态定义**：`dp[i]` 表示以第 `i` 个元素结尾的最大子段和。
    
- **转移方程**：`dp[i] = max(nums[i], dp[i-1] + nums[i])`（要么自己单独成段，要么接在前一段后面）。
    
- **初始化**：`dp[0] = nums[0]`。
    
- **结果**：`max(dp[0...n-1])`。
    

##### 代码实现（Python，空间优化O(1)）

```python
def maxSubArray(nums):
    max_sum = current = nums[0]
    for num in nums[1:]:
        current = max(num, current + num)  # 滚动更新当前最大和
        max_sum = max(max_sum, current)
    return max_sum
```

### 4. 最长非降子序列（LIS）

#### 问题描述

给定数组 `nums`，求最长非降子序列的长度（子序列不要求连续）。

#### 动态规划思路（O(n²)基础版）

- **状态定义**：`dp[i]` 表示以第 `i` 个元素结尾的最长非降子序列长度。
    
- **转移方程**：`dp[i] = max(dp[j] + 1)`（对所有 `j < i` 且 `nums[j] ≤ nums[i]`）。
    
- **初始化**：`dp[i] = 1`（每个元素自身为长度1的子序列）。
    
- **结果**：`max(dp[0...n-1])`。
    

##### 代码实现（C++）

```cpp
int lengthOfLIS(vector<int>& nums) {
    int n = nums.size();
    vector<int> dp(n, 1);
    int max_len = 1;
    for (int i = 1; i < n; ++i) {
        for (int j = 0; j < i; ++j) {
            if (nums[j] <= nums[i]) {
                dp[i] = max(dp[i], dp[j] + 1);
            }
        }
        max_len = max(max_len, dp[i]);
    }
    return max_len;
}
```

#### 优化版（O(n log n)，贪心+二分）

- **思路**：维护一个数组 `tails`，`tails[i]` 表示长度为 `i+1` 的非降子序列的最小尾部元素。
    
- **操作**：对每个 `num`，用二分查找 `tails` 中第一个大于 `num` 的位置，替换或追加。
    

##### 代码实现（Python）

```python
import bisect
def lengthOfLIS(nums):
    tails = []
    for num in nums:
        idx = bisect.bisect_right(tails, num)  # 找第一个大于num的位置
        if idx == len(tails):
            tails.append(num)
        else:
            tails[idx] = num
    return len(tails)
```

### 5. 最长公共子序列（LCS）

#### 问题描述

给定两个字符串 `text1` 和 `text2`，求最长公共子序列的长度（子序列不要求连续）。

#### 动态规划思路

- **状态定义**：`dp[i][j]` 表示 `text1[0..i-1]` 和 `text2[0..j-1]` 的LCS长度。
    
- **转移方程**：
    
    - 若 `text1[i-1] == text2[j-1]`：`dp[i][j] = dp[i-1][j-1] + 1`；
        
    - 否则：`dp[i][j] = max(dp[i-1][j], dp[i][j-1])`。
        
- **初始化**：`dp[i][0] = dp[0][j] = 0`（空字符串LCS为0）。
    

##### 代码实现（C++）

```cpp
int longestCommonSubsequence(string text1, string text2) {
    int m = text1.size(), n = text2.size();
    vector<vector<int>> dp(m + 1, vector<int>(n + 1, 0));
    for (int i = 1; i <= m; ++i) {
        for (int j = 1; j <= n; ++j) {
            if (text1[i-1] == text2[j-1]) {
                dp[i][j] = dp[i-1][j-1] + 1;
            } else {
                dp[i][j] = max(dp[i-1][j], dp[i][j-1]);
            }
        }
    }
    return dp[m][n];
}
```

### 6. 背包问题（0-1背包/完全背包）

#### （1）0-1背包（每个物品选一次）

##### 问题描述

给定 `n` 个物品（重量 `w[i]`，价值 `v[i]`）和背包容量 `C`，求最大价值（物品不可分割）。

##### 动态规划思路

- **状态定义**：`dp[i][j]` 表示前 `i` 个物品、容量 `j` 的最大价值。
    
- **转移方程**：
    
    - 不选第 `i` 个物品：`dp[i][j] = dp[i-1][j]`；
        
    - 选第 `i` 个物品（需 `j ≥ w[i]`）：`dp[i][j] = dp[i-1][j-w[i]] + v[i]`；
        
    - 综上：`dp[i][j] = max(dp[i-1][j], dp[i-1][j-w[i]] + v[i])`。
        
- **优化**：一维数组 `dp[j]`（倒序遍历容量，避免覆盖）。
    

##### 代码实现（Python，一维优化）

```python
def knapsack_01(w, v, C):
    dp = [0] * (C + 1)
    for i in range(len(w)):
        for j in range(C, w[i]-1, -1):  # 倒序遍历容量
            dp[j] = max(dp[j], dp[j - w[i]] + v[i])
    return dp[C]
```

#### （2）完全背包（每个物品选无限次）

##### 动态规划思路

- **转移方程**：`dp[i][j] = max(dp[i-1][j], dp[i][j-w[i]] + v[i])`（正序遍历容量，允许重复选）。
    
- **一维优化**：正序遍历容量 `j`（`for j in range(w[i], C+1)`）。
    

### 7. 三角数组求和（最小路径和）

#### 问题描述

给定一个三角数组（每行元素个数递增），从顶部出发，每次只能向左下或右下移动，求到达底部的最小路径和。

#### 动态规划思路

- **状态定义**：`dp[i][j]` 表示到达 `(i,j)` 的最小路径和。
    
- **转移方程**：
    
    - 顶层：`dp[0][0] = triangle[0][0]`；
        
    - 左边界（`j=0`）：`dp[i][0] = dp[i-1][0] + triangle[i][0]`；
        
    - 右边界（`j=i`）：`dp[i][j] = dp[i-1][j-1] + triangle[i][j]`；
        
    - 内部：`dp[i][j] = min(dp[i-1][j-1], dp[i-1][j]) + triangle[i][j]`。
        
- **优化**：原地修改数组（从下往上计算，节省空间）。
    

##### 代码实现（Python，原地修改）

```python
def minimumTotal(triangle):
    n = len(triangle)
    dp = triangle[-1]  # 底层作为初始状态
    for i in range(n-2, -1, -1):  # 从倒数第二层往上
        for j in range(i+1):
            dp[j] = min(dp[j], dp[j+1]) + triangle[i][j]
    return dp[0]
```

## 三、动态规划总结

### 1. 通用解题步骤

1. **定义状态**：明确 `dp[i]` 或 `dp[i][j]` 的物理意义；
    
2. **写出转移方程**：分析大问题如何由小问题推导而来；
    
3. **初始化边界**：最小子问题的直接解（如 `dp[0]`、`dp[i][0]`）；
    
4. **确定计算顺序**：自底向上（迭代）或自顶向下（递归+记忆化）；
    
5. **输出结果**：根据状态定义提取最终答案。
    

### 2. 常见问题类型

|**问题类型**|**核心思路**|**典型案例**|
|:--|:--|:--|
|线性DP|单序列状态转移（如LIS、最大子段和）|斐波那契、最大子段和|
|二维DP|双序列/网格状态转移（如LCS、背包）|LCS、0-1背包、编辑距离|

|树形DP|后序遍历树节点，合并子树状态|树的最大独立集|

# Trie字典树

## 一、Trie树核心定义与本质

### 1. 基本概念

**Trie树**（又称**前缀树**、**字典树**）是一种**树形数据结构**，专门用于处理**字符串集合的前缀相关操作**（如插入、查找、前缀匹配）。其核心思想是：**用共享前缀减少存储空间，用树形路径表示字符串**。

- **本质**：一种“多叉树”，每个节点代表一个**字符**，从根节点到某一节点的路径构成一个**字符串前缀**，节点标记表示该路径是否为完整单词。
    

### 2. 核心特点

- **前缀共享**：相同前缀的字符串共享树的前几层节点（如“apple”和“app”共享“app”路径）；
    
- **高效前缀操作**：插入、查找、前缀匹配的复杂度均为 **O(L)**（L为字符串长度），与字符串集合大小无关；
    
- **字符集依赖**：节点分支数取决于字符集（如小写字母26个分支，中文则需更大空间）。
    

## 二、Trie树的结构设计

### 1. 节点组成（核心要素）

每个Trie节点包含两部分：

- **子节点集合**：存储当前字符的后续字符（如`children[26]`对应26个小写字母，或用哈希表`unordered_map<char, TrieNode*>`支持任意字符）；
    
- **结束标记**：`isEnd`（布尔值），表示从根节点到当前节点的路径是否构成一个完整单词。
    

**示例代码（C++节点定义）**：

```cpp
struct TrieNode {
    TrieNode* children[26];  // 假设仅处理小写字母（26个分支）
    bool isEnd;              // 标记是否为单词结尾
    TrieNode() : isEnd(false) {
        memset(children, 0, sizeof(children));  // 初始化为空指针
    }
};
```

### 2. 整体结构

- **根节点**：不存储任何字符，作为所有字符串的起始点（空节点）；
    
- **子节点**：根节点的子节点对应字符串的第一个字符，以此类推。
    

**示例**：“apple”和“app”的Trie结构：

```
        (root)  
          |  
          a  
          |  
          p  
         / \  
        p   p  
       /     \  
      l       l  
     /         \  
    e(isEnd)    e(isEnd)  // "app"的e节点isEnd=true，"apple"的e节点isEnd=true  
```

## 三、Trie树核心操作（插入、查找、前缀匹配）

### 1. 插入操作（Insert）

**目标**：将字符串`s`插入Trie树，标记完整单词的结束节点。

**步骤**：

1. 从根节点出发，遍历字符串`s`的每个字符`c`；
    
2. 计算`c`在子节点中的索引（如`idx = c - 'a'`，小写字母）；
    
3. 若当前节点的`children[idx]`为空，则创建新节点；
    
4. 移动到`children[idx]`节点，继续处理下一个字符；
    
5. 字符串遍历结束后，将当前节点的`isEnd`标记为`true`。
    

**代码实现（C++）**：

```cpp
class Trie {
private:
    TrieNode* root;
public:
    Trie() { root = new TrieNode(); }

    void insert(string word) {
        TrieNode* node = root;
        for (char c : word) {
            int idx = c - 'a';  // 假设小写字母
            if (!node->children[idx]) {
                node->children[idx] = new TrieNode();
            }
            node = node->children[idx];
        }
        node->isEnd = true;  // 标记单词结束
    }
};
```

### 2. 查找操作（Search）

**目标**：判断字符串`s`是否在Trie树中（精确匹配）。

**步骤**：

1. 从根节点出发，遍历字符串`s`的每个字符`c`；
    
2. 计算索引`idx = c - 'a'`，若当前节点的`children[idx]`为空，返回`false`；
    
3. 移动到`children[idx]`节点，继续处理下一个字符；
    
4. 遍历结束后，检查当前节点的`isEnd`是否为`true`（确保是完整单词，而非前缀）。
    

**代码实现**：

```cpp
bool search(string word) {
    TrieNode* node = root;
    for (char c : word) {
        int idx = c - 'a';
        if (!node->children[idx]) return false;
        node = node->children[idx];
    }
    return node->isEnd;  // 必须是完整单词
}
```

### 3. 前缀匹配（StartsWith）

**目标**：判断是否存在以字符串`prefix`为前缀的单词。

**步骤**：

1. 与查找操作类似，但无需检查`isEnd`；
    
2. 遍历`prefix`的所有字符，若存在路径则返`true`，否则`false`。
    

**代码实现**：

```cpp
bool startsWith(string prefix) {
    TrieNode* node = root;
    for (char c : prefix) {
        int idx = c - 'a';
        if (!node->children[idx]) return false;
        node = node->children[idx];
    }
    return true;  // 前缀存在即可
}
```

## 四、Trie树的应用场景

### 1. 核心场景（高频）

|**场景**|**原理**|**示例**|
|:--|:--|:--|
|**自动补全**|输入前缀，遍历Trie树找到所有以该前缀开头的单词（DFS收集）|搜索引擎输入“ap”提示“apple”“app”|
|**拼写检查**|查找单词是否在Trie中，不在则提示错误（如IDE下划线标红）|Word文档拼写检查|
|**IP路由（最长前缀匹配）**|路由器用Trie存储IP前缀，匹配最长前缀的路由表项（如CIDR）|网络路由选择|
|**单词游戏**|判断给定字符能否组成Trie中的单词（如Boggle、Scrabble）|拼字游戏验证单词合法性|
|**词频统计**|节点增加`count`字段，记录以该节点为结尾的单词出现次数|统计文本中单词频率|

### 2. 扩展功能

- **删除操作**：从叶子节点向上回溯，删除无用的节点（需判断节点是否还有其他子节点或被其他单词共享）；
    
- **前缀计数**：节点增加`prefixCount`字段，记录经过该节点的字符串数量（如统计以“app”为前缀的单词数）；
    
- **多模式匹配**：结合AC自动机（Aho-Corasick算法），在Trie基础上增加失败指针，实现多字符串同时匹配（如敏感词过滤）。
    

## 五、Trie树的优缺点与对比

### 1. 优点

- **前缀操作高效**：插入、查找、前缀匹配均为O(L)，优于哈希表（哈希表需计算哈希值，且前缀匹配需遍历所有键）；
    
- **无哈希冲突**：树形结构天然避免冲突，无需处理哈希碰撞；
    
- **前缀共享节省空间**：相同前缀的字符串共享节点（尤其适合大量共享前缀的场景）。
    

### 2. 缺点

- **空间开销大**：节点分支数取决于字符集（如中文需数千分支），稀疏场景下空间利用率低；
    
- **实现复杂度较高**：需手动管理节点指针（易内存泄漏），哈希表实现虽灵活但仍有开销。
    

### 3. 与哈希表对比

|**维度**|**Trie树**|**哈希表**|
|:--|:--|:--|
|**前缀匹配**|O(L)，高效（直接遍历前缀路径）|O(N*L)，低效（需遍历所有键检查前缀）|
|**哈希冲突**|无|有（需处理碰撞，如链地址法）|
|**空间效率**|共享前缀时高效，稀疏时低效|平均高效（取决于负载因子）|
|**字符集支持**|灵活（哈希表支持任意字符，Trie需预设分支）|灵活（支持任意字符）|

## 六、Trie树的变种与优化

### 1. 压缩Trie（Radix Tree/Patricia Trie）

**优化思路**：合并只有一个子节点的节点（压缩路径），减少空间开销。

**示例**：将“aaab”和“aabb”的Trie压缩为根→a→a（分支：b→b和 b→b），而非原始的4层节点。

### 2. 后缀树（Suffix Tree）

**扩展**：将所有字符串的后缀插入Trie，支持快速子串查找（如DNA序列匹配）。

### 3. 双数组Trie（Double-Array Trie）

**优化思路**：用两个数组`base`和`check`存储节点状态，替代指针，提升空间效率和缓存命中率（常用于输入法引擎）。

## 七、总结与高频面试问题

### 1. 核心总结

- **结构**：树形结构，节点存储字符和结束标记，路径表示字符串；
    
- **操作**：插入（建路径+标记结束）、查找（遍历路径+检查结束标记）、前缀匹配（遍历前缀路径）；
    
- **优势**：前缀操作O(L)高效，无哈希冲突；
    
- **应用**：自动补全、拼写检查、IP路由、单词游戏；
    
- **优化**：压缩Trie、双数组Trie减少空间开销。
    

### 2. 高频面试问题

1. **Trie树的插入和查找流程？**
    
    答：插入：从根节点遍历字符，不存在则创建节点，结束时标记isEnd=true；查找：遍历字符，路径不存在则返回false，结束时检查isEnd是否为true。
    
2. **Trie树与前缀树的关系？**
    
    答：Trie树又称前缀树，核心功能是高效处理前缀相关操作（插入、查找、前缀匹配）。
    
3. **Trie树的空间复杂度为什么高？**
    
    答：节点分支数取决于字符集（如26个小写字母需26个子节点指针），稀疏场景下大量空指针浪费空间。
    
4. **如何用Trie实现敏感词过滤？**
    
    答：将所有敏感词插入Trie，遍历输入文本，对每个字符检查是否存在敏感词前缀（结合AC自动机的失败指针优化多模式匹配）。
    
5. **Trie树的查找时间复杂度是多少？**
    
    答：O(L)，L为字符串长度（与字符串集合大小无关）。
    


# 跳表

## 一、跳表核心定义与本质

### 1. 基本概念

**跳表（Skip List）** 是一种**有序数据结构**，通过**多层链表索引**实现高效的**动态查找、插入、删除**操作，时间复杂度均为 **O(log n)**（平均/期望），可视为**平衡树（如红黑树）的简化替代方案**。

- **本质**：在**有序单链表**基础上，增加**多级“快速通道”索引**（高层索引是低层索引的子集），通过“跳跃”跳过大量节点，将单链表的O(n)查找优化为O(log n)。
    

### 2. 核心思想

- **空间换时间**：通过额外的多层索引（空间开销O(n)）换取查找效率的提升；
    
- **随机化平衡**：插入节点时随机生成“层数”，避免像平衡树（如红黑树）那样通过复杂旋转维持平衡，实现“概率平衡”。
    

## 二、跳表的结构设计

### 1. 节点组成

每个跳表节点包含两部分：

- **值（value）**：存储数据（如整数、字符串）；
    
- **多层指针数组（forward[]）**：`forward[i]`指向当前节点在第`i`层的下一个节点（第0层为原始链表，最底层包含所有元素）。
    

**示例（C++节点定义）**：

```cpp
const int MAX_LEVEL = 32;  // 最大层数（可根据需求调整）
const double P = 0.5;      // 节点晋升概率（每层有50%概率继续向上）

struct SkipListNode {
    int val;                // 存储的值（假设为整数）
    vector<SkipListNode*> forward;  // 各层指针数组，forward[i]是第i层的下一个节点
    SkipListNode(int v, int level) : val(v), forward(level, nullptr) {}
};
```

### 2. 整体结构

- **头节点（head）**：不存储实际数据，其`forward`数组指向各层的第一个节点（初始时所有指针为`nullptr`）；
    
- **当前最大层数（level）**：记录跳表中实际使用的最大层数（初始为1）；
    
- **各层链表**：第`i`层链表是有序的，包含部分节点（高层节点数约为低层的1/P，如P=0.5时，每层节点数减半）。
    

**示例（含5个元素的跳表，最大3层）**：

```
第2层（高层索引）： head -> 3 -> 7 -> null  
第1层（中层索引）： head -> 3 -> 5 -> 7 -> 9 -> null  
第0层（原始链表）： head -> 1 -> 3 -> 5 -> 7 -> 9 -> null  
```

## 三、跳表核心操作（查找、插入、删除）

### 1. 查找操作（Search）

**目标**：在跳表中查找值为`target`的节点，存在返回`true`，否则`false`。

**步骤**（从最高层开始“跳跃”）：

1. 从**当前最大层**的头节点出发，设当前层为`lvl = level`；
    
2. 若`forward[lvl]`不为空且`forward[lvl]->val < target`，则沿当前层前进（`cur = cur->forward[lvl]`）；
    
3. 否则，**下降一层**（`lvl--`），重复步骤2；
    
4. 当`lvl=0`时，若`forward[0]`不为空且`forward[0]->val == target`，则找到；否则未找到。
    

**代码实现（C++）**：

```cpp
bool search(int target) {
    SkipListNode* cur = head;
    // 从最高层开始向下查找
    for (int lvl = level - 1; lvl >= 0; lvl--) {
        while (cur->forward[lvl] && cur->forward[lvl]->val < target) {
            cur = cur->forward[lvl];  // 同层前进
        }
    }
    // 到达最底层，检查下一个节点
    cur = cur->forward[0];
    return cur != nullptr && cur->val == target;
}
```

### 2. 插入操作（Insert）

**目标**：将值`val`插入跳表，保持有序性，并随机生成节点层数。

**步骤**：

1. **查找插入位置**：用数组`update[MAX_LEVEL]`记录每层应插入位置的前驱节点（即每层最后一个小于`val`的节点）；
    
2. **随机生成层数**：用抛硬币法（如P=0.5）生成新节点层数`new_level`（至少1层，不超过`MAX_LEVEL`）；
    
3. **创建新节点**：用`new_level`初始化节点；
    
4. **更新各层索引**：在`1~new_level`层，将新节点插入到`update[i]`之后（调整指针）。
    

**关键：随机层数生成**

```cpp
int randomLevel() {
    int lvl = 1;
    while ((rand() % 2) == 1 && lvl < MAX_LEVEL) {  // 50%概率升层
        lvl++;
    }
    return lvl;
}
```

**代码实现（C++）**：

```cpp
void insert(int val) {
    vector<SkipListNode*> update(MAX_LEVEL, nullptr);  // 记录各层前驱
    SkipListNode* cur = head;

    // 1. 查找各层插入位置的前驱节点
    for (int lvl = level - 1; lvl >= 0; lvl--) {
        while (cur->forward[lvl] && cur->forward[lvl]->val < val) {
            cur = cur->forward[lvl];
        }
        update[lvl] = cur;  // 第lvl层的前驱是cur
    }

    // 2. 随机生成新节点层数
    int new_level = randomLevel();
    if (new_level > level) {  // 若新层数超过当前最大层，更新前驱
        for (int lvl = level; lvl < new_level; lvl++) {
            update[lvl] = head;
        }
        level = new_level;  // 更新跳表最大层
    }

    // 3. 创建新节点并插入各层
    SkipListNode* newNode = new SkipListNode(val, new_level);
    for (int lvl = 0; lvl < new_level; lvl++) {
        newNode->forward[lvl] = update[lvl]->forward[lvl];
        update[lvl]->forward[lvl] = newNode;
    }
}
```

### 3. 删除操作（Delete）

**目标**：删除跳表中值为`val`的节点。

**步骤**：

1. **查找待删节点位置**：用`update`数组记录各层待删节点的前驱；
    
2. **检查节点存在性**：若最底层（`lvl=0`）的`forward[0]`节点值不等于`val`，则节点不存在；
    
3. **更新各层索引**：在`1~node->level`层，将`update[i]->forward[i]`指向`node->forward[i]`（跳过待删节点）；
    
4. **更新最大层数**：若删除的是最大层唯一节点，需降低跳表最大层。
    

**代码实现（C++）**：

```cpp
void remove(int val) {
    vector<SkipListNode*> update(MAX_LEVEL, nullptr);
    SkipListNode* cur = head;

    // 1. 查找各层待删节点的前驱
    for (int lvl = level - 1; lvl >= 0; lvl--) {
        while (cur->forward[lvl] && cur->forward[lvl]->val < val) {
            cur = cur->forward[lvl];
        }
        update[lvl] = cur;
    }

    // 2. 检查节点是否存在（最底层）
    cur = cur->forward[0];
    if (!cur || cur->val != val) return;  // 节点不存在

    // 3. 在各层删除节点
    int node_level = cur->forward.size();  // 待删节点的层数
    for (int lvl = 0; lvl < node_level; lvl++) {
        update[lvl]->forward[lvl] = cur->forward[lvl];
    }

    // 4. 更新跳表最大层（若删除的是最高层节点）
    while (level > 1 && head->forward[level - 1] == nullptr) {
        level--;
    }

    delete cur;  // 释放内存（实际应用中需注意内存管理）
}
```

## 四、跳表的时间复杂度与空间复杂度

### 1. 时间复杂度

- **查找/插入/删除**：**平均O(log n)**，**最坏O(n)**（概率极低，可通过限制最大层数降低风险）；
    
- **原理**：随机层数使跳表层数期望为O(log n)，每层查找次数为O(1)，总时间O(log n)。
    

### 2. 空间复杂度

- **O(n)**：每个节点平均层数为O(1)（如P=0.5时，期望层数为2），总空间为节点数×平均层数=O(n)。
    

## 五、跳表的优缺点与应用场景

### 1. 优点

- **实现简单**：无需像平衡树（红黑树、AVL树）那样处理复杂的旋转操作；
    
- **高效动态操作**：插入/删除仅需修改指针，无需调整结构；
    
- **范围查询友好**：可直接遍历底层链表实现范围查询（如`[min, max]`）；
    
- **随机化平衡**：通过随机层数避免最坏情况，概率上保证高效性。
    

### 2. 缺点

- **空间开销较大**：需存储多层指针（相比平衡树的O(n)空间，跳表额外开销约2倍）；
    
- **缓存局部性差**：节点分散在内存中，缓存命中率低于数组实现的平衡树；
    
- **最坏情况风险**：极端情况下（如随机数生成偏差）可能退化为单链表（概率极低）。
    

### 3. 应用场景

- **Redis有序集合（ZSet）**：底层用跳表实现，支持O(log n)的插入、删除、排名查询；
    
- **LevelDB/RocksDB**：作为MemTable的索引结构，支持高效范围查询；
    
- **分布式系统**：替代平衡树实现分布式键值存储（如Etcd）；
    
- **算法竞赛**：简化动态有序集合的实现（避免手写平衡树）。
    

## 六、跳表 vs 平衡树（红黑树/AVL树）

|**维度**|**跳表**|**平衡树（红黑树/AVL树）**|
|:--|:--|:--|
|**实现难度**|简单（仅指针操作，无旋转）|复杂（需处理旋转、平衡因子/颜色调整）|
|**时间复杂度**|平均O(log n)（随机化保证）|严格O(log n)（最坏情况也保证）|
|**空间开销**|O(n)（额外指针）|O(n)（节点存储平衡因子/颜色，无多层指针）|
|**范围查询**|高效（直接遍历底层链表）|需中序遍历（O(k + log n)，k为结果数）|
|**缓存友好性**|差（节点分散）|较好（数组+指针，局部性好）|

## 七、总结与高频面试问题

### 1. 核心总结

- **结构**：多层有序链表，高层索引是低层子集，头节点引导查找；
    
- **操作**：查找（从高层跳跃）、插入（随机层数+更新索引）、删除（前驱定位+指针调整）；
    
- **优势**：实现简单、动态操作高效、范围查询友好；
    
- **应用**：Redis ZSet、LevelDB、分布式存储。
    

### 2. 高频面试问题

1. **跳表的时间复杂度是多少？为什么？**
    
    答：平均O(log n)，最坏O(n)（概率极低）。随机层数使层数为O(log n)，每层查找O(1)次，总O(log n)。
    
2. **跳表如何实现随机层数？**
    
    答：用抛硬币法（如P=0.5），每次有50%概率增加一层，直到达到最大层或概率为0。
    
3. **跳表和红黑树的区别？**
    
    答：跳表实现简单（无旋转），范围查询高效，空间开销大；红黑树实现复杂，严格O(log n)，缓存友好。
    
4. **跳表的应用场景？**
    
    答：Redis有序集合、LevelDB索引、分布式键值存储（如Etcd）。
    
5. **跳表的空间复杂度是多少？**
    
    答：O(n)，每个节点平均O(1)层（如P=0.5时期望2层），总空间O(n)。
    


# 并查集与最小生成树

## 一、并查集（Disjoint Set Union, DSU）

### 1. 核心定义与本质

**并查集**是一种用于管理**动态连通性**的数据结构，支持两种核心操作：

- **查找（Find）**：确定元素所属的集合（找到根节点）；
    
- **合并（Union）**：将两个集合合并为一个集合。
    

**本质**：用**树形结构**表示集合，每个集合的根节点唯一标识该集合，通过“父指针”链接节点。

### 2. 核心操作与优化

#### （1）基础操作（未优化）

- **初始化**：每个元素自成一个集合，父指针指向自己（`parent[i] = i`）。
    
- **查找（Find）**：递归或迭代寻找根节点（`parent[x] == x`时的`x`）。
    
    ```cpp
    // 递归查找（未优化）
    int find(int x) {
        if (parent[x] != x) 
        return find(parent[x]);
        return x;
    }
    ```
    
- **合并（Union）**：直接将其中一个集合的根节点指向另一个集合的根节点。
    
    ```cpp
    void unionSets(int x, int y) {
        int rx = find(x), ry = find(y);
        if (rx != ry) parent[rx] = ry;  
        // 合并集合（未优化）
    }
    ```
    

#### （2）优化策略（关键！）

未优化的并查集可能因“链式结构”导致查找效率退化至O(n)，需通过以下优化提升至**近似O(α(n))**（α为反阿克曼函数，增长极慢，近乎常数）：

##### ① 路径压缩（Path Compression）

**原理**：查找时将节点直接挂载到根节点下，缩短后续查找路径。

**实现**（递归/迭代）：

```cpp
// 递归路径压缩（推荐）
int find(int x) {
    if (parent[x] != x) 
        parent[x] = find(parent[x]);  // 递归时将父节点设为根节点
    return parent[x];
}

// 迭代路径压缩（避免栈溢出）
int find(int x) {
    int root = x;
    while (parent[root] != root) root = parent[root];  // 找根节点
    // 路径压缩：所有节点直接指向根
    while (x != root) {
        int next = parent[x];
        parent[x] = root;
        x = next;
    }
    return root;
}
```

##### ② 按秩合并（Union by Rank/Size）

**原理**：合并时将“较矮的树”挂载到“较高的树”下（或按集合大小合并），避免树过高。

- **秩（Rank）**：可表示树的深度（初始0）或集合大小（初始1）。
    
    **实现**（按秩合并+路径压缩）：
    

```cpp
vector<int> parent, rank;  // rank[i]表示以i为根的树的高度（或大小）

void init(int n) {
    parent.resize(n);
    rank.resize(n, 0);  // 初始秩为0（深度）
    for (int i = 0; i < n; ++i) parent[i] = i;
}

int find(int x) { /* 路径压缩实现 */ }

void unionSets(int x, int y) {
    int rx = find(x), ry = find(y);
    if (rx == ry) return;  // 已同属一集合
    // 按秩合并：秩小的树挂载到秩大的树下
    if (rank[rx] < rank[ry]) {
        parent[rx] = ry;
    } else if (rank[rx] > rank[ry]) {
        parent[ry] = rx;
    } else {
        parent[ry] = rx;  // 秩相等时，合并后秩+1
        rank[rx]++;
    }
}
```

### 3. 时间复杂度

- **未优化**：查找O(n)，合并O(n)；
    
- **仅路径压缩**：查找O(log n)，合并O(log n)；
    
- **仅按秩合并**：查找O(log n)，合并O(log n)；
    
- **两者结合**：**近似O(α(n))**（α(n) ≤ 5，n为元素总数），实际可视为常数时间。
    

### 4. 应用场景

- **动态连通性问题**：判断图中两点是否连通（如社交网络好友关系）；
    
- **最小生成树（Kruskal算法）**：判断边的两个端点是否已连通，避免成环；
    
- **图的连通分量计数**：统计图中独立子图的数量；
    
- **等价类划分**：如编译器中的类型统一、图像处理中的区域合并。
    

## 二、最小生成树（Minimum Spanning Tree, MST）

### 1. 核心定义与性质

- **生成树**：带权无向图G=(V,E)的**极小连通子图**，包含G的所有顶点（|V|个）和|V|-1条边，且无环。
    
- **最小生成树（MST）**：所有生成树中**边权之和最小**的生成树。
    

**核心性质**：

- **割性质**：对任意图的割（顶点集划分），连接割的两个部分的**最小权边**必属于某棵MST；
    
- **回路性质**：对图中任意回路，回路中**最大权边**不属于任何MST。
    

### 2. 算法：Kruskal与Prim

#### （1）Kruskal算法（贪心+并查集）

**核心思想**：按边权**从小到大排序**，依次选边，若边的两个端点未连通（用并查集判断），则加入MST，直至选够|V|-1条边。

**步骤**：

1. 将所有边按权值升序排序；
    
2. 初始化并查集（每个顶点自成一个集合）；
    
3. 遍历排序后的边，对每条边(u, v)：
    
    - 若u和v未连通（`find(u) != find(v)`），则将边加入MST，合并u和v所在集合；
        
4. 重复至MST包含|V|-1条边（或遍历完所有边）。
    

**时间复杂度**：O(E log E)（排序边占主导，E为边数），适合**稀疏图**（E较小）。

**代码框架（C++）**：

```cpp
struct Edge { int u, v, w; };  // 边：起点、终点、权值
vector<Edge> edges;  // 存储所有边

// Kruskal算法
int kruskal(int n) {
    sort(edges.begin(), edges.end(), [](Edge a, Edge b) { return a.w < b.w; });  // 按权值排序
    init(n);  // 初始化并查集（parent[0..n-1]，rank[0..n-1]）
    int mst_weight = 0, cnt = 0;  // MST权值和、边数
    for (Edge e : edges) {
        if (find(e.u) != find(e.v)) {  // 未连通
            unionSets(e.u, e.v);
            mst_weight += e.w;
            if (++cnt == n - 1) break;  // 选够n-1条边
        }
    }
    return (cnt == n - 1) ? mst_weight : -1;  // 图不连通时返回-1
}
```

#### （2）Prim算法（贪心+优先队列）

**核心思想**：从一个顶点出发，逐步扩展MST，每次选择**连接MST与非MST顶点的最小权边**。

**步骤**（基于优先队列优化）：

1. 任选一起始顶点s，加入MST集合；
    
2. 用优先队列（小顶堆）存储所有连接MST与非MST顶点的边（权值最小优先）；
    
3. 弹出队列中权值最小的边(u, v)，若v未加入MST：
    
    - 将边(u, v)加入MST，累加权值；
        
    - 将v加入MST，并将v的所有邻边（连接非MST顶点）加入队列；
        
4. 重复至MST包含|V|个顶点（或队列为空）。
    

**时间复杂度**：O(E log V)（优先队列操作占主导），适合**稠密图**（V较小）。

**代码框架（C++）**：

```cpp
typedef pair<int, int> PII;  // (权值, 顶点)
vector<vector<PII>> graph;  // 邻接表：graph[u] = [(v, w), ...]

// Prim算法
int prim(int start, int n) {
    vector<bool> inMST(n, false);  // 标记是否在MST中
    priority_queue<PII, vector<PII>, greater<PII>> pq;  // 小顶堆：(权值, 顶点)
    pq.push({0, start});  // 起始顶点权值0
    int mst_weight = 0, cnt = 0;

    while (!pq.empty() && cnt < n) {
        auto [w, u] = pq.top(); pq.pop();
        if (inMST[u]) continue;  // 已在MST中，跳过
        inMST[u] = true;
        mst_weight += w;
        cnt++;
        // 将u的邻边加入队列（连接非MST顶点）
        for (auto [v, weight] : graph[u]) {
            if (!inMST[v]) {
                pq.push({weight, v});
            }
        }
    }
    return (cnt == n) ? mst_weight : -1;  // 图不连通时返回-1
}
```

### 3. 两种算法对比

|**维度**|**Kruskal算法**|**Prim算法**|
|:--|:--|:--|
|**核心思想**|按边权排序，选最小边（避环）|从点扩展，选连接MST的最小边|
|**数据结构**|并查集（判断连通性）+ 排序|优先队列（选最小边）+ 邻接表|
|**时间复杂度**|O(E log E)（排序主导）|O(E log V)（优先队列主导）|
|**适用场景**|稀疏图（E小，如社交网络）|稠密图（V小，如完全图）|
|**实现难度**|简单（依赖并查集）|中等（需维护优先队列和MST集合）|

### 4. 扩展：MST的唯一性

- **唯一MST**：若图中所有边权**互不相同**，则MST唯一；
    
- **非唯一MST**：若存在权值相同的边，可能有多个MST（但权值和相同）。
    

## 三、总结与高频面试问题

### 1. 核心总结

- **并查集**：动态连通性管理，核心操作`Find`（路径压缩）和`Union`（按秩合并），时间复杂度近似O(α(n))，用于Kruskal算法、连通分量计数；
    
- **最小生成树**：带权无向图的最小权连通子图，Kruskal（排序+并查集，稀疏图）和Prim（优先队列+点扩展，稠密图）是两大算法，时间复杂度分别为O(E log E)和O(E log V)。
    

### 2. 高频面试问题

1. **并查集的优化方法有哪些？**
    
    答：路径压缩（查找时节点直连根）和按秩合并（合并时小树挂大树下），结合后时间复杂度近似O(α(n))。
    
2. **Kruskal和Prim算法的区别？**
    
    答：Kruskal按边排序用并查集避环，适合稀疏图；Prim从点扩展用优先队列选边，适合稠密图。
    
3. **如何用并查集判断图的连通性？**
    
    答：初始化并查集，合并所有边连接的两点，最后统计根节点数量（即连通分量数）。
    
4. **最小生成树的性质？**
    
    答：割性质（最小边必入MST）和回路性质（最大边不入MST）。
    
5. **Kruskal算法中并查集的作用？**
    
    答：判断当前边的两个端点是否已连通，避免MST成环。
    

**记忆口诀**：“并查集管连通性，Find压缩按秩合；Kruskal边排序，并查避环选最小；Prim点扩展，优先队列挑边连；稀疏Kruskal，稠密Prim选；MST权最小，两算法记心间！”
