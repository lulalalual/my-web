塔防双人是一个用 C++ 和 SDL2 写的桌面塔防游戏。它的价值不只是能玩，而是把一个小游戏拆成了清楚的工程模块。

## 游戏怎么跑

游戏每一帧都在做三件事：

```text
处理输入 -> 更新状态 -> 绘制画面
```

## 小代码

```cpp
while (!quit) {
    handleInput();
    update(deltaTime);
    render();
}
```

这就是游戏主循环。玩家移动、怪物前进、炮塔攻击、子弹飞行，都挂在这个循环里。

## 核心模块

- 地图：负责关卡路径和格子位置。
- 怪物：负责移动、血量和奖励。
- 炮塔：负责范围、攻击和升级。
- 玩家：负责移动、技能和金币拾取。
- 资源管理：负责图片、字体和音效加载。

## 项目收获

这个项目能展示游戏工程拆分能力。后续如果继续做，可以补双人协作、关卡编辑器、路径寻路和性能分析。
