二分搜索不是“看到有序数组就套模板”。它真正依赖的是单调性：左边一段不行，右边一段可以，或者反过来。

## 数组二分

```cpp
int l = 0, r = nums.size();
while (l < r) {
    int m = l + (r - l) / 2;
    if (nums[m] >= target) r = m;
    else l = m + 1;
}
```

这段代码找的是第一个大于等于 `target` 的位置。

## 答案二分

有些题不是在数组里找数，而是在答案范围里找最小可行值。

```cpp
while (l < r) {
    int m = l + (r - l) / 2;
    if (check(m)) r = m;
    else l = m + 1;
}
```

`check(m)` 要能回答：答案取 `m` 时行不行。

## 常见坑

- 没有单调性却硬二分。
- 更新边界时把最大值和最小值模板混用。
- `mid` 直接写 `(l + r) / 2`，大数时可能溢出。

写二分前，先用一句话说清楚：什么情况下左边丢掉，什么情况下右边丢掉。
