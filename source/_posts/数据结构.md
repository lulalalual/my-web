数据结构不是为了背名字，而是在回答一个很朴素的问题：数据怎么摆，后面才好用。摆法不同，查找、插入、删除和遍历的成本就不同。

## 先说人话

同样一批数据，可以放成一排，也可以串成链，也可以组织成树，还可以用哈希映射到桶里。没有一种结构永远最好，只有适合当前操作的结构。

~~~text
数据怎么存 -> 操作怎么做 -> 复杂度是多少 -> 场景合不合适
~~~

所以学数据结构时，不要只问“它是什么”，还要问“它适合什么、不适合什么”。

## 数组：连续柜子

数组像一排连续柜子，每个格子都有编号。只要知道下标，就能马上找到元素。

~~~cpp
std::vector<int> nums = {10, 20, 30};
std::cout << nums[1] << std::endl;
~~~

优点是随机访问快，缓存友好。CPU 读取连续内存时效率很高，所以很多场景下 vector 比链表更实用。

缺点是中间插入和删除要移动元素。如果在开头插入一个元素，后面的元素都要往后挪。

适合场景：频繁按下标访问、排序、二分、数据规模相对稳定。

## 链表：一节一节串起来

链表不是连续存放，而是每个节点保存数据和下一个节点地址。

~~~cpp
struct Node {
    int value;
    Node* next;
};
~~~

链表的优势是已知位置时插入删除方便。比如在某个节点后面插入新节点，只需要改几个指针。

但链表不能按下标快速访问。想找第 100 个元素，只能从头走过去。它的缓存局部性也差，因为节点分散在内存各处。

适合场景：需要频繁调整节点连接关系，或者练习指针和链式结构。

## 栈：只关心最近的东西

栈的规则是后进先出。最后放进去的，最先拿出来。

~~~cpp
std::stack<int> st;
st.push(1);
st.push(2);
st.pop();
~~~

栈适合处理“最近状态”。比如括号匹配时，遇到右括号就看最近的左括号；函数调用时，最后调用的函数先返回。

常见场景：括号匹配、单调栈、DFS 模拟、表达式求值、撤销操作。

## 队列：先来的先处理

队列的规则是先进先出，就像排队办业务。

~~~cpp
std::queue<int> q;
q.push(1);
q.push(2);
q.pop();
~~~

BFS 最常用队列，因为 BFS 就是一层一层处理。先进入队列的节点先扩展，所以能保证无权图最短步数。

如果两端都要进出，可以用 deque；如果每次都想拿最大或最小，可以用优先队列。

## 哈希表：用 key 快速找 value

哈希表把 key 通过哈希函数映射到桶里，平均查找复杂度接近 O(1)。

~~~cpp
std::unordered_map<std::string, int> score;
score["Alice"] = 95;
~~~

它适合频繁判断是否出现、统计次数、建立映射关系。

但哈希表不保证顺序。自定义类型还要提供哈希函数。哈希冲突严重时，性能也会下降。

如果需要有序遍历，应该考虑 map 或 set，而不是 unordered_map。

## 树：有层次的数据

树适合表达层级关系，比如文件目录、组织结构、表达式、索引。

二叉树每个节点最多两个孩子。二叉搜索树有左小右大的性质，方便查找。

~~~cpp
struct TreeNode {
    int val;
    TreeNode* left;
    TreeNode* right;
};
~~~

树题最常用递归。递归时先想当前节点要做什么，然后把左右子树交给同样逻辑处理。

常见遍历：前序、中序、后序、层序。前中后序是 DFS，层序是 BFS。

## 堆：快速拿最大或最小

堆不是完全有序结构，它只保证堆顶是最大或最小。

~~~cpp
std::priority_queue<int> pq;
pq.push(3);
pq.push(5);
pq.push(1);
std::cout << pq.top() << std::endl;
~~~

堆适合 TopK、任务调度、合并多个有序链表、Dijkstra 最短路。

不要误以为堆里所有元素都是排好序的。它只保证每次取出的堆顶符合优先级。

## 图：关系网络

图由点和边组成，适合表达“谁和谁有关”。社交关系、课程依赖、地图路线、网络拓扑都可以抽象成图。

常见存法：

- 邻接矩阵：适合点少、边密。
- 邻接表：适合点多、边稀疏。

~~~cpp
std::vector<std::vector<int>> graph(n);
graph[u].push_back(v);
~~~

图的核心操作是遍历。DFS 适合深入搜索，BFS 适合最短步数。

图题最容易漏的是 visited。没有 visited，无向图和有环图很容易死循环。

## 并查集：合并集合

并查集适合处理“两个元素是不是属于同一个集合”。它常用于连通性问题。

~~~cpp
int find(int x) {
    if (parent[x] != x) parent[x] = find(parent[x]);
    return parent[x];
}
~~~

路径压缩会把树压扁，让后续查找更快。

常见场景：朋友圈、岛屿连接、冗余连接、Kruskal 最小生成树。

## 字典树：按字符走路径

字典树适合处理大量字符串前缀。比如判断某个单词是否存在，或者找所有以某个前缀开头的单词。

人话理解：每条从根到节点的路径就是一个前缀。

适合场景：搜索提示、单词匹配、前缀统计。

## 怎么选择

选择数据结构时先看操作：

- 按下标访问多：数组或 vector。
- 已知位置插删多：链表。
- 最近状态：栈。
- 先来先处理：队列。
- 快速查 key：哈希表。
- 要有序：map、set、堆或搜索树。
- 层级关系：树。
- 关系网络：图。
- 集合合并：并查集。

不要先想“我要用高级结构”，先看题目最频繁的操作是什么。

## 常见坑

- 链表没有处理空节点和虚拟头节点。
- vector 插入删除后迭代器失效。
- unordered_map 遍历顺序不稳定。
- 树递归没有终止条件。
- 图遍历忘记 visited。
- priority_queue 只保证堆顶，不保证整体有序。
- 只背复杂度，不会解释为什么。

## 面试怎么说

回答数据结构问题时，可以按“存储方式、操作复杂度、适用场景、缺点”四步讲。

比如数组和链表：数组是连续内存，随机访问 O(1)，但中间插删要移动元素；链表节点分散，随机访问 O(n)，但已知位置插删只改指针。实际工程中很多时候优先 vector，因为连续内存带来的缓存收益很明显。

## 总结

数据结构的核心是取舍。数组、链表、栈、队列、哈希、树、堆、图、并查集都不是孤立知识点。它们解决的是不同操作成本的问题。学会根据场景选择，比背更多名字更重要。

# 数组

- **特点**：内存是连续的
    
- **优点**：
    
    - 适合随机访问，下标访问（随机访问）时间复杂度为O(1)
        
    - 末尾位置增删元素的时间复杂度为O(1)
        
    - 访问元素前后相邻位置的元素很方便
        
- **缺点**：
    
    - 非末尾位置增删元素需要移动大量元素，时间复杂度为O(n)
        
    - 搜索：无序数组搜索O(n)，有序数组搜索O(logn)
        
    - 数组扩容消耗较大
        

### 数组实现代码

> 说人话：数据结构先问“数据怎么摆”。摆法决定查找、插入、删除和遍历的成本，也决定它适合什么题。

```cpp
#include <iostream>
#include <cstring> // 用于memcpy
using namespace std;

class Array {
public:
    // 构造函数，默认大小为10
    Array(int size = 10) : mCur(0), mCap(size) {
        mpArr = new int[mCap]();  // 分配并初始化为0
    }
    
    // 析构函数
    ~Array() {  
        delete[] mpArr;
        mpArr = nullptr;
    }

    // 尾部添加元素
    void push_back(int val) 
    {
        if (mCur == mCap)  // 如果空间已满
        {
            expand(2 * mCap);  // 扩容为当前容量的2倍
        }
        mpArr[mCur++] = val;  // 添加元素并更新计数
    }

    // 尾部删除元素
    void pop_back() {
        if (mCur > 0) {
            mCur--;  // 只需减少当前元素计数
        }
    }

    // 在指定位置插入元素
    void insert(int pos, int val) {
        // 检查位置是否有效
        if (pos < 0 || pos > mCur) {
            return;
        }
        // 如果空间不足则扩容
        if (mCur == mCap) {
            expand(2 * mCap);
        }
        // 将pos之后的元素后移
        for (int i = mCur - 1; i >= pos; i--) 
        {
            mpArr[i + 1] = mpArr[i];
        }
        // 插入新元素
        mpArr[pos] = val;
        mCur++;  // 更新元素计数
    }

    // 按值删除第一个匹配项
    void erase(int val) {
        for (int i = 0; i < mCur; i++) {
            if (mpArr[i] == val) {
                // 将后续元素前移
                for (int j = i; j < mCur - 1; j++) 
                {
                    mpArr[j] = mpArr[j + 1];
                }
                mCur--;  // 更新元素计数
                return;  // 只删除第一个匹配项
            }
        }
    }
    
    // 查找值并输出结果
    void find(int val) {
        for (int i = 0; i < mCur; i++) {
            if (mpArr[i] == val) {
                cout << "Found at index: " << i << endl;
                return;
            }
        }
        cout << "Value not found" << endl;
    }
    
    // 打印数组内容
    void print() {
        cout << "Array: ";
        for (int i = 0; i < mCur; i++) {
            cout << mpArr[i] << " ";
        }
        cout << endl;
    }

private:
    // 扩容函数
    void expand(int size) {
        int* p = new int[size];  // 创建新数组
        memcpy(p, mpArr, sizeof(int) * mCap);  // 复制旧数据
        delete[] mpArr;  // 释放旧数组
        mpArr = p;       // 更新指针
        mCap = size;     // 更新容量
    }

private:
    int* mpArr;   // 指向数组的指针
    int mCap;    // 数组容量
    int mCur;    // 当前元素数量
};

// 测试代码
int main() {
    Array arr(3);  // 初始容量为3
    
    // 添加元素
    arr.push_back(10);
    arr.push_back(20);
    arr.push_back(30);
    arr.print();  // 输出: 10 20 30
    
    // 触发扩容
    arr.push_back(40);
    arr.print();  // 输出: 10 20 30 40
    
    // 插入元素
    arr.insert(2, 25);
    arr.print();  // 输出: 10 20 25 30 40
    
    // 删除元素
    arr.erase(20);
    arr.print();  // 输出: 10 25 30 40
    
    // 查找元素
    arr.find(30);  // 输出: Found at index: 2
    arr.find(99);  // 输出: Value not found
    
    return 0;
}
```


# 链表
## 单向链表

> 说人话：数据结构先问“数据怎么摆”。摆法决定查找、插入、删除和遍历的成本，也决定它适合什么题。

- **特点**：内存不连续，通过指针连接
    
- **优点**：
    
    - 任意位置插入/删除元素时间复杂度O(1)
        
    - 不需要预先分配固定大小的内存，内存利用率高
        
    - 动态扩展方便
        
- **缺点**：
    
    - 不支持随机访问，访问元素需遍历O(n)
        
    - 额外空间存储指针
        
    - 缓存不友好（内存不连续）
        

### 链表实现代码

> 说人话：数据结构先问“数据怎么摆”。摆法决定查找、插入、删除和遍历的成本，也决定它适合什么题。

```cpp
#include <iostream>
using namespace std;

// 链表节点
struct ListNode {
    int val;
    ListNode* next;
    ListNode(int x) : val(x), next(nullptr) {}
};

class LinkedList {
private:
    ListNode* head;  // 头节点指针
    int size;        // 链表长度

public:
    // 构造函数
    LinkedList() : head(nullptr), size(0) {}
    
    // 析构函数
    ~LinkedList() {
        clear();  // 调用clear函数释放所有节点
    }
    
    // 清空链表
    void clear() {
        ListNode* cur = head;
        while (cur) {
            ListNode* temp = cur;
            cur = cur->next;
            delete temp;
        }
        head = nullptr;
        size = 0;
    }
    
    // 头部插入（头插法）
    void push_front(int val) {
        ListNode* newNode = new ListNode(val);
        newNode->next = head;
        head = newNode;
        size++;
    }
    
    // 尾部插入（尾插法）
    void push_back(int val) {
        ListNode* newNode = new ListNode(val);
        if (!head) {
            head = newNode;
        } else {
            ListNode* cur = head;
            // 遍历到最后一个节点
            while (cur->next) {
                cur = cur->next;
            }
            cur->next = newNode;
        }
        size++;
    }
    
    // 在指定位置插入节点
    void insert(int index, int val) {
        if (index < 0 || index > size) {
            cout << "Invalid index!" << endl;
            return;
        }
        
        if (index == 0) {
            push_front(val);  // 头部插入
        } else if (index == size) {
            push_back(val);   // 尾部插入
        } else {
            // 找到插入位置的前驱节点
            ListNode* prev = head;
            for (int i = 0; i < index - 1; i++) {
                prev = prev->next;
            }
            
            ListNode* newNode = new ListNode(val);
            newNode->next = prev->next;
            prev->next = newNode;
            size++;
        }
    }
    
    // 删除指定位置节点（删除单个节点）
    void removeAt(int index) {
        if (index < 0 || index >= size || !head) {
            cout << "Invalid index or empty list!" << endl;
            return;
        }
        
        if (index == 0) {
            // 删除头节点
            ListNode* temp = head;
            head = head->next;
            delete temp;
        } else {
            // 找到要删除节点的前驱节点
            ListNode* prev = head;
            for (int i = 0; i < index - 1; i++) {
                prev = prev->next;
            }
            
            ListNode* toDelete = prev->next;
            prev->next = toDelete->next;
            delete toDelete;
        }
        size--;
    }
    
    // 删除第一个匹配值的节点
    void removeFirst(int val) {
        if (!head) return;
        
        // 处理头节点匹配的情况
        if (head->val == val) {
            ListNode* temp = head;
            head = head->next;
            delete temp;
            size--;
            return;
        }
        
        // 遍历查找匹配节点
        ListNode* prev = head;
        ListNode* cur = head->next;
        while (cur) {
            if (cur->val == val) {
                prev->next = cur->next;
                delete cur;
                size--;
                return;
            }
            prev = cur;
            cur = cur->next;
        }
    }
    
    // 删除所有值为val的节点
    void removeAll(int val) {
        ListNode* cur = head;
        ListNode* prev = nullptr;
        
        while (cur) {
            if (cur->val == val) {
                if (prev) {
                    // 删除中间节点
                    prev->next = cur->next;
                    delete cur;
                    cur = prev->next;
                } else {
                    // 删除头节点
                    head = cur->next;
                    delete cur;
                    cur = head;
                }
                size--;
            } else {
                // 不匹配则继续遍历
                prev = cur;
                cur = cur->next;
            }
        }
    }
    
    // 查找值并返回索引
    int find(int val) {
        ListNode* cur = head;
        int index = 0;
        while (cur) {
            if (cur->val == val) {
                return index;
            }
            cur = cur->next;
            index++;
        }
        return -1;  // 未找到
    }
    
    // 反转链表
    void reverse() {
        ListNode* prev = nullptr;
        ListNode* cur = head;
        while (cur) {
            ListNode* next = cur->next;
            cur->next = prev;
            prev = cur;
            cur = next;
        }
        head = prev;
    }
    
    // 打印链表
    void print() {
        ListNode* cur = head;
        cout << "List: ";
        while (cur) {
            cout << cur->val << " ";
            cur = cur->next;
        }
        cout << endl;
    }
    
    // 获取链表长度
    int getSize() {
        return size;
    }
    
    // 检查链表是否为空
    bool isEmpty() {
        return size == 0;
    }
    
    // 获取指定位置的值
    int get(int index) {
        if (index < 0 || index >= size) {
            throw out_of_range("Index out of range");
        }
        
        ListNode* cur = head;
        for (int i = 0; i < index; i++) {
            cur = cur->next;
        }
        return cur->val;
    }
};

```

### 链表操作

#### 1. 插入操作

> 说人话：这一节讲的是“1. 插入操作”。先别急着背定义，先抓住它解决的问题、代码里怎么写、边界在哪里。

#### (1) 头插法 (push_front)

> 说人话：这一节讲的是“(1) 头插法 (push_front)”。先别急着背定义，先抓住它解决的问题、代码里怎么写、边界在哪里。

```cpp
void push_front(int val) {
```

#### 问题描述

> 说人话：这一节讲的是“解题思路”。先别急着背定义，先抓住它解决的问题、代码里怎么写、边界在哪里。

1. 创建包含n个节点的循环链表
    
2. 从第一个节点开始计数
    
3. 每数到k，删除当前节点
    
4. 重复直到只剩一个节点

> 说人话：这一节讲的是“代码实现”。先别急着背定义，先抓住它解决的问题、代码里怎么写、边界在哪里。

```cpp
#include <iostream>
using namespace std;

struct Person {
    int id;
    Person* next;
    Person(int x) : id(x), next(nullptr) {}
};

int josephus(int n, int k) {
    // 创建循环链表
    Person* head = new Person(1);
    Person* prev = head;
    for (int i = 2; i <= n; i++) {
        prev->next = new Person(i);
        prev = prev->next;
    }
    prev->next = head;  // 形成环
    
    // 模拟报数过程
    Person* cur = head;
    Person* prevNode = nullptr;
    
    while (cur->next != cur) {  // 直到只剩一人
        // 报数k-1次（因为当前节点算第一个）
        for (int i = 1; i < k; i++) {
            prevNode = cur;
            cur = cur->next;
        }
        
        // 删除当前节点
        prevNode->next = cur->next;
        delete cur;
        cur = prevNode->next;
    }
    
    int survivor = cur->id;
    delete cur;
    return survivor;
}

// 测试代码
int main() {
    int n = 7, k = 3;
    cout << "Last survivor with n=" << n << ", k=" << k 
         << " is: " << josephus(n, k) << endl;  // 输出: 4
    return 0;
}
```

```cpp
#include <iostream>
#include <vector>
#include <stdexcept>
using namespace std;

class ArrayStack {
private:
    int* data;         // 存储栈元素的数组
    int capacity;      // 栈的容量
    int topIndex;      // 栈顶索引（-1表示空栈）

public:
    // 构造函数（默认容量10）
    ArrayStack(int cap = 10) : 
    capacity(cap), 
    topIndex(-1) 
    {
        data = new int[capacity];
    }
    
    // 析构函数
    ~ArrayStack() {
        delete[] data;
    }
    
    // 入栈操作
    void push(int val) {
        // 检查是否需要扩容
        if (topIndex == capacity - 1) {
            resize(2 * capacity);  
            // 容量翻倍
        }
        data[++topIndex] = val;    
        // 栈顶指针上移并赋值
    }
    
    // 出栈操作
    int pop() {
        if (isEmpty()) {
            throw out_of_range("Stack is empty");
        }
        return data[topIndex--];  // 返回栈顶元素并下移指针
    }
    
    // 获取栈顶元素
    int top() {
        if (isEmpty()) {
            throw out_of_range("Stack is empty");
        }
        return data[topIndex];
    }
    
    // 判断栈是否为空
    bool isEmpty() {
        return topIndex == -1;
    }
    
    // 获取栈的大小
    int size() {
        return topIndex + 1;
    }
    
    // 打印栈内容（从栈底到栈顶）
    void print() {
        if (isEmpty()) {
            cout << "Stack is empty" << endl;
            return;
        }
        cout << "Stack (bottom to top): ";
        for (int i = 0; i <= topIndex; i++) {
            cout << data[i] << " ";
        }
        cout << endl;
    }
    
    // 调整栈容量
    void resize(int newCapacity) {
        int* newData = new int[newCapacity];
        for (int i = 0; i <= topIndex; i++) {
            newData[i] = data[i];
        }
        delete[] data;
        data = newData;
        capacity = newCapacity;
    }
};
```

```cpp
// 栈节点
struct StackNode {
    int data;
    StackNode* next;
    StackNode(int val) : data(val), next(nullptr) {}
};

class ListStack {
private:
    StackNode* topNode;  // 栈顶节点指针
    int count;           // 栈中元素计数

public:
    // 构造函数
    ListStack() : topNode(nullptr), count(0) {}
    
    // 析构函数
    ~ListStack() {
        while (!isEmpty()) {
            pop();
        }
    }
    
    // 入栈操作
    void push(int val) {
        StackNode* newNode = new StackNode(val);
        newNode->next = topNode;  
        // 新节点指向当前栈顶
        topNode = newNode;        
        // 更新栈顶指针
        count++;
    }
    
    // 出栈操作
    int pop() {
        if (isEmpty()) {
            throw out_of_range("Stack is empty");
        }
        StackNode* temp = topNode;
        int val = temp->data;
        topNode = topNode->next;  
        // 栈顶指针下移
        delete temp;              
        // 释放原栈顶节点
        count--;
        return val;
    }
    
    // 获取栈顶元素
    int top() {
        if (isEmpty()) {
            throw out_of_range("Stack is empty");
        }
        return topNode->data;
    }
    
    // 判断栈是否为空
    bool isEmpty() {
        return topNode == nullptr;
    }
    
    // 获取栈的大小
    int size() {
        return count;
    }
    
    // 打印栈内容（从栈顶到栈底）
    void print() {
        if (isEmpty()) {
            cout << "Stack is empty" << endl;
            return;
        }
        cout << "Stack (top to bottom): ";
        StackNode* cur = topNode;
        while (cur) {
            cout << cur->data << " ";
            cur = cur->next;
        }
        cout << endl;
    }
};

// 测试代码

    ListStack stack;
    
    // 入栈操作
    stack.push(10);
    stack.push(20);
    stack.push(30);
    stack.print();  // 输出: Stack (top to bottom): 30 20 10
    
    // 出栈操作
    cout << "Popped: " << stack.pop() << endl;  // 输出: Popped: 30
    cout << "Top element: " << stack.top() << endl;  
    // 输出: Top element: 20
    
    // 栈大小
    cout << "Stack size: " << stack.size() << endl;  
    // 输出: Stack size: 2
    
    return 0;
}
```

```cpp
class CircularQueue {
private:
    int* data;      // 存储队列元素的数组

    int rear;       // 队尾索引（指向最后一个元素的下一个位置）

        if (isEmpty()) {
            return false;  // 队列空，出队失败

        val = data[front];
        front = (front + 1) % capacity;
        return true;
    }

            return false;

        return front == rear;

    }

    int size() {
        return (rear - front + capacity) % capacity;

            cout << data[idx] << " ";
            idx = (idx + 1) % capacity;

        cout << endl;
```

> 说人话：这一节讲的是“实现代码”。先别急着背定义，先抓住它解决的问题、代码里怎么写、边界在哪里。

```cpp
#include <iostream>
using namespace std;

// 队列节点
struct QueueNode {
    int data;
    QueueNode* next;
    QueueNode(int val) : data(val), next(nullptr) {}
};

class ListQueue {
private:
    QueueNode* front;  // 队头指针
    QueueNode* rear;   // 队尾指针
    int count;         // 队列中元素数量

public:
    ListQueue() : front(nullptr), rear(nullptr), count(0) {}
    
    ~ListQueue() {
        while (!isEmpty()) {
            dequeue();
        }
    }
    
    // 入队操作
    void enqueue(int val) {
        QueueNode* newNode = new QueueNode(val);
        if (isEmpty()) {
            front = rear = newNode;
        } else {
            rear->next = newNode;
            rear = newNode;
        }
        count++;
    }
    
    // 出队操作
    bool dequeue(int& val) {
        if (isEmpty()) {
            return false;
        }
        QueueNode* temp = front;
        val = temp->data;
        front = front->next;
        if (!front) {
            rear = nullptr;  // 队列已空
        }
        delete temp;
        count--;
        return true;
    }
    
    // 获取队头元素
    bool frontElement(int& val) {
        if (isEmpty()) {
            return false;
        }
        val = front->data;
        return true;
    }
    
    // 判断队列是否为空
    bool isEmpty() {
        return front == nullptr;
    }
    
    // 获取队列大小
    int size() {
        return count;
    }
    
    // 打印队列内容
    void print() {
        if (isEmpty()) {
            cout << "Queue is empty" << endl;
            return;
        }
        cout << "Queue (front to rear): ";
        QueueNode* cur = front;
        while (cur) {
            cout << cur->data << " ";
            cur = cur->next;
        }
        cout << endl;
    }
};

// 测试代码
int main() {
    ListQueue queue;
    
    // 入队操作
    queue.enqueue(10);
    queue.enqueue(20);
    queue.enqueue(30);
    queue.print();  // 输出: Queue (front to rear): 10 20 30
    
    // 出队操作
    int val;
    queue.dequeue(val);
    cout << "Dequeued: " << val << endl;  // 输出: Dequeued: 10
    queue.print();  // 输出: Queue (front to rear): 20 30
    
    // 继续入队
    queue.enqueue(40);
    queue.enqueue(50);
    queue.print();  // 输出: Queue (front to rear): 20 30 40 50
    
    cout << "Queue size: " << queue.size() << endl;  // 输出: Queue size: 4
    
    return 0;
}
```

### 设计思路

> 说人话：这一节讲的是“设计思路”。先别急着背定义，先抓住它解决的问题、代码里怎么写、边界在哪里。

- 使用两个栈：`stackIn`（输入栈）和`stackOut`（输出栈）
    
- **入队操作**：直接将元素压入`stackIn`
    
- **出队操作**：
    
    - 如果`stackOut`为空，将`stackIn`的所有元素弹出并压入`stackOut`
        
    - 从`stackOut`弹出栈顶元素
        
- **关键点**：只有当`stackOut`为空时才需要从`stackIn`转移元素
    

### 代码实现

> 说人话：这一节讲的是“代码实现”。先别急着背定义，先抓住它解决的问题、代码里怎么写、边界在哪里。

```cpp
#include <iostream>
#include <stack>
using namespace std;

class MyQueue {
private:
    stack<int> stackIn;
    stack<int> stackOut;

public:
    /** Initialize your data structure here. */
    MyQueue() {}
    
    /** Push element x to the back of queue. */
    void push(int x) {
        stackIn.push(x);
    }
    
    /** Removes the element from in front of queue and returns that element. */
    int pop() {
        if (stackOut.empty()) {
            // 将stackIn中的所有元素转移到stackOut
            while (!stackIn.empty()) {
                stackOut.push(stackIn.top());
                stackIn.pop();
            }
        }
        int val = stackOut.top();
        stackOut.pop();
        return val;
    }
    
    /** Get the front element. */
    int peek() {
        if (stackOut.empty()) {
            while (!stackIn.empty()) {
                stackOut.push(stackIn.top());
                stackIn.pop();
            }
        }
        return stackOut.top();
    }
    
    /** Returns whether the queue is empty. */
    bool empty() {
        return stackIn.empty() && stackOut.empty();
    }
};

// 测试代码
int main() {
    MyQueue q;
    q.push(1);
    q.push(2);
    cout << "Front: " << q.peek() << endl;  // 输出: Front: 1
    cout << "Pop: " << q.pop() << endl;     // 输出: Pop: 1
    cout << "Empty: " << (q.empty() ? "Yes" : "No") << endl;  // 输出: Empty: No
    q.push(3);
    cout << "Pop: " << q.pop() << endl;     // 输出: Pop: 2
    cout << "Pop: " << q.pop() << endl;     // 输出: Pop: 3
    cout << "Empty: " << (q.empty() ? "Yes" : "No") << endl;  // 输出: Empty: Yes
    
    return 0;
}
```

```cpp
#include <iostream>
#include <queue>
using namespace std;

class MyStack {
private:
    queue<int> q1;
    queue<int> q2;
    queue<int>* mainQ;  // 指向非空队列
    queue<int>* auxQ;   // 指向空队列

public:
    MyStack() {
        mainQ = &q1;
        auxQ = &q2;
    }
    
    void push(int x) {
        mainQ->push(x);
    }
    
    int pop() {
        // 将mainQ中除最后一个元素外的所有元素转移到auxQ
        while (mainQ->size() > 1) {
            auxQ->push(mainQ->front());
            mainQ->pop();
        }
        // 弹出最后一个元素
        int val = mainQ->front();
        mainQ->pop();
        // 交换mainQ和auxQ
        swap(mainQ, auxQ);
        return val;
    }
    
    int top() {
        // 将mainQ中除最后一个元素外的所有元素转移到auxQ
        while (mainQ->size() > 1) {
            auxQ->push(mainQ->front());
            mainQ->pop();
        }
        // 获取最后一个元素
        int val = mainQ->front();
        // 将其也转移到auxQ
        auxQ->push(val);
        mainQ->pop();
        // 交换mainQ和auxQ
        swap(mainQ, auxQ);
        return val;
    }
    
    bool empty() {
        return mainQ->empty() && auxQ->empty();
    }
};

// 测试代码
```

## 三、大根堆（Max Heap）

> 说人话：内存相关内容先问一句：这块资源从哪里来，什么时候释放，谁负责释放。C++ 很多 bug 都是这条线没想清楚。

### 定义

> 说人话：这一节讲的是“定义”。先别急着背定义，先抓住它解决的问题、代码里怎么写、边界在哪里。

- **堆序性质**：每个节点的值都大于或等于其子节点的值
    
- **根节点**：整棵树的最大值
    
- **应用**：优先队列（最大优先）、堆排序
    

### 代码实现

> 说人话：这一节讲的是“代码实现”。先别急着背定义，先抓住它解决的问题、代码里怎么写、边界在哪里。

```cpp
#include <iostream>
#include <vector>
#include <stdexcept>
using namespace std;

class MaxHeap {
private:
    vector<int> data;
    
    // 获取父节点索引
    int parent(int i) { return (i - 1) / 2; }
    
    // 获取左子节点索引
    int left(int i) { return 2 * i + 1; }
    
    // 获取右子节点索引
    int right(int i) { return 2 * i + 2; }
    
    // 上浮操作
    void swim(int i) {
        while (i > 0 && data[parent(i)] < data[i]) {
            swap(data[parent(i)], data[i]);
            i = parent(i);
        }
    }
    
    // 下沉操作
    void sink(int i) {
        int n = data.size();
        while (true) {
            int l = left(i);
            int r = right(i);
            int largest = i;
            
            if (l < n && data[l] > data[largest]) {
                largest = l;
            }
            if (r < n && data[r] > data[largest]) {
                largest = r;
            }
            
            if (largest == i) break;
            
            swap(data[i], data[largest]);
            i = largest;
        }
    }

public:
    // 插入元素
    void push(int val) {
        data.push_back(val);
        swim(data.size() - 1);
    }
    
    // 删除堆顶元素
    int pop() {
        if (data.empty()) {
            throw runtime_error("Heap is empty");
        }
        
        int max_val = data[0];
        data[0] = data.back();
        data.pop_back();
        if (!data.empty()) {
            sink(0);
        }
        return max_val;
    }
    
    // 获取堆顶元素
    int top() {
        if (data.empty()) {
            throw runtime_error("Heap is empty");
        }
        return data[0];
    }
    
    // 判断堆是否为空
    bool empty() {
        return data.empty();
    }
    
    // 获取堆大小
    int size() {
        return data.size();
    }
    
    // 打印堆
    void print() {
        cout << "Max Heap: ";
        for (int val : data) {
            cout << val << " ";
        }
        cout << endl;
    }
};

// 测试代码
int main() {
    MaxHeap maxHeap;
    
    // 插入元素
    maxHeap.push(10);
    maxHeap.push(20);
    maxHeap.push(15);
    maxHeap.push(40);
    maxHeap.push(50);
    maxHeap.push(100);
    maxHeap.push(25);
    
    maxHeap.print();  // 输出: Max Heap: 100 50 25 40 10 15 20
    
    // 删除堆顶元素
    cout << "Pop: " << maxHeap.pop() << endl;  // 输出: Pop: 100
    maxHeap.print();  // 输出: Max Heap: 50 40 25 20 10 15
    
    // 获取堆顶元素
    cout << "Top: " << maxHeap.top() << endl;  // 输出: Top: 50
    
    return 0;
}
```

### 上浮操作（Swim/Sift Up）

> 说人话：这一节讲的是“上浮操作（Swim/Sift Up）”。先别急着背定义，先抓住它解决的问题、代码里怎么写、边界在哪里。

**目的**：将新插入的元素向上调整到合适位置

**操作步骤**：

1. 比较当前节点与其父节点
    
2. 如果违反堆序性质，交换两者
    
3. 继续向上比较，直到满足堆序性质
    

**代码实现**：

```cpp
void swim(int i) {
    while (i > 0 && data[parent(i)] < data[i]) {  // 大根堆
        swap(data[parent(i)], data[i]);
        i = parent(i);
    }
}
```

**时间复杂度**：O(log n)

### 下沉操作（Sink/Sift Down）

> 说人话：这一节讲的是“下沉操作（Sink/Sift Down）”。先别急着背定义，先抓住它解决的问题、代码里怎么写、边界在哪里。

**目的**：将堆顶元素向下调整到合适位置

**操作步骤**：

1. 比较当前节点与其子节点
    
2. 如果违反堆序性质，与较大（或较小）的子节点交换
    
3. 继续向下比较，直到满足堆序性质
    

**代码实现**：

```cpp
void sink(int i) {
    int n = data.size();
    while (true) {
        int l = left(i);
        int r = right(i);
        int largest = i;  // 大根堆
        
        if (l < n && data[l] > data[largest]) {
            largest = l;
        }
        if (r < n && data[r] > data[largest]) {
            largest = r;
        }
        
        if (largest == i) break;
        
        swap(data[i], data[largest]);
        i = largest;
    }
}
```

**时间复杂度**：O(log n)

```cpp
#include <iostream>
#include <vector>
#include <queue>
using namespace std;

class PriorityQueue {
private:
    priority_queue<int> maxPQ;  // 默认大根堆
    priority_queue<int, vector<int>, greater<int>> minPQ;  // 小根堆
    
public:
    // 插入元素（最大优先）
    void push_max(int val) {
        maxPQ.push(val);
    }
    
    // 删除最大元素
    int pop_max() {
        int val = maxPQ.top();
        maxPQ.pop();
        return val;
    }
    
    // 获取最大元素
    int top_max() {
        return maxPQ.top();
    }
    
    // 插入元素（最小优先）
    void push_min(int val) {
        minPQ.push(val);
    }
    
    // 删除最小元素
    int pop_min() {
        int val = minPQ.top();
        minPQ.pop();
        return val;
    }
    
    // 获取最小元素
    int top_min() {
        return minPQ.top();
    }
    
    // 判断队列是否为空
    bool empty() {
        return maxPQ.empty() && minPQ.empty();
    }

// 手动实现优先队列
template<typename T>
class MyPriorityQueue {
private:
    vector<T> heap;
    
    void swim(int i) {
        while (i > 0 && heap[(i - 1) / 2] < heap[i]) {
            swap(heap[(i - 1) / 2], heap[i]);
            i = (i - 1) / 2;
        }
    }
    
    void sink(int i) {
        int n = heap.size();
        while (true) {
            int l = 2 * i + 1;
            int r = 2 * i + 2;
            int largest = i;
            
            if (l < n && heap[l] > heap[largest]) largest = l;
            if (r < n && heap[r] > heap[largest]) largest = r;
            
            if (largest == i) break;
            
            swap(heap[i], heap[largest]);
            i = largest;
        }
    }

public:
    void push(T val) {
        heap.push_back(val);
        swim(heap.size() - 1);
    }
    
    T pop() {
        T val = heap[0];
        heap[0] = heap.back();
        heap.pop_back();
        if (!heap.empty()) sink(0);
        return val;
    }
    
    T top() {
        return heap[0];
    }
    
    bool empty() {
        return heap.empty();
    }
    
    int size() {
        return heap.size();
    }
};
```

```cpp
void heapify(vector<int>& arr, int n, int i) {
    int largest = i;
    int l = 2 * i + 1;
    int r = 2 * i + 2;
    
    if (l < n && arr[l] > arr[largest]) largest = l;

    }
```

```cpp
#include <iostream>
#include <vector>
#include <string>
#include <functional>
using namespace std;

class LinearProbingHashTable {
private:
    enum EntryType { ACTIVE, EMPTY, DELETED };
    
    struct HashEntry {
        int key;
        string value;
        EntryType info;
        
        HashEntry(int k = 0, const string& v = "", EntryType i = EMPTY)
            : key(k), value(v), info(i) {}
    };
    
    vector<HashEntry> table;
    int current_size;
    int capacity;
    
    // 哈希函数
    int hash(int key) {
        return key % capacity;
    }
    
    // 寻找下一个位置
    int findPos(int key) {
        int current_pos = hash(key);
        int offset = 1;
        
        while (table[current_pos].info != EMPTY &&
               table[current_pos].key != key) {
            current_pos = (current_pos + offset) % capacity;
            // 防止无限循环
            if (current_pos == hash(key)) {
                return -1;  // 表满
            }
        }
        return current_pos;
    }
    
    // 重新哈希
    void rehash() {
        vector<HashEntry> old_table = table;
        capacity *= 2;
        table.resize(capacity);
        
        // 初始化新表
        for (int i = 0; i < capacity; i++) {
            table[i] = HashEntry();
        }
        
        current_size = 0;
        
        // 重新插入所有元素
        for (const auto& entry : old_table) {
            if (entry.info == ACTIVE) {
                insert(entry.key, entry.value);
            }
        }
    }

public:
    LinearProbingHashTable(int cap = 11) 
        : capacity(cap), current_size(0) {
        table.resize(capacity);
        for (int i = 0; i < capacity; i++) {
            table[i] = HashEntry();
        }
    }
    
    // 插入操作
    void insert(int key, const string& value) {
        if (current_size >= capacity / 2) {
            rehash();  // 负载因子超过0.5时扩容
        }
        
        int pos = findPos(key);
        if (pos == -1) {
            rehash();
            insert(key, value);  // 递归调用
            return;
        }
        
        if (table[pos].info != ACTIVE) {
            current_size++;
        }
        table[pos] = HashEntry(key, value, ACTIVE);
    }
    
    // 查找操作
    string find(int key) {
        int pos = findPos(key);
        if (pos != -1 && table[pos].info == ACTIVE) {
            return table[pos].value;
        }
        return "";  // 未找到
    }
    
    // 删除操作
    void remove(int key) {
        int pos = findPos(key);
        if (pos != -1 && table[pos].info == ACTIVE) {
            table[pos].info = DELETED;
            current_size--;
        }
    }
    
    // 获取大小
    int size() {
        return current_size;
    }
    
    // 判断是否为空
    bool empty() {
        return current_size == 0;
    }
    
    // 打印哈希表
    void print() {
        cout << "Linear Probing Hash Table:" << endl;
        for (int i = 0; i < capacity; i++) {
            if (table[i].info == ACTIVE) {
                cout << "[" << i << "]: (" << table[i].key 
                     << ", " << table[i].value << ")" << endl;
            } else if (table[i].info == DELETED) {
                cout << "[" << i << "]: DELETED" << endl;
            } else {
                cout << "[" << i << "]: EMPTY" << endl;
            }
        }
    }
};
```

```cpp
#include <iostream>
#include <vector>
#include <list>
#include <string>

class SeparateChainingHashTable {
private:
    vector<list<pair<int, string>>> table;
    int capacity;
    int current_size;
    
    // 哈希函数
    int hash(int key) {
        return key % capacity;
    }

public:
    SeparateChainingHashTable(int cap = 11) 
        : capacity(cap), current_size(0) {
        table.resize(capacity);
    }
    
    // 插入操作
    void insert(int key, const string& value) {
        int pos = hash(key);
        list<pair<int, string>>& chain = table[pos];
        
        // 检查键是否已存在
        for (auto& pair : chain) {
            if (pair.first == key) {
                pair.second = value;  // 更新值
                return;
            }
        }
        
        // 添加新键值对
        chain.emplace_back(key, value);
        current_size++;
        
        // 负载因子过高时扩容
        if (current_size > capacity * 2) {
            rehash();
        }
    }
    
    // 查找操作
    string find(int key) {
        int pos = hash(key);
        list<pair<int, string>>& chain = table[pos];
        
        for (const auto& pair : chain) {
            if (pair.first == key) {
                return pair.second;
            }
        }
        return "";  // 未找到
    }
    
    // 删除操作
    void remove(int key) {
        int pos = hash(key);
        list<pair<int, string>>& chain = table[pos];
        
        for (auto it = chain.begin(); it != chain.end(); ++it) {
            if (it->first == key) {
                chain.erase(it);
                current_size--;
                return;
            }
        }
    }
    
    // 重新哈希
    void rehash() {
        vector<list<pair<int, string>>> old_table = table;
        capacity *= 2;
        table.clear();
        table.resize(capacity);
        current_size = 0;
        
        // 重新插入所有元素
        for (const auto& chain : old_table) {
            for (const auto& pair : chain) {
                insert(pair.first, pair.second);
            }
        }
    }
    
    // 获取大小
    int size() {
        return current_size;
    }
    
    // 判断是否为空
    bool empty() {
        return current_size == 0;
    }
    
    // 打印哈希表
    void print() {
        cout << "Separate Chaining Hash Table:" << endl;
        for (int i = 0; i < capacity; i++) {
            cout << "[" << i << "]: ";
            for (const auto& pair : table[i]) {
                cout << "(" << pair.first << "," << pair.second << ") ";
            }
            cout << endl;
        }
    }
};
```

> 说人话：这一节讲的是“2. 缓存系统（LRU Cache）”。先别急着背定义，先抓住它解决的问题、代码里怎么写、边界在哪里。

```cpp
#include <unordered_map>
#include <list>
#include <utility>
using namespace std;

class LRUCache {
private:
    int capacity;
    list<pair<int, int>> cache_list;  // 双向链表：(key, value)
    unordered_map<int, list<pair<int, int>>::iterator> cache_map;
    
public:
    LRUCache(int cap) : capacity(cap) {}
    
    int get(int key) {
        if (cache_map.find(key) == cache_map.end()) {
            return -1;
        }
        
        // 移动到链表头部
        cache_list.splice(cache_list.begin(), cache_list, cache_map[key]);
        return cache_map[key]->second;
    }
    
    void put(int key, int value) {
        if (cache_map.find(key) != cache_map.end()) {
            // 更新现有键
            cache_map[key]->second = value;
            cache_list.splice(cache_list.begin(), cache_list, cache_map[key]);
            return;
        }
        
        if (cache_list.size() == capacity) {
            // 删除最久未使用的元素
            int old_key = cache_list.back().first;
            cache_list.pop_back();
            cache_map.erase(old_key);
        }
        
        // 插入新元素
        cache_list.emplace_front(key, value);
        cache_map[key] = cache_list.begin();
    }
};
```

### 3.布谷鸟哈希

> 说人话：数据结构先问“数据怎么摆”。摆法决定查找、插入、删除和遍历的成本，也决定它适合什么题。
核心思想： 
使用两个哈希表和两个哈希函数。插入数据时，如果位置被占了，就“踢”走原来的数据，把原来的数据放到另一个表的对应位置中，就像布谷鸟寄生一样。如果踢的次数太多，就认为插入失败。

```c++
class CuckooHashTable {
private:
    vector<list<pair<int, string>>> table1, table2; // 两个独立的哈希表
    int capacity;
    
    // 两个完全不同的哈希函数，保证映射位置不同
    int hash1(int key) { return key % capacity; }
    int hash2(int key) { return (key * 31) % capacity; }
    
    // 递归辅助函数：尝试插入，如果被踢则递归处理
    // 参数：key, value, 当前尝试的表id, 递归深度（防止死循环）
    bool insertHelper(int key, const string& value, int table_id, int depth) {
        // 终止条件：如果递归太深（说明一直在踢皮球，死循环了），就放弃
        if (depth > 2 * capacity) return false;  
        
        if (table_id == 0) {
            int pos = hash1(key); // 在表1里找位置
            if (table1[pos].empty()) {
                // 如果位置空，直接放进去
                table1[pos].push_back({key, value});
                return true;
            } else {
                // 如果位置被占了，把原来的数据“踢”出去
                auto old = table1[pos].front(); // 获取被踢的数据
                table1[pos].clear(); // 清空当前位置（实际工程中可能是替换）
                table1[pos].push_back({key, value}); // 放进新数据
                
                // 让被踢出去的数据去表2找位置（递归）
                return insertHelper(old.first, old.second, 1, depth + 1);
            }
        } else {
            // 逻辑同上，只是换成了表2和hash2
            int pos = hash2(key);
            if (table2[pos].empty()) {
                table2[pos].push_back({key, value});
                return true;
            } else {
                auto old = table2[pos].front();
                table2[pos].clear();
                table2[pos].push_back({key, value});
                
                // 被踢出去的数据回到表1找位置
                return insertHelper(old.first, old.second, 0, depth + 1);
            }
        }
    }

public:
    bool insert(int key, const string& value) {
        // 默认从表0开始插入
        return insertHelper(key, value, 0, 0);
    }
};
```

### 4.一致性哈希

> 说人话：数据结构先问“数据怎么摆”。摆法决定查找、插入、删除和遍历的成本，也决定它适合什么题。
核心思想：
解决分布式服务器扩容时数据迁移量过大的问题。想象一个圆环，服务器和数据都按哈希值分布在这个环上，数据顺时针找最近的服务器。这样，增减服务器时，只有相邻的数据会受到影响。

```c++
class ConsistentHash {
private:
    map<int, string> ring;  // 哈希环：有序映射（自动排序模拟顺时针）
    int virtual_nodes;      // 虚拟节点数量
    
    // 通用哈希函数
    int hash(const string& key) {
        // 使用 FNV-1a 算法计算哈希值，并限制在 2^20 范围内
        return fnv1a_hash(key) % (1 << 20);
    }
    
public:
    // 添加一个物理服务器
    void addServer(const string& server) {
        // 为了解决数据倾斜（服务器分布不均），给一台服务器创建多个虚拟节点
        for (int i = 0; i < virtual_nodes; i++) {
            // 虚拟节点命名：server#0, server#1...
            string virtual_node = server + "#" + to_string(i);
            int hash_val = hash(virtual_node);
            // 将虚拟节点映射到环上，并指向真实的服务器名
            ring[hash_val] = server;
        }
    }
    
    // 根据数据key，找到负责它的服务器
    string getServer(const string& key) {
        int hash_val = hash(key); // 计算数据的哈希值
        
        // 关键操作：在环上顺时针找第一个 >= 数据哈希值的节点
        auto it = ring.lower_bound(hash_val);

        if (it == ring.end()) {
            it = ring.begin();
        }
        
        // 返回找到的那个节点对应的服务器名字
        return it->second;
    }
};
```

```cpp
    if (!root) return new TreeNode(val);
    if (val < root->val) {
        root->left = insertRecursive(root->left, val);
    } else if (val > root->val) {
        root->right = insertRecursive(root->right, val);
    }
    return root;
}

TreeNode* insertIterative(TreeNode* root, int val) {
    TreeNode* newNode = new TreeNode(val);
    if (!root) return newNode;
    
    TreeNode* current = root;
    TreeNode* parent = nullptr;
    
    while (current) {
        parent = current;
        if (val < current->val) {
            current = current->left;
        } else if (val > current->val) {
            current = current->right;
        } else {
            delete newNode;
            return root;
        }
    }
    
    if (val < parent->val) parent->left = newNode;
    else parent->right = newNode;
    return root;
}

// 查询操作
bool searchRecursive(TreeNode* root, int val) {
    if (!root) return false;
    if (root->val == val) return true;
    return val < root->val ? searchRecursive(root->left, val) 
                          : searchRecursive(root->right, val);
}

bool searchIterative(TreeNode* root, int val) {
    TreeNode* current = root;
    while (current) {
        if (val == current->val) return true;
        if (val < current->val) current = current->left;
        else current = current->right;
    }
    return false;
}

// 删除操作
TreeNode* findMin(TreeNode* node) {
    while (node->left) node = node->left;
    return node;
}

TreeNode* deleteRecursive(TreeNode* root, int val) {
    if (!root) return nullptr;
    
    if (val < root->val) {
        root->left = deleteRecursive(root->left, val);
    } else if (val > root->val) {
        root->right = deleteRecursive(root->right, val);
    } else {
        if (!root->left && !root->right) {
            delete root;
            return nullptr;
        } else if (!root->left || !root->right) {
            TreeNode* temp = root->left ? root->left : root->right;
            delete root;
            return temp;
        } else {
            TreeNode* minNode = findMin(root->right);
            root->val = minNode->val;
            root->right = deleteRecursive(root->right, minNode->val);
        }
    }
    return root;
}
```

# AVL树：自平衡二叉搜索树

## 一、AVL树基本概念

> 说人话：数据结构先问“数据怎么摆”。摆法决定查找、插入、删除和遍历的成本，也决定它适合什么题。

### 1. 定义与特性

> 说人话：这一节讲的是“1. 定义与特性”。先别急着背定义，先抓住它解决的问题、代码里怎么写、边界在哪里。

- **AVL树**：最早的自平衡二叉搜索树，由Adelson-Velsky和Landis发明
    
- **平衡因子(Balance Factor)**：对于任意节点，其左子树高度减去右子树高度的值
    
    - BF ∈ {-1, 0, 1} 表示树是平衡的
        
    - |BF| > 1 表示树不平衡，需要旋转调整
        
- **高度**：节点到最深叶节点的边数（空树高度为-1）
    

### 2. 平衡性维护

> 说人话：项目复盘要看链路和取舍：为什么这样拆模块，出了问题怎么定位，后续还能怎么优化。

- 插入/删除操作后检查平衡因子
    
- 若发现不平衡节点，通过旋转恢复平衡
    
- 旋转后更新受影响节点的高度

```cpp
TreeNode* leftRotate(TreeNode* z) {
```

```cpp
struct AVLNode {
    int key;
    int height;
    AVLNode* left;
    AVLNode* right;
    AVLNode(int k) : key(k), height(1), left(nullptr), right(nullptr) {}
};

// 辅助函数
int height(AVLNode* node) {
    return node ? node->height : 0;
}

int getBalance(AVLNode* node) {
    return node ? height(node->left) - height(node->right) : 0;
}

void updateHeight(AVLNode* node) {
    if (node) {
        node->height = 1 + max(height(node->left), height(node->right));
    }
}

// 插入操作
AVLNode* insert(AVLNode* node, int key) {
    // 1. 标准BST插入
    if (!node) return new AVLNode(key);
    
    if (key < node->key) {
        node->left = insert(node->left, key);
    } else if (key > node->key) {
        node->right = insert(node->right, key);
    } else {
        return node; // 重复键不允许
    }
    
    // 2. 更新高度
    updateHeight(node);
    
    // 3. 获取平衡因子
    int balance = getBalance(node);
    
    // 4. 根据不平衡类型旋转
    // LL情况
    if (balance > 1 && key < node->left->key) {
        return rightRotate(node);
    }
    
    // RR情况
    if (balance < -1 && key > node->right->key) {
        return leftRotate(node);
    }
    
    // LR情况
    if (balance > 1 && key > node->left->key) {
        node->left = leftRotate(node->left);
        return rightRotate(node);
    }
    
    // RL情况
    if (balance < -1 && key < node->right->key) {
        node->right = rightRotate(node->right);
        return leftRotate(node);
    }
    
    return node; // 无需旋转
}
```

> 说人话：这一节讲的是“代码实现”。先别急着背定义，先抓住它解决的问题、代码里怎么写、边界在哪里。

```cpp
// 查找最小值节点
AVLNode* minValueNode(AVLNode* node) {
    AVLNode* current = node;
    while (current && current->left) {
        current = current->left;
    }
    return current;
}

// 删除操作
AVLNode* remove(AVLNode* node, int key) {
    // 1. 标准BST删除
    if (!node) return nullptr;
    
    if (key < node->key) {
        node->left = remove(node->left, key);
    } else if (key > node->key) {
        node->right = remove(node->right, key);
    } else {
        // 节点有一个或零个子节点
        if (!node->left || !node->right) {
            AVLNode* temp = node->left ? node->left : node->right;
            if (!temp) {
                temp = node;
                node = nullptr;
            } else {
                // 复制非空子节点内容
                *node = *(temp);
            }
            delete temp;
        } else {
            // 节点有两个子节点：获取后继节点（右子树的最小节点）
            AVLNode* temp = minValueNode(node->right);
            node->key = temp->key;
            node->right = remove(node->right, temp->key);
        }
    }
    
    // 如果树只有一个节点，直接返回
    if (!node) return nullptr;
    
    // 2. 更新高度
    updateHeight(node);
    
    // 3. 获取平衡因子
    int balance = getBalance(node);
    
    // 4. 根据不平衡类型旋转
    // LL情况
    if (balance > 1 && getBalance(node->left) >= 0) {
        return rightRotate(node);
    }
    
    // LR情况
    if (balance > 1 && getBalance(node->left) < 0) {
        node->left = leftRotate(node->left);
        return rightRotate(node);
    }
    
    // RR情况
    if (balance < -1 && getBalance(node->right) <= 0) {
        return leftRotate(node);
    }
    
    // RL情况
    if (balance < -1 && getBalance(node->right) > 0) {
        node->right = rightRotate(node->right);
        return leftRotate(node);
    }
    
    return node;
}
```

```cpp
#include <iostream>
#include <algorithm>
using namespace std;

class AVLTree {
private:
    struct Node {
        int key;
        int height;
        Node* left;
        Node* right;
        Node(int k) : key(k), height(1), left(nullptr), right(nullptr) {}
    };
    
    Node* root;
    
    // 辅助函数
    int height(Node* node) {
        return node ? node->height : 0;
    }
    
    int getBalance(Node* node) {
        return node ? height(node->left) - height(node->right) : 0;
    }
    
    void updateHeight(Node* node) {
        if (node) {
            node->height = 1 + max(height(node->left), height(node->right));
        }
    }
    
    Node* rightRotate(Node* y) {
        Node* x = y->left;
        Node* T2 = x->right;
        
        x->right = y;
        y->left = T2;
        
        updateHeight(y);
        updateHeight(x);
        
        return x;
    }
    
    Node* leftRotate(Node* x) {
        Node* y = x->right;
        Node* T2 = y->left;
        
        y->left = x;
        x->right = T2;
        
        updateHeight(x);
        updateHeight(y);
        
        return y;
    }
    
    Node* insert(Node* node, int key) {
        if (!node) return new Node(key);
        
        if (key < node->key) {
            node->left = insert(node->left, key);
        } else if (key > node->key) {
            node->right = insert(node->right, key);
        } else {
            return node; // 重复键不允许
        }
        
        updateHeight(node);
        int balance = getBalance(node);
        
        // LL
        if (balance > 1 && key < node->left->key) {
            return rightRotate(node);
        }
        
        // RR
        if (balance < -1 && key > node->right->key) {
            return leftRotate(node);
        }
        
        // LR
        if (balance > 1 && key > node->left->key) {
            node->left = leftRotate(node->left);
            return rightRotate(node);
        }
        
        // RL
        if (balance < -1 && key < node->right->key) {
            node->right = rightRotate(node->right);
            return leftRotate(node);
        }
        
        return node;
    }
    
    Node* minValueNode(Node* node) {
        Node* current = node;
        while (current && current->left) {
            current = current->left;
        }
        return current;
    }
    
    Node* remove(Node* node, int key) {
        if (!node) return nullptr;
        
        if (key < node->key) {
            node->left = remove(node->left, key);
        } else if (key > node->key) {
            node->right = remove(node->right, key);
        } else {
            if (!node->left || !node->right) {
                Node* temp = node->left ? node->left : node->right;
                if (!temp) {
                    temp = node;
                    node = nullptr;
                } else {
                    *node = *temp;
                }
                delete temp;
            } else {
                Node* temp = minValueNode(node->right);
                node->key = temp->key;
                node->right = remove(node->right, temp->key);
            }
        }
        
        if (!node) return nullptr;
        
        updateHeight(node);
        int balance = getBalance(node);
        
        // LL
        if (balance > 1 && getBalance(node->left) >= 0) {
            return rightRotate(node);
        }
        
        // LR
        if (balance > 1 && getBalance(node->left) < 0) {
            node->left = leftRotate(node->left);
            return rightRotate(node);
        }
        
        // RR
        if (balance < -1 && getBalance(node->right) <= 0) {
            return leftRotate(node);
        }
        
        // RL
        if (balance < -1 && getBalance(node->right) > 0) {
            node->right = rightRotate(node->right);
            return leftRotate(node);
        }
        
        return node;
    }
    
    void inOrder(Node* node) {
        if (node) {
            inOrder(node->left);
            cout << node->key << "(" << getBalance(node) << ") ";
            inOrder(node->right);
        }
    }
    
public:
    AVLTree() : root(nullptr) {}
    
    void insert(int key) {
        root = insert(root, key);
    }
    
    void remove(int key) {
        root = remove(root, key);
    }
    
    void display() {
        cout << "AVL树中序遍历: ";
        inOrder(root);
        cout << endl;
    }
};
```

```cpp
enum Color { RED, BLACK };

struct RBNode {
    int key;
    Color color;
    RBNode* left;
    RBNode* right;
    RBNode* parent;
    RBNode(int k) : key(k), color(RED), left(NIL), right(NIL), parent(NIL) {}
};

RBNode* NIL = new RBNode(0);
RBNode* root = NIL;

void insertFixup(RBNode* z) {
    while (z->parent->color == RED) {
        if (z->parent == z->parent->parent->left) {
            RBNode* y = z->parent->parent->right; // 叔节点
            
            if (y->color == RED) {
                // Case 1: 叔节点是红色
                z->parent->color = BLACK;
                y->color = BLACK;
                z->parent->parent->color = RED;
                z = z->parent->parent;
            } else {
                if (z == z->parent->right) {
                    // Case 2: 叔节点是黑色，z是右孩子
                    z = z->parent;
                    leftRotate(root, z);
                }
                // Case 3: 叔节点是黑色，z是左孩子
                z->parent->color = BLACK;
                z->parent->parent->color = RED;
                rightRotate(root, z->parent->parent);
            }
        } else {
            // 对称情况
            RBNode* y = z->parent->parent->left; // 叔节点
            
            if (y->color == RED) {
                // Case 1
                z->parent->color = BLACK;
                y->color = BLACK;
                z->parent->parent->color = RED;
                z = z->parent->parent;
            } else {
                if (z == z->parent->left) {
                    // Case 2
                    z = z->parent;
                    rightRotate(root, z);
                }
                // Case 3
                z->parent->color = BLACK;
                z->parent->parent->color = RED;
                leftRotate(root, z->parent->parent);
            }
        }
    }
    root->color = BLACK;
}

void insert(int key) {
    RBNode* z = new RBNode(key);
    RBNode* y = NIL;
    RBNode* x = root;
    
    // 标准BST插入
    while (x != NIL) {
        y = x;
        if (z->key < x->key) {
            x = x->left;
        } else {
            x = x->right;
        }
    }
    
    z->parent = y;
    if (y == NIL) {
        root = z;
    } else if (z->key < y->key) {
        y->left = z;
    } else {
        y->right = z;
    }
    
    z->left = NIL;
    z->right = NIL;
    z->color = RED;
    
    insertFixup(z);
}
```

```cpp
    struct Node {
        int key;
        Color color;
        Node* left;
        Node* right;
        Node* parent;
        Node(int k) : key(k), color(RED), left(nullptr), right(nullptr), parent(nullptr) {}

    Node* NIL;
    Node* root;
    
    void leftRotate(Node* x) {
        Node* y = x->right;
        x->right = y->left;
        
        if (y->left != NIL) {
            y->left->parent = x;
        }
        
        y->parent = x->parent;
        
        if (x->parent == NIL) {
            root = y;
        } else if (x == x->parent->left) {
            x->parent->left = y;
        } else {
            x->parent->right = y;
        }
        
        y->left = x;
        x->parent = y;
    }
    
    void rightRotate(Node* y) {
        Node* x = y->left;
        y->left = x->right;
        
        if (x->right != NIL) {
            x->right->parent = y;
        }
        
        x->parent = y->parent;
        
        if (y->parent == NIL) {
            root = x;
        } else if (y == y->parent->left) {
            y->parent->left = x;
        } else {
            y->parent->right = x;
        }
        
        x->right = y;
        y->parent = x;
    }

        while (z->parent->color == RED) {
            if (z->parent == z->parent->parent->left) {
                Node* y = z->parent->parent->right;
                
                if (y->color == RED) {
                    z->parent->color = BLACK;
                    y->color = BLACK;
                    z->parent->parent->color = RED;
                    z = z->parent->parent;

                    if (z == z->parent->right) {
                        z = z->parent;
                        leftRotate(z);
                    }
                    z->parent->color = BLACK;
                    z->parent->parent->color = RED;
                    rightRotate(z->parent->parent);
                }

                Node* y = z->parent->parent->left;
                
                if (y->color == RED) {
                    z->parent->color = BLACK;
                    y->color = BLACK;
                    z->parent->parent->color = RED;
                    z = z->parent->parent;
                } else {
                    if (z == z->parent->left) {
                        z = z->parent;
                        rightRotate(z);
                    }
                    z->parent->color = BLACK;
                    z->parent->parent->color = RED;
                    leftRotate(z->parent->parent);
                }
            }
        }
        root->color = BLACK;
    }

                    w = x->parent->left;

                    x = x->parent;

                    w->left->color = BLACK;
                    rightRotate(x->parent);
                    x = root;

        root = NIL;

        Node* y = NIL;
        Node* x = root;
        
        while (x != NIL) {
            y = x;
            if (z->key < x->key) {
                x = x->left;
            } else {
                x = x->right;
            }
        }
        
        z->parent = y;
        if (y == NIL) {
            root = z;
        } else if (z->key < y->key) {
            y->left = z;
        } else {
            y->right = z;
        }

        z->right = NIL;

        if (z == NIL) return;

        if (z->left == NIL) {
            x = z->right;
            transplant(z, z->right);

            transplant(z, z->left);

            if (y->parent == z) {

                y->right = z->right;
                y->right->parent = y;

            y->left = z->left;
            y->left->parent = y;
            y->color = z->color;
        }
        
        if (y_original_color == BLACK) {
            deleteFixup(x);
        }
        
        delete z;
    }
    
    Node* search(Node* node, int key) {
        if (node == NIL || key == node->key) {
            return node;
        }
        if (key < node->key) {
            return search(node->left, key);
        } else {
            return search(node->right, key);
        }
    }

        inOrderTraversal(root);
        cout << endl;
```

## 一、B-树（B-Tree）：多路平衡查找树

> 说人话：数据结构先问“数据怎么摆”。摆法决定查找、插入、删除和遍历的成本，也决定它适合什么题。

### 1. B-树的定义与核心性质

> 说人话：数据结构先问“数据怎么摆”。摆法决定查找、插入、删除和遍历的成本，也决定它适合什么题。

**B-树**（又称B树）是一种**多路平衡查找树**，专为磁盘等外存设备设计，通过减少IO次数提升效率。

#### 核心性质（设阶数为 m，即每个节点最多有 m 个子节点）：

> 说人话：这一节讲的是“核心性质（设阶数为 m，即每个节点最多有 m 个子节点）”。先别急着背定义，先抓住它解决的问题、代码里怎么写、边界在哪里。

1. **节点结构**：每个节点包含 k 个关键字（k \leq m-1）和 k+1 个子节点指针（k+1 \leq m）。
    
2. **有序性**：节点内关键字按升序排列：key_1 < key_2 < ... < key_k。
    
3. **平衡性**：所有叶子节点在同一层（深度相同）。
    
4. **最小关键字限制**：除根节点外，每个节点至少有 \lceil m/2 \rceil - 1 个关键字（即至少 \lceil m/2 \rceil 个子节点）。
    

### 2. B-树的插入操作

> 说人话：数据结构先问“数据怎么摆”。摆法决定查找、插入、删除和遍历的成本，也决定它适合什么题。

#### 插入步骤：

3. **平衡性**：所有叶子节点在同一层，且链表有序。
