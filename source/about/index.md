这里是青阳寻雪的个人技术博客，使用 `Hexo + AnZhiYu` 搭建，部署在 Vercel，内容托管在 GitHub。

## 站点定位

这里会长期记录三类内容：

- 项目复盘：把项目中的设计、拆分、问题和取舍讲清楚。
- 技术笔记：重点记录 C++、分布式系统、算法、AI 开发、游戏开发和面试知识。
- 学习路线：把零散知识点整理成可以持续补充的目录。

## 当前专栏

- 计算机八股文笔记
- 算法笔记
- AI 开发笔记
- 分布式系统
- 游戏开发
- 项目合集

## 维护方式

文章写在 `source/_posts/`，项目入口写在 `source/projects/index.md`。修改后提交到 GitHub，Vercel 会自动重新部署。

```bash
npm run dev
npm run build
git add .
git commit -m "更新博客内容"
git push
```
