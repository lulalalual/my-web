这里是青阳寻雪的个人技术博客。这个站点主要记录项目复盘、代码笔记和学习路线。

## 写什么

- C++：把语法、对象、指针、模板这些内容讲清楚。
- 算法：用短代码解释常见题型。
- AI 开发：记录 AI 功能怎么真正放进产品。
- 项目复盘：把做过的项目拆开讲，说明思路和取舍。

## 怎么维护

文章都写在 `source/_posts/`。修改后执行：

```bash
npm run build
git add .
git commit -m "更新博客"
git push
```

这个站点会跟着 GitHub 自动部署。
